下列哪一項是智慧型手機常見的辨識技術？
A　Bluetooth
B　IrDa
C　NFC
D　ZigBee
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2040031
題目解析：
在辨識功能中，智慧型手機亦具有NFC、影像辨識（如QR Code掃描）、指紋辨識等功能。Bluetooth、IrDa、ZigBee皆為資料傳輸技術。

關於行動支付，下列敘述哪一項正確？
A　以移動存取設備透過移動網路，所啟動的支付行為
B　以固定存取設備透過移動網路，所啟動的支付行為
C　以移動存取設備透過固定網路，所啟動的支付行為
D　以固定存取設備透過固定網路，所啟動的支付行為
正確解答為：A　您的作答值為：A　答對了。TNO：O2040032
題目解析：
行動支付是指以移動存取設備透過移動網路，所啟動的支付行為，在不需使用現金、支票或信用卡的情況下，消費者可使用智慧型手機支付各項服務或數位及實體商品的費用。

C　利用悠遊卡坐捷運是行動支付的應用
D　行動支付也可在實體商店中使用
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2040033
題目解析：
將物聯網用於行動支付的商業應用時，最重要的技術就是使用者的身分識別，目前已有許多行動裝置搭配NFC、RFID等技術作為身分辨識使用。

下列哪一項並非現今智慧型手機中常見的感測器？
A　電子羅盤
B　光感測器
C　三軸加速度計
D　超音波感測器
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2040039
題目解析：
智慧型手機包含了物聯網中智慧物件的所有特點，在感知功能中，其內建了許多感測元件如三軸加速度計、電子羅盤、光感測器、麥克風、GPS定位系統等。超音波感測器相對之下由於體積較大，無法滿足智慧型手機的輕薄需求，且其應用主要在於距離量測，因此極少出現在智慧型手機中。

關於第五代行動通訊技術標準（5G），下列敘述哪一項錯誤？
A　5G比先前的4G LTE蜂巢式網路快100倍
B　5G靜態傳輸速率可達到10Gbps
C　5G網路和4G網路的時間延遲都是30-70毫秒
D　5G網路將不僅僅為手機提供服務，而且還將成為一般性的家庭和辦公網路提供商，與有線網路提供商競爭
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2040040
題目解析：
5G網路其中一個優點是較低的網路延遲（更快的回應時間），在同等條件下5G的延遲低於1毫秒，而4G為30-70毫秒。5G網路的主要優勢在於，資料傳輸速率遠遠高於以前的蜂巢式網路，最高可達10 Gbit/s，比先前的4G LTE蜂巢式網路快100倍。由於資料傳輸更快更便利，5G網路將不僅僅為手機提供服務，而且還將成為一般性的家庭和辦公網路提供商，與有線網路提供商競爭。

關於智慧型手機作業系統，下列敘述哪些正確？(複選題)
A　OS X為Apple公司針對行動裝置所開發的作業系統
B　Android的核心基於Linux
C　Windows也有出手機作業系統
D　iOS與Android官方皆免費開放原始碼給所有作業系統開發程式
正確解答為：BC　您的作答值為：A　回答錯誤了。TNO：O2040041
題目解析：
目前智慧型手機主要的作業系統有Google的Android、Apple公司針對行動裝置所開發的iOS以及Microsoft所推出的Windows Phone（Windows Mobile）。其中僅有Android有官方免費開放原始碼給所有作業系統開發程式，Android是基於Linux的開放原始碼的行動作業系統，大家都能取得Android原始碼，並在其上開發應用或改良系統，Android亦為使用最廣泛的智慧型手機作業系統。

關於智慧型手機，下列敘述哪些錯誤？(複選題)
A　可用指紋解開手機鎖
B　內建衛星定位系統，可以在戶外提供手機的位置資訊
C　所有智慧型手機皆已內建ZigBee無線通訊
D　內建的電子羅盤可以偵測手機的晃動
正確解答為：CD　您的作答值為：A　回答錯誤了。TNO：O2040042
