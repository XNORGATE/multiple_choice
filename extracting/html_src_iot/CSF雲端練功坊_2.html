目前在物聯網的建置環境中，最常需要考量的問題為下列哪一項？
A　設備的維護
B　異質網路的整合
C　使用者的接受程度
D　傳輸的數據流量
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2010031
題目解析：
物聯網中，物件與物件之間的通訊是達成智慧應用的根本，當各物件採取不同的通訊協定，例如：Wi-Fi、ZigBee、LTE-A等，勢必屏障物件間的通透性，因此需有進行異質網路的整合，將是物聯網環境建置階段最需要事先考量的項目。
2
目前是網際網路的時代，生活中存在許多的通訊協定和技術因而形成異質網路的環境，關於異質網路，下列敘述哪一項正確？
A　在物聯網的應用中，不同的設備會因為使用不同的網路技術而無法溝通
B　不同廠牌的網路設備無法相容
C　因為作業系統不一致，而造成網路的封包規格不同
D　網路的設定很難統一，造成使用者無法使用
正確解答為：A　您的作答值為：A　答對了。TNO：O2010032
題目解析：
在物聯網的應用中，異質網路的共存性是一項非常大的挑戰，起因於各式各樣的感測器可能使用不同通訊協定來傳輸資料，例如WiFi、ZigBee、Bluetooth等，就像是一個說中文的人要與說德文、法文、葡萄牙文的人溝通一樣。

物聯網和雲端運算的技術是不可分的，關於雲端運算技術，下列敘述哪一項錯誤？
A　雲端運算是一種基於網際網路的運算方式
B　依照使用者的需求，方便且快速地提供各式各樣的服務
C　常分成三層服務架構：雲端設備、雲端網路和雲端資料庫
D　使用者因為透過網路上網，即可使用龐大的電腦及網路資源，可以節省很多成本
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2010033
題目解析：
一般而言我們將雲端運算分為三層架構，分別為：雲端設備（IaaS）、雲端平台（PaaS）以及雲端軟體（SaaS）。

透過穿戴式裝置智慧手錶收集個人運動與健康數據，沒有使用到下列哪一種技術？
A　大數據分析技術
B　雲端技術
C　影像辨識技術
D　感測技術
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2010034
題目解析：
穿戴式裝置透過感測器以感測技術收集使用者運動與健康數據，透過通訊技術上傳至雲端，透過雲端技術與大數據分析技術，針對使用者提供個人化應用。

物聯網的定義中存在三層的架構，關於物聯網應用層，下列敘述哪一項錯誤？
A　應用層是物聯網與各行業間進行的專業技術融合
B　需依照各行業或用戶的需求來進行感知數據分析
C　例如使用洗衣機時，洗衣機會根據省電的時間來調整洗衣服的時間，幫用戶節省電費的一種應用行為
D　應用層的開發環境跟PC計算機的開發方式差不多，系統穩定性和多樣化是第一的考量
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2010035
題目解析：
物聯網應用層範圍遍及日常生活中的食衣住行育樂，在開發上因應不同的領域將會有各種不同的考量與挑戰。

物聯網的底層通訊技術是以無線短距離通訊技術為主，其中，NFC也越來越受到注目，關於NFC的應用，下列敘述哪一項錯誤？
A　NFC的特點是使用的簡易性
B　NFC系統的傳輸範圍為20公分以內
C　NFC應用主要是可以當做電子錢包使用
D　NFC不相容於一般的RFID標準，尤其是14443A
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2010036
題目解析：
只要RFID標籤相容於NFC讀寫器，就能正常運作，包含所有相容於ISO-14443A的標籤。

A　人工智慧（Artificial Intelligence）
B　5G
C　邊緣運算（Edge Computing）

關於物聯網和無線感測器網路中的網路機制6LoWPAN，下列敘述哪一項錯誤？
A　6LoWPAN可以視為精簡版的IPv6
B　具有短距離、低速率、低耗電和低記憶體使用的特性
C　大概的架構可以分為長距離通訊模式和短距離通訊模式兩種
D　6LoWPAN主要是讓物聯網的設備都可以使用IPv6來上網
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2010038
題目解析：
6LOWPAN跟WiFi一樣主要是使用2.4 GHz的頻段，但發射功率只有WiFi的1%，這也使得6LOWPAN的傳輸距離因此而受到了限制，6LoWPAN屬於短距離通訊技術。

網際網路是物聯網的基礎建設，其中，無線區域網路也是網際網路中的一部分，關於無線區域網路，下列敘述哪一項錯誤？
A　目前被廣泛使用的是802.11b、802.11g、802.11n和802.11ac
B　連結網路的方式有基礎建設模式（Infrastructure Mode）與隨建即連模式（Ad Hoc Mode）
C　不管是隨建即連模式（Ad Hoc Mode）和基礎建設模式（Infrastructure Mode），都只提供競爭週期的傳輸方式
D　改用CSMA/CA的碰撞偵測方式以避免Hidden Terminal問題
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2010039
題目解析：
在基礎建設模式中，傳輸的方式可以分為兩種方式，分別為免競爭週期（Contention-Free Period）與競爭週期（Contention Period）。

EPCglobal是物聯網中的一項國際商業物流標準，關於EPCglobal，下列敘述哪一項錯誤？
A　其網路系統是由RFID標籤、讀取器和後端系統所組成
B　透過電子方式共享資料，讓企業間可以合作
C　主要應用在物流資料處理的自動化，可以自動取得所有相關資訊
D　主要的產品編碼是4G
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2010040
題目解析：
RFID讀取器所讀取的EPC碼（產品電子代碼）是用來記錄市面上所流通的商品的代碼。EPC Global制訂了編碼標準，並提供了現行傳統實體條碼與EPC碼轉換的標準。相較於傳統實體條碼，EPC 碼能賦予每個實體一個獨一無二的編號，藉此物件相關資訊得以在散佈全球的EPC Network中存取，進而建立資訊交換標準，使得資訊系統能藉由更為豐富、詳細，並更具時效優勢的物件資訊描述，提供更有價值的資訊服務。
