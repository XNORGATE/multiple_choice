下列哪些為ZigBee網路層支援的網路拓樸？(複選題)
A　星狀拓樸
B　樹狀拓樸
C　環狀拓樸
D　網狀拓樸
正確解答為：ABD　您的作答值為：B　回答錯誤了。TNO：O2050001
題目解析：
Zigbee的網路結構特殊，支援的網路拓樸形態主要為星狀拓樸、樹狀拓樸及網狀拓樸。
2
下列哪一種標準的媒體存取控制層包含在ZigBee的全功能裝置中？
A　IEEE 802.3
B　IEEE 802.11n
C　IEEE 802.15.4
D　IEEE 802.20
正確解答為：C　您的作答值為：B　回答錯誤了。TNO：O2050002
題目解析：
ZigBee是一種與IEEE 802.15.4標準結合的一項低速率無線個人區域網路。Zigbee的網路架構較為特殊，實體層（Physical Layer）與媒體存取控制層（Medium Access Control Layer, MAC）使用地是IEEE 802.15.4標準架構，而網路層（Network Layer）與應用層（Application Layer）則是使用Zigbee Alliance制訂的標準。

下列哪一項不是目前最常使用的無線通訊技術？
A　Bluetooth
B　IrDA
C　IEEE 802.11
D　ZigBee
正確解答為：B　您的作答值為：B　答對了。TNO：O2050003
題目解析：
IrDA紅外通訊技術利用紅外線來傳遞數據，雖是無線通訊技術的一種，但因其傳輸距離短、傳輸速率低、易受障礙物影響等缺點，因此僅用於電視或冷氣的遙控器上。

下列哪些是藍牙（Bluetooth）的特性？(複選題)
A　成本低
B　長距離

下列哪些是無線區域網路的連結架構？(複選題)
A　基礎建設模式（Infrastructure Mode）
B　點對點模式（Point-to-Point Mode）
C　隨建即連模式（Ad Hoc Mode）
D　集中式模式（Central Mode）
正確解答為：AC　您的作答值為：B　回答錯誤了。TNO：O2050007
題目解析：
無線區域網路架構當中，主要分為兩種連結架構：基礎建設模式（Infrastructure Mode）與隨建即連模式（Ad-Hoc Mode），基礎建設模式下有存取點，所有的網路裝置都必須連結到存取點（Access Point）。在隨建即連模式下是沒有存取點，不需要一個存取點來集中控管網路資源。這樣的網路架構的目的在於一個小區域內則可以使用點對點傳輸，讓彼此互相溝通、通訊並相互連結。
8
在無線區域網路中的基礎建設模式（Infrastructure Mode）中，傳輸的方式應為下列哪一種？
A　免競爭週期
B　競爭週期
C　免競爭週期與競爭週期
D　公平分配週期
正確解答為：C　您的作答值為：B　回答錯誤了。TNO：O2050008
題目解析：
在基礎建設模式當中，將傳輸的型態分成兩種，第一種是免競爭週期（Contention-Free Period），另一種的傳輸型態則是競爭週期（Contention Period）。
9
在無線區域網路中的隨建即連模式（Ad Hoc Mode）中，傳輸的方式為下列哪一種？
A　免競爭週期
B　競爭週期
C　免競爭週期與競爭週期
D　公平分配週期
正確解答為：B　您的作答值為：B　答對了。TNO：O2050009
題目解析：
在隨建即連模式當中，因無中央管控協調者，因此傳輸的型態僅有競爭週期（Contention Period）一種。

若在無線區域網路中使用傳統的CSMA/CD，所有可以通訊的鄰居間不一定可以互相聽到彼此，是因為會產生下列哪一種問題？
A　暴露節點問題
B　控制節點問題
C　隱藏節點問題
D　隨意節點問題
正確解答為：C　您的作答值為：B　回答錯誤了。TNO：O2050010
題目解析：
在CSMA/CD的標準協定下，想傳輸的設備在發送訊息前須先確認附近的鄰居是否處於傳送資料的狀態，當所有鄰居皆不在傳輸資料狀態時才可進行傳輸。但此協定依然有可能發生資料碰撞的問題，原因在於設備的鄰居可能處在接收資料的狀態，造成該設備誤以為鄰居沒有在傳送資料，這種問題我們稱為隱藏節點問題（Hidden Terminal Problem）。

下列哪一種無線射頻接收技術，用於第四代（4G LTE）與第五代（5G NR）行動通訊技術標準上面？
A　TDM（Time Division Multiplexing）
B　FDM（Frequency Division Multiplexing）
C　OFDM（Orthogonal Frequency Division Multiplexing）
D　OTDM（Orthogonal Time Division Multiplexing）
正確解答為：C　您的作答值為：B　回答錯誤了。TNO：O2050011
題目解析：
LTE所使用的射頻技術為OFDM（Orthogonal Frequency Division Multiplexing）。2010年12月6日國際電信聯盟-無線電通訊部門（ITU-R）承認LTE以及其他3.9技術，雖不符合IMT-Advanced的要求，仍可以被認為是「4G」，但前提是它們是先行者。在這樣的狀況下，LTE-Advanced的標準在2010年正式問世。新世代5G行動通訊技術亦承續使用OFDM射頻技術。

下列哪一種網路架構是以IPv6通訊協定為基礎，針對IEEE 802.15.4的低功率、低可靠度網路裝置的互通網際網路協定？
A　Wi-Fi
B　Bluetooth
C　NFC
D　6LoWPAN
正確解答為：D　您的作答值為：B　回答錯誤了。TNO：O2050012
題目解析：
6LoWPAN可以被視為是精簡版的IPv6，因此6LoWPAN架構很適合讓低電量低頻寬需求的裝置運行IPv6的標準。因此具有短距離、低速率、低耗電、低記憶體使用量等特性的IEEE 802.15.4網路適合運行6LoWPAN架構，在IEEE 802.15.4網路上運行6LoWPAN，使所有的裝置都採用IPv6的方式連結上網路。

IEEE 802.15.4的描述，下列哪一項錯誤？
A　一種屬於低速無線個人網路
B　具有短距離、小體積、低工作週期、省電、低成本等特性
C　每個IEEE 802.15.4網路可以有多個網路協調者
D　可支援星狀（Star）及點對點（Peer-to-Peer）兩種網路拓撲
正確解答為：C　您的作答值為：B　回答錯誤了。TNO：O2050013
題目解析：
每個IEEE 802.15.4網路只能有一個網路協調者。

下列哪些是LTE-A的關鍵技術？(複選題)
A　載波聚合
B　中繼技術
C　協同多點傳輸
D　多天線增強
正確解答為：ABCD　您的作答值為：B　回答錯誤了。TNO：O2050014
題目解析：
LTE-A需要定一個可支援到40～100MHz頻寬的頻段，但要在一個國家裡找到一段連續完整的40MHz以上頻譜的頻段相當困難，因此國際電信聯盟允許透過載波聚合（Carrier Aggregation, CA）的方式，來達到40MHz以上的頻寬，其關鍵技術有包含載波聚合、協同多點傳輸、多天線增強、中繼技術。
