1
感測器的作用是將一種能量轉換成另一種能量形式，故有不少學者稱它為下列哪一種名稱？
A　Transducer
B　Transmitter
C　Receiver
D　Converter
正確解答為：A　您的作答值為：A　答對了。TNO：O2060031
題目解析：
感測器的作用是將一種能量轉換成另一種能量形式，所以不少學者也用「換能器－Transducer」來稱謂「感測器－Sensor」。
2
下列哪一項裝置能夠探測外界訊號及物理條件（光、熱、溼度），並將探測之資訊傳給其他裝置？
A　感測器
B　微控制器
C　數位訊號處理器
D　特殊應用積體電路
正確解答為：A　您的作答值為：A　答對了。TNO：O2060032
題目解析：
感測器有如人的五官一樣，可用來探測外界訊號及物理條件（光、熱、濕度等），並將探測之資訊傳給其他裝置。
3
許多物聯網的應用中，我們常會需要佈建數量眾多的感測器，下列哪一項不屬於感測器的特性？
A　成本低
B　運算速度快
C　耗電低
D　體積小
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2060033
題目解析：
應用於物聯網系統的感測器特性，主要包含低成本、低耗電、小體積，至於運算速度則並非主要的特性。

下列哪一種環境感測器可以測得可見光數據？
A　紅外線感測器
B　陽光感測器
C　紫外線感測器
D　色彩感測器
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2060034
題目解析：
陽光感測器可取得紫外線、可見光及紅外光三種數據。
5
下列哪一種微處理器為目前感測器暨嵌入式系統的主流？
A　特殊應用積體電路（ASIC）
B　數位訊號處理器（DSP）
C　現場可程式化邏輯閘陣列（FPGA）
D　微控制器（MCU）
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2060035
題目解析：
微控制器（Micro Control Unit, MCU）亦稱單晶片處理器，顧名思義即是同時將CPU、RAM、ROM、I/O、A/D、D/A、Flash等相關運算與記憶功能整合在一起，可將MCU看作為一個微型的電腦，並因此可大幅降低感測器的體積、成本與耗能，並增加感測器的應用與設計彈性。目前MCU已成為感測器與嵌入式系統的主流。
6
為了縮短感測器的開發時程，並降低微處理器的重新設計成本，下列哪一種微處理器比較適合用於產品開發時可能不斷變更設計的電路系統？
A　特殊應用積體電路（ASIC）
B　數位訊號處理器（DSP）
C　現場可程式化邏輯閘陣列（FPGA）
D　微控制器（MCU）
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2060036
題目解析：
現場可程式化邏輯閘陣列（Field-Programmable Gate Arrays, FPGA）是指可讓使用者利用編程的方式改變其內部邏輯閘元件組態的微處理器，因此適用於產品開發時可能不斷變更設計的感測器中，可縮短感測器的開發時間並降低微處理器重新設計的成本。

A　特殊應用積體電路（ASIC）
B　數位訊號處理器（DSP）
C　現場可程式化邏輯閘陣列（FPGA）
D　微控制器（MCU）

MCU亦稱單晶片處理器，顧名思義即是同時將CPU、RAM、ROM、I/O、A/D、D/A、Flash等相關運算與記憶功能整合在一起，可將MCU看作為一個微型的電腦，並因此可大幅降低感測器的體積、成本與耗能，並增加感測器的應用與設計彈性。

下列哪一種微處理器是指依特定用途而設計，它相當省電，因此適用於特殊需求的感測器電路系統中？
A　特殊應用積體電路（ASIC）
B　數位訊號處理器（DSP）
C　現場可程式化邏輯閘陣列（FPGA）
D　微控制器（MCU）
正確解答為：A　您的作答值為：A　答對了。TNO：O2060038
題目解析：
特殊應用積體電路（Application-specific integrated circuit, ASIC）是指依特定用途而設計的微處理器，它相當省電，因此適用於具有特殊需求的感測器中。
9
下列哪一種記憶體較適合存放感測器電路系統中，所需要的程式或運算指令？
A　Flash ROM
B　DRAM
C　SRAM
D　Disk
正確解答為：A　您的作答值為：A　答對了。TNO：O2060039
題目解析：
「儲存程式碼」主要是存放感測器所需之程式或運算指令，讓感測器能夠知道確切的工作內容與相關工作的細項，大多存放Flash ROM中。
10
下列哪一種記憶體較適合存放感測器電路系統中，所探測到的數據？
A　RAM
B　Flash
C　ROM
D　Disk
正確解答為：A　您的作答值為：A　答對了。TNO：O2060040
題目解析：
因感測器需要時常更新與運算感測數據，為了使後續的資訊運算、處理或傳送能夠快速運行，感測器大多將感測數據暫時存放於記憶體RAM中。

下列哪一種感測器可以感知磁場方向，因此具有簡易辨別方向的能力？
A　電子羅盤
B　陀螺儀
C　移動偵測器
D　震動感測器
正確解答為：A　您的作答值為：A　答對了。TNO：O2060041
題目解析：
電子羅盤可以感知環境的磁場方向，此感測器能想像成是一個指北針，因此裝上此感測模組的感測器將具有簡易辨別方位的能力。

A　電子羅盤

C　移動偵測器
D　震動感測器

微波移動感測器是利用微波訊號，並且配合杜普勒效應的原理感測物體的移動，因此它時常使用於保全方面。

A　超音波感測器

D　雷達感測器

電子羅盤可取得方位，可用於自駕車、無人機及船舶的導航。

感測器為整合不同的硬體模組，模組間必須透過傳輸介面才得以成功交換資料，下列哪些為常見的溝通介面？(複選題)
A　UART
B　SPI
C　I2C
D　JTAG
正確解答為：ABC　您的作答值為：A　回答錯誤了。TNO：O2060044
題目解析：
感測器亦為了整合不同的硬體模組，模組間必須要透過傳輸介面才得以成功交換資料，常見的溝通介面有UART（Universal Asynchronous Receiver/ Transmitter）、SPI（Serial Peripheral Interface）及I2C（Inter-Integrated Circuit）三種。

為使智慧型感測器能長期穩定工作，因此感測器的電源模組必須具備下列哪些特性？(複選題)
A　低自放電（Low self-discharge）
B　高蓄電容量（High capacity/volume）
C　較短的充電時間（Shorter recharge time）
D　高電流下有效率地充電（Efficient recharging at high current）
正確解答為：ABC　您的作答值為：A　回答錯誤了。TNO：O2060045
題目解析：
感測器電源供應模組需要具備以下特點：長壽命、低自放電、電壓穩定、低成本、高蓄電容量、在低電流下有效率地充電、較短的充電時間等特性。
