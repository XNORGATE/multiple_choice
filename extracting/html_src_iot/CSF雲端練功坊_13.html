使用物聯網技術來偵測老年人的生理/心理狀態的應用，我們稱之為智慧醫療服務，這項服務最受重視的是獨居老人的照護與關懷，下列哪一種設備不適用於獨居老人居家照護的環境中？
A　攝影機
B　警示鈴
C　網路影像電話
D　特殊氣體感測器
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2030061
題目解析：
透過特殊氣體感測器可感測獨居老人生活環境的氣體濃度，如一氧化碳、瓦斯等氣體，照護人員可透過遠端網路觀看家中攝影機的錄像，並了解家中特殊氣體濃度，並可即時掌握任何異常狀況，若發生任何異常，更可自動響起警示鈴，供就近醫護人員即時處理。「網路影像電話」的網路影像電話視角太小，且不易監控居家環境各個角落，不適用於獨居老人之物聯網應用中。

關於物聯網技術，下列哪一項錯誤？
A　將原有商品與物聯網技術結合，大幅提升商業價值
B　搭配的APP能透過網路與雲端平台連結
C　透過使用砧板與物聯網的技術結合，達到智慧飲食監控的目的
D　物聯網的網路技術，對攝取的食材進行辨識與分析，使人保持身體健康
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2030062
題目解析：
利用物聯網的感知技術，進一步判斷使用者所攝取的食材進行辨識與分析，或是針對環境中的危害粒子進行評估，使人可以保持在一個健康的生活中。

在物聯網的智慧醫療應用中，智慧藥盒服藥提醒系統也是其中的一項著名的應用，關於智慧藥盒服藥提醒系統，下列敘述哪一項錯誤？
A　主要包含兩個硬體架構，主控制板與智慧藥盒
B　當病患需要吃藥時，智慧藥盒將提醒病人應該拿取哪一格藥物
C　主控制板主要負責對藥盒資訊的同步以及控制命令的下達
D　智慧藥盒具備紅外線無線傳輸協定與微控制器、LED燈與語音提醒，由於使用紅外線協定，所以藥盒可以隨身攜帶，隨時提醒
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2030063
題目解析：
具備物聯網特性的智慧藥盒，可藉由Bluetooth或Wi-Fi通訊技術與智慧手機通訊，當使用者透過手機收到來自智慧藥盒的用藥提醒時，即可進行取藥用藥的動作。因此，使用者無需時時刻刻將藥盒帶在身邊，而紅外線通訊技術因訊號容易被遮蔽導致傳輸中斷且限制放置處，因此紅外線通訊技術並不適合用於物聯網的通訊。

在物聯網的智慧醫療應用中，智慧藥盒服藥提醒系統是其中的一項應用。若我們希望使用者隨身攜帶的手機能夠隨時接收智慧藥盒發送的用藥提醒，下列哪一種無線通訊協定最不適合使用於智慧藥盒中？
A　ZigBee
B　Bluetooth
C　5G
D　Wi-Fi
正確解答為：A　您的作答值為：A　答對了。TNO：O2030064
題目解析：
目前智慧手機已普遍具備Bluetooth、4G/5G、Wi-Fi無線通訊的能力，可接收智慧藥盒發送的用藥提醒。

醫療管理是醫院最重要的管理項目之一，特別需要物聯網技術協助的原因為下列哪一項？
A　因為需要管理藥品、血液以及消毒物品等項目，若有疏失，造成病人服錯藥物或是輸血錯誤，甚至手術器材與針筒未消毒而導致急性過敏或敗血症等，都可能危及一個人的生命
B　因為醫師不太喜歡管設備
C　因為流程太複雜，而且醫院的職員工作太多，造成效率不彰
D　因為醫院是封閉式系統，很難導入新的管理資訊系統，用來協助器材等的管理
正確解答為：A　您的作答值為：A　答對了。TNO：O2030065
題目解析：
由於藥品、血液、消毒物品種類與數量眾多，不易有效藉由人工管理。基於生命安全的因素，相關醫療材料與器具的管理，也成為醫療管理最需要物聯網技術協助的項目之一。
6
在物聯網的應用中，醫療管理也是重要的技術應用之一，為了降低人為的管理疏失，甚至提高醫院整體的運行效率，在醫療設備上面加裝感測器是必要的設計，下列哪一項不屬於目前物聯網醫療管理應用的項目？
A　藥品
B　實驗、手術器材和針筒
C　醫療廢棄物管理
D　醫院空病床
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2030066
題目解析：
藥品、實驗/手術器材和針筒、醫療廢棄物的使用或管理不當，容易造成病患身體不適，乃至於生命危害，由於人工管理不易，因此可藉由物聯網的醫療管理應用，降低人為的管理疏失，甚至提高醫院整體的運行效率。

A　嵌入式微控設備
B　晶圓技術

隨著資通訊、嵌入式微控設備、物聯網、雲端計算及大數據等技術的成長以及整合，使物聯網的應用進入新的世代。

下列那一項是物聯網應用的醫療感測元件，被用來檢測病患青光眼的變化情況？
A　眼壓感測隱形眼鏡
B　血壓感測器
C　心跳感測器
D　三軸加速度計
正確解答為：A　您的作答值為：A　答對了。TNO：O2030068
題目解析：
「眼壓感測隱形眼鏡」，透過鏡片中壓力感測裝置，可以紀錄到角膜壓力和眼球液壓的變化，並被用來檢測病患青光眼的變化情況。
9
在物聯網的醫療應用中，眼壓感測隱形眼鏡是一項著名的應用，關於眼壓感測隱形眼鏡，下列敘述哪一項正確？
A　透過鏡片中壓力感測裝置的紀錄，可以紀錄到角膜壓力和眼球液壓的變化
B　必須透過有線裝置，連接到眼鏡上面，因此需要配戴專用眼鏡
C　除了檢測青光眼外，還可以調整視力，和修正老花眼
D　需要常常拔下來充電
正確解答為：A　您的作答值為：A　答對了。TNO：O2030069
題目解析：
「必須透過有線裝置，連接到眼鏡上面，因此需要配戴專用眼鏡」與「需要常常拔下來充電」 的條件，勢必造成使用者在使用上的不便，甚至是身體傷害；「除了檢測青光眼外，還可以調整視力，和修正老花眼」則是現在隱形眼鏡無法達到的醫療功能。

關於電子病歷（EMR），以下敘述哪一項錯誤？
A　電子病歷（EMR）可減少診斷病患的副作用
B　電子病歷（EMR）包含個人運動健康紀錄
C　電子病歷（EMR）包含病患個人資料
D　電子病歷（EMR）包含病患藥物使用、過敏與免疫史
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2030070
題目解析：
電子病歷（EMR）為病患電子化的病歷檔，包含的內容如病患個人資料、病史、健康與疾病的檢驗與病程報告、藥物使用、過敏與免疫史等。
