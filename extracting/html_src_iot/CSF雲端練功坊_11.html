A　物聯網於安全之應用
B　物聯網於食品之應用
C　物聯網於物流之應用
D　物聯網於醫療之應用
正確解答為：A　您的作答值為：A　答對了。TNO：O2030031
題目解析：
警務人員具人臉辨識功能的道路監控系統，可用以協尋失蹤在外的失智老人（感知層），當辨識出可能的失蹤人口，便可將相關畫面與地理資訊傳遞給警務人員及相關家屬（網路層），以防老人在外遭遇危險（安全應用）。

交通執法與服務e化系統計畫，建置交通資訊匯流平台，透過物聯網與感測器技術，可將交通局路口影像CCTV、警察廣播電台即時路況資訊系統、國道高速公路局路況加以整合，並自動將相關資訊傳遞至交通警察大隊勤務中心，由專人負責監看路況，達到發現快、反應快、處置快的目標，下列哪一項不屬於此套平台的相關運用？
A　避開車禍現場
B　交通違規檢舉
C　注意施工路段
D　停車場車位查詢
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2030032
題目解析：
透過物聯網與感測器技術，交通資訊匯流平台可整合：路口影像資訊、即時路況資訊、國道高速公路路，並即時傳遞相關資訊給相關單位，乃屬於路況相關之智慧交通應用，因而可提醒鴐駛人避開車禍與施工相關道路，亦可達到檢舉交通違規的功能。而「停車場車位查詢」與題目所述的路況場景無相關性。

A　三軸加速度計
B　血壓感測器
C　紅外線感測器
D　超音波感測器
正確解答為：A　您的作答值為：A　答對了。TNO：O2030034

在病患醫療管理方面，不包含下列哪一個系統？
A　電子病歷（Electronic Medical Records, EMR）
B　電子診斷系統（Electronic Diagnosis System, EDS）
C　電子健康紀錄（Electronic Health Record, EHR）
D　雲端醫療資訊系統（Hospital Information System Cloud, HIS Cloud）
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2030035
題目解析：
在病患醫療管理方面，有效的整合管理病患的醫療資訊，如就診資料、健康紀錄及其日常生活的生理資訊等，可分為電子病歷（Electronic Medical Records, EMR）、電子健康紀錄（Electronic Health Record, EHR）、雲端醫療資訊系統（Hospital Information System Cloud, HIS Cloud）等三大核心技術。
6
警車於公路上追捕搶劫現行犯，且道路前方交通信號燈的紅燈已全部亮起，一旦警車靠近交通信號燈，該方向之號誌將會自動轉為綠燈，關於交通信號燈，下列敘述哪一項正確？
A　交通信號燈一定是警察事前通知交通局，並透過各地交通局的人工控制後，燈號才會改變
B　交通信號燈內嵌三軸加速度計，可監控警車與逃犯的車速
C　交通信號燈的管制系統提升警方辦案速度
D　此系統僅可用在警車上
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2030036
題目解析：
「交通信號燈一定是警察事前通知交通局，並透過各地交通局的人工控制後，燈號才會改變」：透過物聯網感知層與網路層技術的即時感知、傳遞訊息、智能控制，即可達到此應用情境，至於原文所述之過程，勢必浪費不少時間，甚至可能導致一連串難以想像的交通災害；「交通信號燈內嵌三軸加速度計，可監控警車與逃犯的車速」：必須透過具有測量距離功能的感測器（例如：超音波感測器），方可測量距離與速度，三軸加速度計無相應功能；「此系統僅可用在警車上」：此系統亦可使用在救護車或其他車輛緊急事件的應用中。

智慧型影像檢索系統可透過影像辨識與聯網技術，幫助警員節省大量的搜尋時間，提升辦案效率，下列哪一項不是智慧型影像檢索系統之應用？
A　透過影像辨識技術分辨失竊的車體顏色與車牌字樣
B　透過影像辨識技術確認逃犯長相
C　透過影像辨識技術及道路上多個攝影機，追蹤歹徒的行車路線
D　透過影像辨識技術，了解物流車所載送的骨董畫品，其溼度保存狀態
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2030037
題目解析：
「透過影像辨識技術分辨失竊的車體顏色與車牌字樣（檢索車輛特徵）」、「透過影像辨識技術確認逃犯長相（檢索人臉特徵）」、「透過影像辨識技術及道路上多個攝影機，追蹤歹徒的行車路線（檢索車輛特徵）」。以上三個選項，皆為智慧型影像檢索系統之相關應用，而濕度目前無法使用影像辨識技術取得，必須使用濕度感測器。

為了確保市民的行車安全，阿勇伯自行開發了一套物聯網行車應用系統。該系統可透過網路即時調閱各個路口監視器（所有監視器皆具有車牌辨識功能），並維護多台車輛間的通訊紀錄與位置資訊（多台車輛與系統間具無線寬頻連網與衛星定位系統功能），若該系統可順利推展給警方使用，無法落實下列哪一種行為？
A　警車資訊整合
B　家暴事件通報
C　歷史軌跡與即時追蹤
D　可疑車輛與活動範圍
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2030038
題目解析：
透過據車牌辨識功能（感知層）的監視器、監視器間的資訊融合（網路層）、多台車輛與系統間具無線寬頻連網與衛星定位系統功能（網路層），可發展與道路交通相關的物聯網智慧應用。而「家暴事件通報」明顯落於此範圍之外。

A　中央控制系統
B　灑水系統
C　空調系統
D　報表系統

關於物聯網於健康照護的應用，下列敘述哪一項錯誤？
A　醫院透過三軸加速度計劇烈變化的數據，推論老人可能跌倒
B　智慧藥盒的主要目的為提醒病人吃藥，並檢測服藥行為是否正確
C　病人可利用眼壓感測隱形眼鏡來改善近視
D　醫院在病人的股動脈中植入微血壓感測器，可隨時檢驗病人血壓
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2030040
題目解析：
眼壓感測隱形眼鏡，透過鏡片中壓力感測裝置，可以紀錄到角膜壓力和眼球液壓的變化，但無法藉由配戴隱形眼鏡來改善近視狀況。

下列哪一項不屬於健康醫療照顧的感測器？
A　血壓感測器
B　三軸加速度計
C　心跳感測器
D　霍爾感測器
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2030041
題目解析：
健康醫療照護應用主要用於人體與心理健康層面，血壓感測器可偵測人體血壓健康、三軸加速度計可偵測跌倒行為、心跳感測器可偵測人體心跳次數與頻率，而霍爾感測器為感測磁場變化，與人體/心理健康層面無相關特性。
