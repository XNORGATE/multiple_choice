下列哪一種電源模組的優點為體積小、放電穩定且可長時間保存，但缺點是含有重金屬汞且價格高？
A　水銀電池
B　鎳氫電池
C　鋰電池
D　鉛酸電池
正確解答為：A　您的作答值為：A　答對了。TNO：O2060047
題目解析：
水銀電池即為常見的鈕扣電池，它體積小、放電穩定且可長時間保存，常用於手錶、遙控器、計算機中，但是缺點為內含重金屬汞、價格昂貴、無法充電重複使用。

智慧型感測器通常使用下列哪一種電源模組，因其擁有較高電容量，且具有較低記憶效應以及低環境污染等特性？
A　鎳氫電池
B　水銀電池
C　鉛酸電池
D　太陽能電池
正確解答為：A　您的作答值為：A　答對了。TNO：O2060048
題目解析：
相較於水銀電池，鎳氫電池擁有較高的電容量，並且有較低記憶效應（當電池多次在沒有完全把電用完，且又再次充電時，會使充電電池容量減少的現象）以及低環境汙染。

下列哪一項是專為無線感測電路所開發出來的嵌入式作業系統？
A　TinyOS
B　Linux
C　Unix
D　OS2
正確解答為：A　您的作答值為：A　答對了。TNO：O2060049
題目解析：
為了讓感測器中各個模組能夠彼此協調工作，部分感測器需有作業系統才能運作，而在感測器所使用的作業系統中（不同於一般的Linux、Unix和OS2），TinyOS為一套已被廣泛使用的作業系統，該系統所使用的開發語言是nesC。
