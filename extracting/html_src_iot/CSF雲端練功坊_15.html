下列哪些是第三方支付想解決的問題？(複選題)
A　雙方契約無法同時履行
B　缺乏信任基礎
C　商家有寄出商品卻收不到貨款的風險
D　消費者網路購物可以分期付款
正確解答為：ABC　您的作答值為：A　回答錯誤了。TNO：O2040016
題目解析：
目前的第三方支付之所以盛行，是因其可解決現金交易與線上購物長久以來的雙方契約無法同時履行、缺乏信任基礎、商家有寄出商品卻收不到貨款的風險等問題。

下列哪一項不是物聯網收集巨量資料並分析之後，進行決策的案例？
A　農業上收集土壤的溼度，用以調節灌溉的水量
B　光感測器感應到物體，自動打開電燈
C　智慧電網中透過各類環境感測器、智慧電表，了解用電量後針對發電及配電做良好的調度
D　老年人量測心跳、血壓，上傳至雲端紀錄健康資訊
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2040017
題目解析：
運用智慧電網中的各種元件收集用戶端的用電資訊，可以提供公共事業智慧化配電管理的依據，減少不必要的電力耗損。在智慧醫療的方面，老年人可以量測心跳、血壓，上傳至雲端紀錄健康資訊，自動化的管理降低了人為錯誤的機率提升了病患的安全性，再者運用各種感測元件使得病人的生理資訊能夠隨時被醫院監控，可以更輕鬆地處理突發事件的產生。在智慧農業上，可以透過收集土壤的溫度與溼度等資訊，用以調節灌溉的水量、空調的調整、燈光的調整等。選項B的光感測器感應到物體就自動打開電燈，屬於自動化處理，不須經過巨量資料分析。

B　需求即用

下列哪一項不是雲端運算的相關技術？
A　集中式計算
B　虛擬化技術
C　網際網路技術
D　自主運算
正確解答為：A　您的作答值為：A　答對了。TNO：O2040020
題目解析：
雲端運算快速的進展，主要來自於許多技術的進步，特別是硬體（例如虛擬化技術、多核心CPU）、網際網路技術（例如Web服務、服務導向架構、Web 2.0）、分散式計算（例如公用計算、網格計算）和系統管理（例如自主運算、資料中心自動化）。

下列哪一項不屬於雲端運算中基礎設施即服務（IaaS）的特點之一？
A　提供虛擬主機
B　提供多媒體服務
C　提供儲存裝置
D　提供負載平衡
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2040021
題目解析：
雲端運算的最底層所提供的服務，即為基礎設施即服務，其主要是提供計算、儲存裝置、通訊等資源的虛擬化或虛擬機。雲端經營的業者，可以透過資源池的管理工具依據使用的需求來動態調整虛擬機的數量和運作。
7
下列哪一項服務不屬於雲端運算中的軟體即服務（SaaS）之一？
A　Salesforce.com
B　Google Apps
C　Google App Engine
D　Microsoft Office 365
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2040022
題目解析：
在雲端服務的最上層服務，就是軟體即服務。在軟體即服務中主要是將應用程式在雲端上安裝和執行、而雲端的使用者透過終端使用軟體，而無須管理相關的基礎設施和平台。軟體即服務的例子有Google Apps、Microsoft Office 365、Salesforce.com、Quickbooks Online等。Google App Engine主要提供程式執行與開發的平台，屬於平台即服務（PaaS）類型。

下列哪一項屬於雲端運算中的平台即服務（PaaS）之一？
A　顧客關係管理
B　伺服器
C　防火牆
D　程式語言的執行環境
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2040023
題目解析：
在雲端運算的第二層服務，就是平台即服務。在平台即服務中主要是提供開發的平台，包括作業系統、程式語言的執行環境、資料庫、網頁伺服器等。

A　公有雲
B　私有雲
C　混合雲
D　社區雲

A　IrDA
B　Wi-Fi
C　RFID
D　Bluetooth
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2040025
題目解析：
為了有效管理種類及數量繁多的貨物，當物流業者從各方廠商收取到大量的貨物時，便在其中裝置RFID辨識標籤，並透過網路與系統的連線將貨物的儲存位置、庫存數量、運送狀況上傳至雲端的資料庫中，使得管理上能更有效率及便利性。

C　OTS（Online Tracking Systems）

將物聯網技術應用於倉儲管理，實現倉庫的存貨、盤點、取貨的自動化操作，從而提高作業效率，是屬於下列哪一種技術？
A　CRC（Cyclic Redundancy Check）
B　HC（Hamming Code）
C　EPC（Electronic Product Code）
D　ECC（Error Correction Code）
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2040030
題目解析：
透過物聯網EPC（Electronic Product Code）技術的應用，可以達到倉儲管理，實現倉庫的存貨、盤點、取貨的自動化操作，從而提高作業效率。CRC、HC、ECC皆是用來檢查錯誤的機制。
