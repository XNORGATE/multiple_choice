目前國際商品條碼GS1主要由下列哪兩大系統整合而成？(複選題)
A　通用產品代碼（UPC）
B　歐洲商品條碼（EAN）
C　美國統一編碼協會（UCC）
D　食品工業統一碼（UGPIC）
正確解答為：BC　您的作答值為：A　回答錯誤了。TNO：O2060001
題目解析：
為了推動自動識別與電子商務標準的兩大組織達到真正合一，EAN和UCC兩組織於2005年正式更名為GS1（Global Standard No. 1）也就是世界第一標準，立意結合一百多個會員國，領導全球標準之設計與推廣，以促進大眾利益。

磁條識別的三個部分不包含下列哪一項？
A　資料掃描
B　資料確認
C　輸出結果
D　磁性消除
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2060002
題目解析：
磁條識別的三個部分為資料掃描、資料確認、輸出結果。

一般商品包裝上都印製有條碼，台灣的國家代碼為下列哪一項？
A　471
B　886
C　957
D　977
正確解答為：A　您的作答值為：A　答對了。TNO：O2060003
題目解析：
一般商品包裝上都會印製一維條碼，用來代表該商品的出產國、公司、編號等，以台灣來說，印在條碼上的國家代碼維471。886是台灣電話號碼的國際區號。
4
一維條碼敘述，下列哪一項錯誤？
A　水平或垂直的方向都有表達意思
B　結構具備平行黑線和號碼
C　最基本的條碼系統就是一維條碼
D　ISBN碼，屬於一維條碼的範圍
正確解答為：A　您的作答值為：A　答對了。TNO：O2060004
題目解析：
一維條碼僅在水平方向描述訊息，二維條碼則同時利用水平與垂直方向描述訊息。
5
在台灣若有新書出版或新期刊創立應向下列哪一個機關申請ISBN與ISSN代碼？
A　國家圖書館
B　中央研究院
C　台北市立圖書館
D　中華民國商品條碼策進會
正確解答為：A　您的作答值為：A　答對了。TNO：O2060005
題目解析：
ISBN與ISSN的條碼編號申請是獨立於國家EAN編號系統的，不過1991年國際標準書號總部為提倡圖書與期刊條碼化，函告各出版社，其出版品的ISBN與ISSN可併入EAN系統，不必再向該國EAN負責機構申請條碼編號，也不需要再付任何費用。目前在台灣，負責ISBN與ISSN申請的機構是中央圖書館，若有新書出版或新期刊創立，皆須向國家圖書館國際標準書號中心申請。

二維條碼依其結構可分為下列哪兩大類型？(複選題)
A　堆疊式
B　矩陣式
C　點陣式
D　混合式
正確解答為：AB　您的作答值為：A　回答錯誤了。TNO：O2060006
題目解析：
為了提高資料密度，又要在一個固定面積上印出所需資料，因此在一維條碼的基礎上向二維條碼方向擴展以及利用圖像識別原理，採用新的幾何形體和結構設計出二維條碼。發展出堆疊式（Stacked）二維條碼及矩陣式（Matrix）二維條碼，構成現今二維條碼的兩大類型。

目前我國政府經濟部商業司主要推廣下列哪一種二維條碼？
A　PDF417
B　Datamatrix
C　Maxicode
D　Code1
正確解答為：A　您的作答值為：A　答對了。TNO：O2060007
題目解析：
PDF417、Datamatrix、Maxicode三種二維條碼為美國自動辨識協會（AIM）的標準，其中PDF417較早成為標準並商品化，也被政府經濟部商業司所採用並推廣。
8
歐洲商品條碼（EAN）為防止讀取資料錯誤情形發生，必須有一檢查碼，它位於歐洲商品條碼（EAN）中的哪一個位置？
A　最右邊
B　最左邊
C　中間
D　上面
正確解答為：A　您的作答值為：A　答對了。TNO：O2060008
題目解析：
EAN碼（European Article Number）的全名為歐洲商品條碼，源於西元1977年，由歐洲十二個工業國家所共同發展出來的一種條碼。EAN碼比較特別的是，當初所設計的目的，就是希望可以達到雙向掃描處理，因此為了防止錯誤讀取資料的情況發生，在EAN最右邊的位置，加上了檢查碼的設計。

QR Code最早被日本應用於下列哪一種產業？
A　汽車業
B　食品業
C　家電業
D　電子業
正確解答為：A　您的作答值為：A　答對了。TNO：O2060009
題目解析：
QR Code（Quick Response Code）也是二維條碼的一種，於1994年由日本DENSO WAVE公司發明。QR Code最常見於日本，為目前日本最流行的二維空間條碼，最早應用於汽車業的行銷部門。和普通條碼相比，QR碼可以儲存更多資料，也無需要像普通條碼般在掃描時需要直線對準掃描器。

主動式RFID標籤由下列哪些元件所組成？(複選題)
A　天線
B　晶片
C　電池
D　讀取器
正確解答為：ABC　您的作答值為：A　回答錯誤了。TNO：O2060010
題目解析：
電子標籤具有唯一的電子代碼，可貼附在物體上以識別目標物件，標籤包含天線、晶片及電力來源三部分。標籤內的晶片包含了四種主要元件：邏輯單元、記憶體、電源控制器和調變電路，主要就是用來計算傳送和接收所需要的內容。

無線射頻辨識敘述，下列哪一項錯誤？
A　一種無線通訊技術
B　RFID標籤在讀取上並不受尺寸大小與形狀的限制
C　可以透雪、霧、冰、塗料、塵垢等惡劣環境來閱讀標籤
D　相容性高
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2060011
題目解析：
射頻識別技術目前面臨最大的挑戰在於相容性，因為射頻識別系統的主要廠商提供的都是專用系統，導致不同的應用和不同的行業採用不同廠商的頻率和協議標準，這種混亂和割據的狀況已經限制了整個射頻識別行業的增長。

在物聯網的基本架構中，用來感受外界各種刺激並即時做出反應的技術，對應到下列物聯網的哪一層？
A　應用層
B　網路層
C　感知層
D　實體層
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2060012
題目解析：
感測器技術就是對應到物聯網的感知層，感測器就是用來感受外界各種刺激並即時做出反應的技術，也相當於人類的五官。
13
RFID的通訊原理是依據電場和磁場的變化來產生電能，可分為電磁感應以及下列哪一項？
A　微波共振
B　核磁共振
C　電子共振
D　原子共振
正確解答為：A　您的作答值為：A　答對了。TNO：O2060013
題目解析：
RFID的通信原理是依據電場和磁場的變化來產生感應電能，可分為電磁感應（傳輸距離較短，僅限于10公分以內，常見的非接觸式的智慧卡就是采用此原理）及微波共振（傳輸距離長，可達15公尺，資料傳輸頻寬較高，傳輸效果佳）兩種。

A　讀取器
B　應答器
C　收發器
D　感應器

題目解析：
RFID系統主要是由讀取器（Reader）、電子標籤（Tag）、天線（Antenna）以及應用系統（Application System）所組成。
15
僅針對單一使用者口音做少量的詞彙指令，用戶須預先錄製語音標籤，以特徵比對的方式進行識別並執行對應動作，此是指語音識別的下列哪一個特性？
A　特定人指令辨識
B　不特定人指令辨識
C　不特定人大量詞彙辨識
D　不特定人的連續語音辨識
正確解答為：A　您的作答值為：A　答對了。TNO：O2060015
題目解析：
特定人指令辨識：僅針對單一使用者口音做少量的詞彙指令，用戶須預先錄製語音標籤（Voice Tags），以特徵比對（Pattern Match）的方式進行識別並執行對應動作。其優點在於無語言限制且準確率高，缺點則是使用者負擔大，支援指令數少。
