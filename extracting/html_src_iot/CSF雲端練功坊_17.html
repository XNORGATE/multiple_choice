行動支付主要搭配下列哪一項物聯網技術？
A　NFC
B　IrDA
C　ZigBee
D　Bluetooth
正確解答為：A　您的作答值為：A　答對了。TNO：O2040046
題目解析：
透過NFC技術，智慧型手機即可提供行動支付服務。Bluetooth、IrDa、ZigBee皆為資料傳輸技術。
2
隨著物聯網技術的演進，行動支付也越來越受歡迎，關於行動支付的應用，最常發生在下列哪一項生活需求中？
A　悠遊卡通關與零售店購物
B　家電控制
C　身分識別
D　門禁控制
正確解答為：A　您的作答值為：A　答對了。TNO：O2040047
題目解析：
行動支付在不需使用現金、支票或信用卡的情況下，消費者可使用智慧型手機支付各項服務或數位及實體商品的費用。在生活需求中，行動支付的應用最常發生在零售店購物或悠遊卡通關的應用上。
3
NFC技術搭配行動裝置的應用越來越受歡迎，關於NFC應用，下列敘述哪一項錯誤？
A　NFC技術不僅搭載的應用平台越來越多，應用的方法和模式也在不斷擴展，特別是利用手機付款的領域
B　使用者透過NFC手機輕觸海報，可以獲得海報的網頁資訊
C　在台灣，透過NFC技術，消費者只要持悠遊卡輕觸感應區，就能消費與購物，較以往攜帶零錢的方式更為方便
D　由於NFC的技術瓶頸，使用者只能使用NFC手機單向傳輸，無法彼此互傳檔案或相片資料
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2040048
題目解析：
智慧型手機已經能透過NFC技術實現雙向傳輸，將兩台具備NFC功能的設備的互相連接，可達到設備的資訊互通的目的，例如朋友彼此互傳名片、檔案或相片等資料。

NFC技術已經普遍應用於生活當中，下列那些項目屬於NFC的應用範圍？(複選題)
A　行動支付
B　數據交換和傳輸
C　電子票券
D　休閒娛樂
正確解答為：ABCD　您的作答值為：A　回答錯誤了。TNO：O2040049
題目解析：
NFC技術已經普遍應用於生活當中，如行動支付、數據交換、傳輸、電子票券及休閒娛樂等應用皆可使用NFC技術。
5
NFC技術已經普遍應用於生活當中，關於其規格和技術，下列敘述哪一項錯誤？
A　NFC主要是短距離無線通訊，又稱近場通訊
B　能夠讓設備進行非接觸式點對點通訊、讀取和寫入，但無法模擬非接觸式RFID標籤系統
C　NFC可視為手機內SIM卡的另一個延伸介面
D　此技術是由好幾間廠商共同研發
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2040050
題目解析：
距離無線通訊（Near Field Communication ，簡稱NFC），是由Nokia、Philips和Sony等多間廠商共同研發的一種短距離的高頻無線通訊技術，又稱近場通訊，它能夠讓設備進行非接觸式點對點通訊，讀取／寫入非接觸式卡，並在安全晶片的配合下，可以模擬非接觸式RFID系統，NFC可視為手機內SIM卡的另一個延伸介面，SIM卡本質上是一種接觸性的智慧卡，只能與手機機板接觸連接，而NFC技術卻有機會讓SIM卡從原有的接觸介面，以無線的方式再延伸出去。
6
NFC技術已經普遍應用於生活當中，下列哪一項不是NFC可支援的非接觸式RFID標籤？
A　ISO 18092
B　ISO 14443
C　ISO 15693
D　ISO 22000
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2040051
題目解析：
NFC論壇於2006年6月正式對外公佈的NFC技術架構中，最底層為無線射頻層，可適用於各種不同的非接觸RFID標籤，目前主要支援的三大通訊協定為ISO 18092、ISO 14443、ISO 15693。ISO 22000則為食品安全管理系統標準。
7
NFC技術已經普遍應用於生活當中，當消費者使用NFC於商業購物時，NFC會啟動行動支付的功能，此時NFC手機主要是扮演下列哪一種應用角色？
A　NFC裝置扮演卡片的角色，自我模擬成一張相當於RFID系統的IC智慧卡
B　NFC裝置啟動數據交換功能，只是傳輸距離較短
C　NFC裝置扮演讀取器的角色，用來讀取對方的IC智慧卡資料
D　NFC裝置扮演隨身碟的角色，把對方銀行卡的數據讀到隨身碟中
正確解答為：A　您的作答值為：A　答對了。TNO：O2040052
題目解析：
目前NFC技術較多應用在於讓消費者能夠使用NFC直接進行商業購物，此時，NFC手機會啟動行動支付的功能並且讓自己進入所謂的卡片模擬模式，在此模式中，NFC 裝置扮演卡片的角色，自我模擬成一張相當於RFID系統的IC智慧卡（Smart Card），在此模式下，NFC裝置便可替代目前大量使用的IC卡（包括信用卡）、商場刷卡、公車卡、門禁管制卡，車票卡，電子門票等等，對於個人使用非常便利。

NFC技術已經普遍應用於生活當中，消費者使用NFC於商業應用的最大優點，應為下列哪一項？
A　相容度高，不但可取代信用卡、IC智慧卡甚至相容於RFID高頻標籤
B　省電，可以使用手機的電池
C　多樣性，可以從手機執行多樣性的軟體
D　效率高，只要靠近就可以執行對應的動作
正確解答為：A　您的作答值為：A　答對了。TNO：O2040053
題目解析：
目前NFC技術較多應用在於讓消費者能夠使用NFC直接進行商業購物，此種使用方式的極大優點，那就是相容度高，不但可取代信用卡、IC智慧卡甚至相容於 RFID高頻標籤。

關於NFC應用領域，下列敘述哪一項錯誤？
A　洗衣機內建NFC功能，當NFC Tag內建在衣服鈕扣內，這樣洗衣機就可根據衣服材質來設定洗衣模式
B　若家電壞了，可用NFC手機感應其機器的錯誤碼，並透過5G或Wi-Fi發送資料給維修站
C　NFC電子門鎖，可以用手機去感應開門，出門就不用帶鑰匙
D　NFC鍵盤，桌上型電腦可以透過NFC介面連結鍵盤，省略鍵盤接線的問題
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2040054
題目解析：
隨著智慧手機（Smart Phone）的普及和發展，具有NFC 功能的手機越來越受到市場的青睞，也因此發展了許多NFC的應用，如洗衣機內建NFC功能，當NFC Tag內建在衣服鈕扣內，這樣洗衣機就可根據衣服材質來設定洗衣模式；若家電壞了，可用 NFC手機感應其機器的錯誤碼，並透過5G或Wi-Fi發送資料給維修站；NFC電子門鎖，可以用手機去感應開門，出門就不用帶鑰匙等方便的應用。目前NFC技術雖已有應用於鍵盤中，但由於通訊距離限制，目前NFC鍵盤主要應用於智慧型手機中，或僅用於藍牙鍵盤的簡易配對。

目前智慧型手機在支援商業上的應用中，以QR Code最為風行，關於QR Code的應用，下列敘述哪一項錯誤？
A　使用者以QR Code取代實體票券，電影開演前於入口掃描手機QR Code，快速驗票入場，大幅提升消費者便利性
B　使用者的家電若壞了，可用手機掃描保固卡上的QR Code，並可透過5G或Wi-Fi發送資料給維修站，可以簡化報修流程
C　使用者可以透過手機掃描QR Code來登入需要保密性的軟體或門禁設施，可以省去使用者記憶密碼的麻煩
D　QR Code必須透過專業廠商產生，無法由個人自行產生
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2040055
題目解析：
QR Code掃描功能為目前智慧型手機在支援商業上的應用中最為風行的技術，QR Code有其標準化編碼模式，遵照其格式大家都可以自行產生QR Code，亦有相當多工具或網站可協助使用者產生客製化的QR Code，使用者可透過QR Code取代實體票券，例如透過手機購買電影票或高鐵票，即可於入口處掃描手機上的QR Code快速驗票入場、使用者的家電若壞了，可用手機掃描保固卡上的QR Code，即可發送維修資訊給維修中心。使用者可以透過手機掃描QR Code來登入需要保密性的軟體或門禁設施，省去記憶密碼的麻煩。

將物聯網用於行動支付的商業應用時，最重要的技術就是使用者的身分識別，下列哪一項技術不適合拿來搭配行動裝置，以作為身分辨識使用？
A　RFID
B　NFC
C　Bluetooth
D　IrDA
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2040056
題目解析：
將物聯網用於行動支付的商業應用時，最重要的技術就是使用者的身分識別，目前已有許多行動裝置搭配NFC、RFID、Bluetooth等技術作為身分辨識使用。紅外線之所以不適合用來做為身分辨識主要有兩點，其一是紅外線傳輸在安全性的控制上明顯不足，其二是傳送資料時兩個紅外線介面必須正對（Line-of-Sight），中間不能有任何阻擋，因此在使用上略嫌不便。
12
EPCglobal是歐洲盛行的物流標準，EPCglobal可以跟物聯網的技術結合的原因為下列哪一項？
A　因為EPCglobal網路使用產品電子碼（EPC碼）、無線射頻識別技術與資訊網路等科技，可以提供物聯網資訊交換的平台
B　因為物聯網需要更多的業界標準
C　因為EPCglobal和物聯網一樣都是歐洲最喜歡的科技技術
D　因為EPCglobal和物聯網都一樣是以網路為傳輸的媒介
正確解答為：A　您的作答值為：A　答對了。TNO：O2040057
題目解析：
物聯網的目的是要讓任何物品能夠隨時的互相溝通，為了能夠與物聯網技術結合，EPCglobal 網路使用產品電子碼（簡稱EPC碼）、無線射頻識別技術（Radio Frequency Identification, RFID）與資訊網路等科技，建立一個RFID全球標準架構，為供應鏈追求自動化、提供高效率和資訊準確性的物件資訊交換平台。
13
EPCglobal是歐洲盛行的物流標準，關於EPCglobal的說明，下列敘述哪一項錯誤？
A　在EPCglobal裡，最重要的概念是結構框架，內容包括軟體、硬體、資料標準以及核心服務
B　目標是希望經由EPC碼之使用，強化商業流和電腦應用的共同目標，達成有效供應鏈管理
C　共分為三層：網路層、實體層以及硬體層
D　產品主要以EPC碼為主
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2040058
題目解析：
EPCglobal的結構框架共分為三層：Identify識別層、Capture擷取層以及Exchange交換層。
14
在EPCglobal的結構框架中，主要分為三層架構，關於三層架構，下列敘述哪一項錯誤？
A　Identify識別層：在本層中，EPCglobal定義了物件識別標準，以便進行物件識別和交換流程的執行
B　Capture擷取層：使用者在自己的應用範圍內為新物件定義EPC碼，或讀取EPC碼以追蹤其他物件的移動
C　Exchange交換層：EPCglobal結構框架定義出EPC的資料交換標準，提供方法讓終端用戶、約定用戶或是一般大眾分享EPC資料
D　Application軟體層：由EPCglobal定義了軟體介面，讓不同的廠商可以在商業應用時更方便的交換數據
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2040059
題目解析：
EPCglobal 的結構框架共分為三層：Identify識別層、Capture擷取層以及Exchange交換層。並不包含軟體層。
15
在商業應用的EPC編碼標準中，下列哪一種編碼標準是不存在的？
A　序列化全球交易品項識別碼（SGTIN）
B　通用識別碼（GID）
C　序列化運送容器碼（SSCC）
D　美國資訊交換標準代碼（ASCII）
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2040060
題目解析：
EPC碼被設計為支援各種產業中不同識別需求的編碼方案，不同的編碼方案對應不同的領域；編碼包括：通用識別碼（GID）、序列化全球交易品項識別碼（SGTIN）、序列化運送容器碼（SSCC）、全球位址識別碼（GLN）、可回收資產全球識別碼（GRAI）、個別資產全球識別碼（GIAI）、全球服務關係碼（GSRN）、全球文件類型識別碼（GDTI）、以及美國國防部專用的DOD結構碼。ASCII是基於拉丁字母的一套電腦編碼系統。
