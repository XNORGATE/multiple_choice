關於物聯網於鐵道交通之應用，下列敘述哪些正確？(複選題)
A　列車接近路口，柵欄自動放下
B　於鐵路旁安裝震動感測器，感測地震，使列車在地震時自動停下
C　當鐵道上有異常震動時，自動將位置與感測資訊通知管理中心
D　鐵道旁架設隔離網，防止民眾進入軌道
正確解答為：ABC　您的作答值為：A　回答錯誤了。TNO：O2020046
題目解析：
物聯網運用在鐵道交通中能夠提供各式自動化服務，譬如：列車接近路口，柵欄自動放下；於鐵路旁安裝震動感測器，感測地震，使列車在地震時自動停下；當鐵道上有異常震動時，自動將位置與感測資訊通知管理中心。而鐵道旁架設隔離網，防止民眾進入軌道，這不能算是自動化的服務，因而不屬於物聯網在鐵道交通之範疇。

關於物聯網感測器之功能，下列敘述哪一項錯誤？
A　裝設煙霧感測器，自動感應是否有疑似火災的情況發生
B　裝設紅外線感測器，自動感應廁所是否有人進入廁所
C　裝設磁簧開關，自動感應門是否關上
D　裝設超音波感測器，自動感應電腦機械是否故障
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2020047
題目解析：
超音波感測器適用於距離的測量或是判斷是否前方有物體，無法用在判斷電腦是否故障。

關於物聯網於居家監測之應用，下列敘述哪些正確？(複選題)
A　裝設磁簧開關，自動感應所有門窗是否關上
B　設定住戶在家時間，當門窗在異常時間被打開，自動發送訊息到住戶手機
C　裝設紅外線感測器，當有人靠近時會自動打開照明系統並啟動攝影機
D　裝設感測器，當火災發生時，自動灑水
正確解答為：ABCD　您的作答值為：A　回答錯誤了。TNO：O2020048
題目解析：
以上皆為物聯網於居家監測系統所提供之自動化服務。
4
關於物聯網在公共建設節能之應用，下列敘述哪些正確？(複選題)
A　用溫度感測器偵測溫度變化，自動調整體育館的冷氣
B　用紅外線感測器偵測廁所是否有人，自動開關照明系統
C　用光感測器偵測亮度，自動調整房子內的燈光
D　用溼度感測器偵測火災，當火災發生，啟動自動灑水系統
正確解答為：ABC　您的作答值為：A　回答錯誤了。TNO：O2020049
題目解析：
感測火災的發生一般而言是藉由煙霧感測器感測火災濃煙，配合著溫度感測器感測燃燒溫度，方能作為判定火災是否發生的基礎感測元件。而對於濕度感測器一般是用在土壤的濕度是否啟動灑水裝置，或是感測室內溼度，用以判斷是否啟動除濕裝置等。

關於物聯網於門禁管理之應用，下列敘述哪一項正確？
A　用磁簧開關偵測門是否開啟
B　用於開門的電子標籤內含有讀寫器
C　被動式電子標籤可以自行將資料上傳到雲端
D　相較於讀寫器，被動式電子標籤具有較充裕的蓄電能力
正確解答為：A　您的作答值為：A　答對了。TNO：O2020050
題目解析：
電子標籤與讀寫器是彼此獨立的裝置，不包含讀寫器，而被動式電子標籤內並無電池，且無法自行將資料上傳至雲端，必須仰賴讀寫器讀取後透過網路層才能將資料上傳到雲端。

關於物聯網在校園之應用，下列敘述哪一項錯誤？
A　在校園花卉中，以溼度感測器感測土壤溼潤程度，自動啟動灑水器
B　在校園的車道上使用壓力感測器感測汽車速度，作為汽車超速之依據
C　以溫度感測器測量教室氣溫，作為教室自動調節冷氣之依據
D　在校園廁所中，使用紅外線感測器感測學生出入，自動開關電燈
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2020051
題目解析：
壓力感測器可用在被測物給予壓力大小的測量，亦可將壓力大小換作重量，譬如：汽車給予的壓力數值換算為該汽車的重量，因此，壓力感測器無法測量汽車的速度。

關於物聯網在電子道路收費系統（ETC）方面的應用，下列敘述哪一項錯誤？
A　電子道路收費系統的讀取器可以感測電子標籤內所含的內容
B　每輛車上加裝的電子標籤，等於是車子的身分證，可紀錄車子的行走路徑以便導航
C　電子道路收費系統的訊號具可穿透性，因此電子標籤可粘貼於車子裡面
D　安裝在車上的電子道路收費系統的電子標籤是被動式的RFID
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2020052
題目解析：
電子標籤只能作為對汽車的身分辨識，不會紀錄汽車的行走路徑。

下列哪一種感測器，可以用以感測路面顛簸？
A　重力感測器
B　壓力感測器
C　溫溼度感測器
D　光感測器
正確解答為：A　您的作答值為：A　答對了。TNO：O2020053
題目解析：
重力感測器（G-Sensor）又稱為加速度傳感器，可偵測特定方向的加速度，常見為三軸加速度計，可針對物體的位移進行三維空間的加速度測量，因此能夠用於偵測路面顛簸狀況、加減速、離心力、是否發生碰撞等應用。

利用高速公路即時路況系統來規劃行程，避開塞車路段是應用到下列哪些技術？(複選題)
A　雲端資料庫
B　雲端運算
C　無線網路
D　測速技術
正確解答為：ABCD　您的作答值為：A　回答錯誤了。TNO：O2020054
題目解析：
透過架設在高速公路旁的測速器裝置對車輛進行車速的測量，而測量的車速資訊將再透過無線網路上傳至雲端資料庫系統進行雲端運算，以分析、萃取、紀錄與整合出各路段的交通狀態，供用路人參考。

關於電子道路收費系統（ETC）的說明，下列敘述哪一項錯誤？
A　車型辨識系統會將車型及車速資料送至主控系統
B　備有錄影及照相系統，以取得車牌及車體顏色資訊
C　扣款系統是利用微波技術查詢eTag上的資訊
D　系統會即時核對車上人員身分，以計算車輛通行費
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2020055
題目解析：
ETC系統並不會辨識車上人員，且車輛通行費並不是依據車上人員身分，而是依據車輛行駛高速公路的距離、車輛車種等等。

關於物聯網所使用的相關技術，下列敘述哪一項錯誤？
A　智慧電網可改善用電效率，以達到節能省電的目的
B　RFID可用於辨識車牌，強化停車場的管理效率
C　ZigBee是一種低耗能的無線通訊技術，適合用在感測器的資料傳送上
D　Bluetooth是一種長距離的無線通訊技術，用於建築物之間的資料傳送
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2020057

Bluetooth是一種短距離的無線通訊技術，能讓行動裝置，在短距離間交換資料，以形成個人區域網路（PAN）。

隨著物聯網的蓬勃發展，未來住家會越來越便利，下列哪些是智慧家庭可能的應用？(複選題)
A　早上起床時，窗簾自動拉開讓陽光照進來
B　家中溫度過高時，冷氣或電風扇會自動開啟調整環境的溫度
C　洗衣機會根據省電時間自動洗衣
D　有人闖入屋內時，防盜系統會響鈴並通知警方
正確解答為：ABCD　您的作答值為：A　回答錯誤了。TNO：O2020059
題目解析：
選項A為家庭中安裝光感測器的應用，選項B為家庭中安裝溫度感測器的應用，選項C為家庭中運行智慧電網系統，選項D為安裝在家庭大門上的紅外線感測器，若夜晚有人闖入，紅外線感測器將會觸發防盜系統進行響鈴並通知警方。

智慧路燈節能系統沒有透過下列哪一個感測器協同合作？
A　光感測器
B　紅外線感測器
C　三軸加速度感測器
D　壓力感測器
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2020060
題目解析：
智慧路燈節能系統因路燈不會移動，沒有三維空間的加速度測量需求，因此沒有使用到三軸加速度感測器。
