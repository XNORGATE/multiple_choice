下列敘述哪一項不屬於物聯網在社區管理的應用？
A　管理員利用RFID辨識收件者包裹，自動取得住戶地址
B　根據感應磁卡辨識住戶身分，電梯自動選擇樓層
C　使用超音波感測器測量社區噪音，作為判斷音量是否超量之依據
D　使用防盜系統進行偵測，有人闖入時通知住戶手機與管理員
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2020031
題目解析：
超音波感測器適用於距離的測量或是判斷是否前方有物體，無法用在噪音的測量。

下列敘述哪一項不屬於物聯網對生活所帶來的影響？
A　物聯網設備完全無法與舊有電器做結合
B　實現不同控制系統間的信息傳遞與共享
C　減少傳統電器之間電線與控制器的限制
D　提高系統合作間的可靠性及安全性
正確解答為：A　您的作答值為：A　答對了。TNO：O2020032
題目解析：
現有許多智慧設備可讓舊有電器具有聯網能力，使之也成為物聯網中的一員，例如：在傳統設備的插頭與牆壁上的插座間，額外界接智慧電表，透過智慧電表能夠將該電器的電量使用資訊傳遞到雲端中，進行後續的分析與儲存。

全家便利商店自2005年起，就參與工研院環境與能源研究所所開發的網路型分散式能源管理系統設計技術，實際針對便利商店的電力能源，開發出一套可以動態管理耗能設備的能源管理系統，下列敘述哪一項不屬於導入物聯網節能系統的好處？
A　眾多的耗電設備得以節能，節省業者的開支
B　店員的作業操作更加地有效率，無需一一開關調整
C　自動化的評估與分析設備每一時段用電，計算最大效益
D　可以提升店內顧客的流動率，使營業更有效益
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2020035
題目解析：
便利商店導入物聯網節能系統能夠有效管理店內電器相關設備，進而節省能源開支。然而，對於是否能夠提升顧客的流動率，則不是節能系統所探討的範疇。

A　利用感測器和無線通訊技術結合的資訊系統
B　利用大數據的分析達到電量資訊的即時監控功能

關於物聯網應用於居家節能中，下列敘述哪一項正確？
A　物聯網功能的家電，其單位時間的耗電一定比一般家電少
B　當偵測到室內無人時可以自動關閉空轉的家電，減少不必要的浪費
C　所有可以節能的物聯網產品，皆是使用低耗電傳輸模式，絕對不可以使用較高耗電的傳輸方式
D　物聯網產品可以降低電器在單位時間所消耗的功率，以達到省電的效果
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2020037
題目解析：
物聯網技術應用在居家節能中，可透過紅外線感測器判斷家中是否有人員，如一段時間沒有感測到有人員，則發送關閉訊號至冷氣開關控制裝置來關閉冷氣，以減少不必要的電能浪費。

在物聯網之各種通訊協議中，下列敘述哪些正確？(複選題)
A　Wi-Fi是一個建立於IEEE 802.11標準的無線區域網路裝置通訊協定
B　Bluetooth的傳輸速率比Wi-Fi快
C　Bluetooth的頻帶與ZigBee沒有重疊
D　Wi-Fi的傳輸距離比Bluetooth遠
正確解答為：AD　您的作答值為：A　回答錯誤了。TNO：O2020038
題目解析：
藍牙傳輸速度為720kbps（千位元/秒），Wi-Fi則為11mbps（百萬位元/秒），因此，Wi-Fi較藍牙傳輸速度快。Zigbee的頻帶為2.4GHZ，而藍牙的頻帶為2.4至2.485 GHz，因此有頻帶重疊。

關於物聯網應用在電子道路收費系統（ETC），下列敘述哪一項錯誤？

B　使用藍牙接收電子標籤的資料
C　需在收費門架上放置讀寫器，讀取電子標籤的內容
D　電子標籤無需電池就可以運作

下列哪一項應用不屬於物聯網在車載網路之範疇？
A　停車場裝設紅外線感測器，判斷車位是否為空
B　在停車場內，自動將車子目前所在的位置到停車格的指示燈打開
C　塞車在路上，駕駛將塞車情況打電話通知朋友
D　衛星導航根據道路擁塞的情況自動重新規劃路徑
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2020041
題目解析：
物聯網運用在車載網路中能夠提供駕駛自動化的服務，譬如：在開車上路前，智慧型行車記錄器將依據裝設在其他車輛上的智慧型行車記錄器所感知的路況資訊，建議駕駛前往目的地的最佳路線。而使用電話來通知朋友，這不能算是自動化的服務，因而不屬於物聯網在車載網路之範疇。

關於物聯網於家電控制之應用，下列敘述哪些正確？(複選題)
A　藍牙可提供長距離傳輸能力，可在高雄直接控制台北家中的電器
B　如欲用沒有紅外線功能的手機控制冷氣，必須透過多網閘道器
C　在物聯網中，家電可以回傳其狀態給使用者
D　家電之間傳輸資料大部分使用Wi-Fi而不用藍牙，是因為Wi-Fi比較省電
正確解答為：BC　您的作答值為：A　回答錯誤了。TNO：O2020044
題目解析：
欲在高雄控制在台北家中的電器，必須藉由多個網路間的資料交換方能達成，諸如：電信網路（3G、4G）、資料網路（WiFi、藍牙），並非僅靠藍牙就能達成。而就省電而言，藍牙較WiFi來的省電，但傳輸距離則WiFi較藍牙遠。
