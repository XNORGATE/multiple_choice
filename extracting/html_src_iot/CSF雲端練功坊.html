8
下列哪些不屬於感知層的技術？(複選題)
A　ZigBee
B　HSDPA
C　RFID
D　QR Code
正確解答為：AB　您的作答值為：A　回答錯誤了。TNO：O2010008
題目解析：
ZigBee是一種低速短距離傳輸的無線網路協定，底層採用IEEE 802.15.4標準規範的媒體存取層與實體層。High Speed Downlink Packet Access（HSDPA）高速下行封包接入技術是一種行動通訊協定，又稱為3.5G，屬於W-CDMA技術的延伸。Radio Frequency Identification（RFID）無線射頻識別技術、Quick Response Code（QR code）二維條碼辨識技術，兩者皆為物聯網感知層技術。
