關於物聯網的技術，下列哪一項被公認是目前業界最廣泛的應用？
A　RFID標籤技術
B　汽車電子胎壓偵測
C　購物網站人數偵測
D　汽車空氣污染偵測
正確解答為：A　您的作答值為：A　答對了。TNO：O2010016
題目解析：
RFID標籤技術的無線電訊號是通過調成無線電頻率的電磁場，把數據從附著在物品上的標籤上傳送出去，以自動辨識與追蹤該物品。已有許多行業都運用了無線射頻辨識技術，例如：物流追蹤、倉庫管理、牲畜與寵物身份辨識、公司員工門禁管理等等，另外相當普及的悠遊卡、一卡通、iCash等，也都是使用RFID標籤技術。
2
關於物聯網的敘述，下列哪一項錯誤？
A　物聯網（Internet of Things）是網際網路、傳統電信網等資訊承載體，讓所有能行使獨立功能的普通物體實作獨立運作的網路
B　物聯網可以將物體植入各種微型感應晶片，來賦予物體智能化
C　物聯網可透過網路的資料傳輸來進行各項智能服務，例如：內建RFID、G-Sensors、電子羅盤等感測器
D　所謂的物聯網，是我們日常生活中所接觸的物體或設備，有能力連上網路，當很多物體連上網路後，便成為一個具大的物聯網
正確解答為：A　您的作答值為：A　答對了。TNO：O2010017
題目解析：
物聯網是感測器裝備到各種真實物體上，並通過網際網路連接起來，讓所有能行使獨立功能的普通物體實作互聯互通的網路。
3
處在網際網路的世代，我們會說網際網路改變人類的生活模式，主要的原因為下列哪一項？
A　因為為了上網需要購買大量電腦，增加電腦公司的生意
B　因為顛覆了過去的商業行為，上網購物已經是大家習慣的一種消費方式
C　因為可以上網打電動，學生開始不愛運動
D　因為網際網路增加了更多的就業機會
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2010018
題目解析：
隨著網路的進步，從Web 1.0、Web 2.0逐步發展到O2O（offline to online, online to offline），無論是使用者或是商業店家，皆可透過網際網路達到虛實整合，讓上網購物逐漸被大眾接受且習慣，顛覆了過去的商業行為。
4
下列哪一項描述，不存在於網際網路的發展歷史中？
A　全球資訊網是網際網路中最盛行的運用，提供資源儲存與查詢
B　Email及網路電話取代傳統的郵寄信件和電話功能
C　線上地圖讓使用者出門不必帶厚重的旅遊指南
D　線上購物網站，提供了新的人際互動，建立起緊密的人際網路
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2010019
題目解析：
社群網路與即時通訊軟體，提供了新的人際互動，建立起緊密的人際網路。
5
物聯網主要讓物體與物體間依照使用者定義好的規則，來達到資訊獲得的目的，並自動執行任務或使用者的指令，下列哪一項並不屬於相關應用？
A　智能冰箱感知到冰箱內食材不夠時，可以自動上網訂購
B　智能空調可以根據感知室內的人數，來調整冷氣的溫度
C　智能床墊可以感知躺在床上病人的姿勢，藉以通知護士該病人是否需要調整睡姿，以免產生不良影響
D　執行智能搜尋，當使用者在Google網頁上下指令後，便可以搜尋到網路的資料，再轉發給使用者所有的資訊
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2010020
題目解析：
Google網頁搜尋的功能主要針對使用者所輸入的查詢字樣進行網路探勘、比對與回報，其中並未實現物聯網定義物體與物體間的互動規則，因此不屬於物聯網相關應用。
6
在物聯網中，存在著許多新的技術和定義，關於網宇實體系統（CPS）的概念，下列敘述哪一項錯誤？
A　是一種結合運算、通訊和控制於一體的系統
B　該系統強調實體裝置和電腦運算網路的連結
C　該系統可以擔任網路世界和實體世界的橋樑
D　主要目標是利用物聯網來協助照顧人類的健康系統
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2010021
題目解析：
網宇實體系統（Cyber Physical System, CPS），主要強調數位世界（Cyber）與實體世界（Physical）之間的互動，融合實體感測、資訊運算、網路通訊、進階控制等技術，可達到各式各樣的物聯網應用。並不僅止於人類健康系統之應用。
7
組成一個完整的物聯網系統，需要具備三種不同的工作內容，下列哪一項不包含在內？
A　全面感知
B　可靠傳輸
C　即時運算
D　智能處理
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2010022
題目解析：
完整的物聯網系統，需要具備全面感知（感知層）→可靠傳輸（網路層）→智能處理（應用層）。
8
關於物聯網基本架構的敘述，下列哪一項錯誤？
A　感知層有如人體結構中的神經，使智慧物件具有聯網能力，能夠將感測資訊傳遞至網際網路
B　物聯網的基本架構可依照不同的工作內容劃分為實體層、感知層、網路層及應用層
C　應用層是物聯網與各領域產業間進行的專業技術融合，並依照各行業或是用戶的需求網路層中的感知數據進行分析處理
D　網路層用以將智慧物件所收集到的數據資料，透過雲端進一步的進行儲存及有效地管理與運算
正確解答為：A　您的作答值為：A　答對了。TNO：O2010023
題目解析：
「網路層」有如人體結構中的神經，使智慧物件具有聯網能力，能夠將感測資訊傳遞至網際網路。
9
物聯網的通訊技術中，主要是以無線短距離通訊為主，其中業界應用最廣泛的就是RFID，關於RFID，下列敘述哪一項錯誤？
A　是一種無線通訊技術
B　通常是由接收器、電子標籤組成的系統
C　在識別的過程中，無需建立機械或光學的接觸
D　最常見的例子就是生活中的大眾捷運悠遊卡
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2010024
題目解析：
無線射頻技術（Radio-frequency Identification, RFID）是一種無線通訊技術，通常是由讀寫器（Reader/Writer）與標籤（Tag）組成的硬體系統。
10
物聯網的通訊技術中，主要是以無線短距離通訊為主，無線感測器網路（WSN）也是其中的一項技術，關於無線感測器網路（WSN），下列敘述哪一項錯誤？
A　是一群布建在空間中、擁有不同感知元件的感測器所組成
B　主要是透過RFID無線感知技術
C　無線感測器網路（WSN）常用於感知環境的變化
D　可藉由布建無線感測器網路（WSN）在工廠四周，用以偵測工廠所排放的二氧化碳或其他空氣的汙染
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2010025
題目解析：
無線感測網路（Wireless Sensor Networks）藉由各種感測元件感知外在環境的物理資訊，具備物聯網感知層的特性，而非藉由RFID無線射頻辨識技術。
11
物聯網是基於嵌入式技術所發展出來的，關於嵌入式技術，下列敘述哪一項錯誤？
A　嵌入式設備的高速發展以及對網際網路的強烈需求，進而催生了物聯網的興起
B　嵌入式設計綜合了微型電腦、感應器和電子電路等技術
C　主要是針對每樣物品的應用而設計出的專用系統
D　嵌入式設備最常用的作業系統是微軟的Windows
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2010026
題目解析：
嵌入式系統是一種為了某一特定功能與需求所設計的系統，在一般消費性電子產品上的嵌入式作業系統有Linux、Palm 與Windows CE；在網路設備上則有VxWorks、pSOSystem3、Nucleus與QNX等。

智慧家電是物聯網中的一項著名應用，關於智慧家電，下列敘述哪一項錯誤？
A　透過嵌入式技術將網路與各種家電結合
B　不同家電透過上網功能可以相互溝通
C　使用者可以掌握每項家電的使用狀況，形成一個家用的小物聯網
D　世界上最大的智慧家電製造商為Google
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2010027
題目解析：
物聯網即為物物相連的網路，透過物聯網技術的發展，各家廠商紛紛在普通家電中賦予額外更多智慧化的功能，如Amazon、Apple、Google、小米、LG等，都是世界知名的智慧家電製造商。
13
物聯網的架構，有感知層、應用層、實體層、網路層，依據使用者的觀點，由上而下的順序，下列哪一項正確？
A　感知層→應用層→實體層→網路層
B　應用層→網路層→感知層→實體層
C　網路層→應用層→實體層→感知層
D　實體層→應用層→感知層→網路層
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2010028
題目解析：
物聯網的架構，依據使用者的觀點，由上（應用）而下（實體）的順序為應用層、網路層、感知層及實體層。
14
目前是網際網路的時代，關於手機上所使用的無線電信網路，下列敘述哪一項錯誤？
A　無線電信網路是由基地台和行動電話設備所組成
B　無線電信網路的1G、2G等指的是數據傳輸量，也就是根據數據的傳輸量來分類
C　使用者不僅可以使用手機進行語音通訊，亦能透過手機使用電子郵件、網頁瀏覽等功能
D　基地台服務範圍內的手機可以用無線的方式與基地台傳輸語音
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2010029
題目解析：
無線電信網路的1G、2G等指的是無線電信發展的時代（Generation），而非數據傳輸的計量單位（Gigabytes）。
15
目前是網際網路的時代，手機的應用也越來越普及，關於無線數據網路，下列敘述哪一項錯誤？
A　無線數據網路在我們生活中已經非常普及，其通訊範圍的大小會使用到不同的通訊技術
B　WiMAX是一種遠距離、高速率的通訊標準，涵蓋範圍可以達48公里左右
C　Wi-Fi則是最常用的標準，通訊範圍可以從10公尺到100多公尺
D　Bluetooth通訊技術可提供至少數百公尺以上的通訊範圍
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2010030
題目解析：
傳統的藍牙技術，傳輸距離大約為30英尺（約10公尺），發展至藍牙4.0規格後，有效傳輸距離可提升至200英尺（約60公尺）。
