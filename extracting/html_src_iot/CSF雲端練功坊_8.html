物聯網可以應用在節能的服務中，下列哪一項不是主要的節能項目？
A　太陽光電的節能設備
B　環境監控
C　空調等的用電設備管理
D　生產馬達的運轉效能
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2020076
題目解析：
運用物聯網技術在節能中，能夠對運轉馬達進行工作排程，提升馬達的運轉效率，但無法提升生產馬達本身設計上的運轉效能。

物聯網和能源監控的結合，是目前最受矚目的一項技術，而中華電信更是當中的先驅者，以中華電信所提供的iEN智慧節能系統來說，下列哪一項不是他們所提供的服務？
A　電費模擬分析
B　用電排程管理
C　用電統計分析
D　家電故障診斷
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2020077
題目解析：
中華電信自行研發的iEN智慧節能系統，可提供多項能源管理服務，包括：電量即時監控、需量控制、程序控制、用電排程管理、警告通報、電費模擬分析、系統管理、用電統計分析與報表。透過這些服務，可建置一個能源管理的公司環境。

假設要監控橋樑的即時安全性，並且建立預警系統，該使用下列哪一種技術？
A　Wi-Fi定位識別技術
B　感測器網路技術
C　Bluetooth傳輸技術
D　RFID物件識別技術
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2020078
題目解析：
欲監控橋樑的即時安全性，須搭配各式感測器（感測器網路技術）協同對橋梁進行整體性的測量，包括：鋼纜是否脫落、鋼索外皮拉扯程度、橋身傾斜狀況。
4
假設要監控橋樑的即時安全性，並且建立預警系統，在現有的感測器技術中，並非所有橋樑的損壞都可偵測，下列哪一種損壞無法被偵測到？
A　鋼纜脫落
B　鋼索外皮扯裂
C　橋身傾斜度
D　橋樑路面破損
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2020079
題目解析：
鋼纜脫落、鋼索外皮拉扯、橋身傾斜度可以搭配各式感測器協同測量，諸如：三軸加速度感測器、壓力感測器、陀螺儀，但對於橋梁路面的破損，無法用感測器感測判斷。

若要比較無線感測器網路（WSN）和傳統有線系統，下列敘述哪些錯誤？(複選題)
A　無線感測器網路因不受到實體配線的侷限，在系統建置上比有線網路系統更為方便
B　無線感測器網路的資料傳輸速率與穩定性比有線網路系統更高
C　無線感測器網路不必考量電量限制，因此可提供比有線網路系統更龐大與複雜的運算資源
D　無線感測器網路可以機動調整感測節點的位置，更符合系統建置彈性
正確解答為：BC　您的作答值為：A　回答錯誤了。TNO：O2020080
題目解析：
無線感測器網路不受到實體配線的侷限，所以在空間的配置上更彈性，但會遭遇空間物體的阻隔影響，使得降低資料傳輸的穩定性與速度。無線感測器大都裝置電池，因此具有電量上的限制，如應用在需要較複雜運算的網路場景，將造成網路生命期下降。

無線感測器網路（WSN）被廣泛應用在智慧居家生活中，是因為無線感測器網路具有下列哪些特點？(複選題)
A　無線感測器節點的售價便宜
B　無線感測器網路可包含不同類型或不同通訊協定的感測器
C　無線感測器的體積微小
D　無線感測器網路支援遠端控制，工程師可以長距離維護系統
正確解答為：ABCD　您的作答值為：A　回答錯誤了。TNO：O2020081
題目解析：
無線感測器的價格低廉且體積微小，因此能夠被大量佈撒在監控區域中進行各種感測應用工作，此外，無線感測器網路可包含不同類型或不同通訊協定的感測器，協同合作共同執行任務，而透過網際網路的傳輸，工程師亦可遠端控制與維護無線感測器的運作。

在智慧居家生活應用中，下列哪一項不是被偵測/監控的主要項目？
A　溫度、溼度和照度
B　室內人員的移動和靜止
C　噪音和震動
D　室內人員吃飯的時間長短
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2020082
題目解析：
利用溫度、溼度與照度感測器監控居家環境舒適度，作為自動化調節冷氣強弱與窗簾開關之依據；利用紅外線感測器偵測室內人員是否正在移動，作為自動化開啟電燈照明之依據；利用噪音與震動感測器監控室內聲音與動作，作為自動化是否需提醒室內人員降低音量之依據。而利用感測器對於室內人員的吃飯時間長短進行偵測，因感測器難以精確地判斷出室內人員吃飯的時間長短，也將無法提供任何自動化服務。

在物聯網的智慧居家生活應用當中，當系統偵測到周遭環境的要素之後，後續的系統會提供下列哪一項服務？
A　推論並提供適切服務，增加人機互動的多樣性
B　紀錄並通知相關的權責單位
C　充分顯示所有的資訊給使用者
D　推送數據到雲端資料庫
正確解答為：A　您的作答值為：A　答對了。TNO：O2020083
題目解析：
將物聯網技術應用在居家生活中將使得居家設備具有人性化的控制能力、能夠自動推論與提供服務，以及增加人機互動的多樣性。

在物聯網的智慧居家生活應用中，下列哪一項不是應用系統所提供的人機互動設計？
A　系統利用溫溼度感測器可以了解目前環境的狀態
B　系統利用RFID技術可以辨識使用者身分

一般來說，物聯網中的智慧交通系統可分為車輛與道路兩個部分，這兩個部分分別有其對應的應用系統，關於這兩部分系統，下列敘述哪一項正確？
A　車輛部分是指車載物聯網電腦，道路部分則是指智慧交通系統
B　車輛部分是指車牌辨識系統，道路部分則是指路面顛簸偵測系統
C　車輛部分是指車輛修護系統，道路部分則是指路面修護系統
D　車輛部分是指車輛銷售系統，道路部分則是指方向標示系統
正確解答為：A　您的作答值為：A　答對了。TNO：O2020085
題目解析：
智慧交通系統可分為車輛與道路兩個部分，在車輛部分指的是裝配在車輛上的智慧型感知行車紀錄器，亦可稱之為車載物聯網電腦，用以記錄車內外行車狀況，並透過網際網路上傳雲端進行分析、儲存及分享；在道路部分指的是智慧交通系統，包括：高速公路即時路況系統、電子道路收費系統、緊急醫療通知系統、先進大眾運輸服務系統與長隧道監控管理系統。

在物聯網所應用的智慧交通系統中，最常搭配的車輛系統是偵測駕駛人的生理狀況，避免駕駛人因身體問題產生行車危險，下列哪一項不是目前使用的生理偵測系統？
A　建置於駕駛員前座的攝影鏡頭，用以擷取駕駛人眼睛狀態，若閉眼超過一段時間，就會發出警示
B　將感測器裝置於安全帶上，用來偵測駕駛人的心跳等生理狀態
C　駕駛座上裝置酒精濃度測試系統，駕駛人必須通過測試才能開車
D　座椅姿勢系統，若坐姿不對則無法開車
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2020086
題目解析：
駕駛的身高與體重不同，以及乘坐的姿勢與習慣亦不相同，因此，無法以駕駛的坐姿是否正確來判定該駕駛是否能開車。

一般來說，人們出門大部分以車為代步工具，為了提升行車時的安全性，行車紀錄器日漸普及，而在目前物聯網的時代，智慧型感知行車紀錄器也變得越來越受歡迎，關於智慧型感知行車紀錄器，下列敘述哪一項錯誤？
A　智慧型感知行車紀錄器可以紀錄行車資訊
B　智慧型感知行車紀錄器也可透過網路傳播得知行駛路況
C　智慧型感知行車紀錄器可以方便得知周遭的環境資訊
D　智慧型感知行車紀錄器不需要連上網際網路，使用非常方便
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2020089
題目解析：
車內外各式感測器的感測資料量大，諸如：圖片、聲音與文字，因此，將無法全部儲存在智慧型感知行車紀錄器中，反而是需要透過智慧型感知行車紀錄器的即時連線上網將資料上傳至雲端伺服器進行分析與儲存，此外，分析後的資料將進一步分享給其他用戶。
15
一般來說，人們出門大部分以車為代步工具，為了提升行車時的安全性，使得行車紀錄器日漸普及，而在目前物聯網的時代，智慧型感知行車紀錄器也變得越來越受歡迎，關於行車紀錄器上可能使用到的感測器，下列敘述哪一項錯誤？
A　利用溫溼度感測器監控車外溫溼度
B　三軸加速度計感測路面平坦度
C　音量感測器感測目前道路音量
D　重量感測器偵測目前車上有多少人
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2020090
題目解析：
乘客體重的不同或是車內載運的貨物重量，皆將影響重量感測器無法精確判斷出車內有多少人。
