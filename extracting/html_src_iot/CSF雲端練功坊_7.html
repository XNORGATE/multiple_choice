物聯網智慧生活整合應用12 / 1580 %
1
我們希望可將物聯網中的技術與生活中的應用充分結合，關於物聯網中結合汽車、交通的應用領域的技術，我們稱之為下列哪一項？
A　車聯網
B　物物聯網
C　交通聯網
D　物流網
正確解答為：A　您的作答值為：A　答對了。TNO：O2020061
題目解析：
車聯網（Internet of Vehicles, IOV）是物聯網在交通領域中的具體呈現，通過與車輛有關的技術、設備（如車輛電子標識、感測器、無線網路通訊、衛星定位和大數據處理等現代資訊通信技術的應用），對所有於網中之車輛、行人和道路基礎設施的屬性、靜態和動態訊息進行有效辨識，並將資料彙整於後端平台進行智慧化管理和服務。
2
我們希望可將物聯網中的技術與生活中的應用充分結合，而車聯網是其中的一項重要應用，關於在車聯網的主要應用中，智慧汽車主要功能為下列哪一種？
A　汽車間可以相互通訊、聊天發信息
B　可以在極短時間內掌控車輛由內至外的每個狀況並即時反應，提升智慧汽車時間與空間的行動力
C　倒車時，可以知道後方的物體
D　導航系統
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2020062
題目解析：
宏達電（HTC）為納智捷汽車所開發的「THINK＋」系統，該系統內建行動通訊模組、NAVI視頻模組，具備影音娛樂、電話通訊、衛星導航、行車安全等功能，可以在極短時間內運算與處理訊息，預知並掌控車輛由內至外的每個狀況並即時反應，提升智慧汽車時間與空間的行動力。
3
我們希望可將物聯網中的技術與生活中的應用充分結合，關於在車聯網的應用中，有一項稱之為智慧交通，其主要的功用為下列哪一項？
A　主動通知乘客捷運或公車下一班車到達的時間
B　提供捷運或公車上影音的3D多媒體功能
C　讓乘客更方便的線上訂票
D　可以更方便的找到上車的地方
正確解答為：A　您的作答值為：A　答對了。TNO：O2020063
題目解析：
先進大眾運輸服務系統，是將物聯網技術應用至智慧公車中，智慧公車配置了各式物聯網感知設施，例如：影像偵測系統、GPS定位系統等各類智慧系統，這些感測到的資料將隨時透過網路回報至資訊系統控制中心，中心接收到公車定位資訊後，計算出公車抵達各個站牌的時間，並透過網路將計算結果傳至各個公車站的智慧型站牌，或是網路上的大眾運輸服務平台，想要搭公車的民眾，可透過智慧型站牌或智慧型手機知道下一班公車的到站時間。
4
我們希望可將物聯網中的技術與生活中的應用充分結合，其中關於物聯網中結合汽車、交通的應用領域的技術為車聯網，根據物聯網的定義，下列哪一項不是車聯網的服務？
A　車商主動通知汽車駕駛驗車的時間
B　倒車時可以警告後方的物體，避免碰撞
C　提供汽車駕駛在行車時，最適時適地的周邊環境資訊與安全服務，例如哪裡有空的停車位等資訊
D　汽車駕駛打瞌睡時，汽車本身會主動發出警告
正確解答為：A　您的作答值為：A　答對了。TNO：O2020064
題目解析：
車商主動通知汽車駕駛驗車的時間，為人工主動通知，並無利用到車聯網中的所提供之自動化服務。
5
物聯網的應用與生活結合一直是大家努力的目標，將物聯網的技術與生活中的計程車業者結合，所提供的雲端車隊管理的車聯網服務，主要的應用是指下列哪一種？
A　企業可以精準的掌握車隊所在，可以動態調派計程車到乘客需要的地方
B　出去玩的時候，彼此可以掌握其他車輛/車隊的位置
C　提供汽車駕駛在行車時的交通路況通知
D　汽車駕駛打瞌睡時，會主動發出警告
正確解答為：A　您的作答值為：A　答對了。TNO：O2020065
題目解析：
選項A相較於B、C、D，較為符合物聯網結合計程車業者所提供之服務，例如計程車叫車服務，乘客手機APP一叫車，計程車公司即可立即透過乘客位置與鄰近計程車狀態，進行派車作業，乘客亦可隨時知道計程車所在位置。
6
物聯網對於物流業也有著很大的幫助，下列哪一項為智慧物流的主要服務？
A　企業可以對其商品從生產、包裝、運輸和倉儲管理、批發零售等環節，進行更完善的掌控和管理
B　顧客買東西的時候，可以知道商品的製作過程
C　廠商可以知道顧客的喜好，提供更好的服務
D　業者可以知道目前產品的獲利狀況
正確解答為：A　您的作答值為：A　答對了。TNO：O2020066
題目解析：
如何讓發貨人、收貨人可以即時獲得每個物流節點的貨物精確訊息；讓配銷、運送商可以更有效率進行貨物運送的監控、倉儲管理、運輸貨櫃管理等，都是物流產業結合物聯網可以達到的目標。
7
企業若導入物聯網到其內部應用中，下列哪一項不是該企業可以直接提升的項目？
A　加快生產速度
B　提高產品品質
C　提升生產線的作業管理機制
D　促進新產品的研發速度
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2020067
題目解析：
選項D的產品研發速度取決於研發人員，與物聯網技術的導入沒有相關。
8
一家企業若要提升本身的價值，需要引進新一代的物聯網庫存管理技術，下列哪一項技術可以用來協助庫存管理？
A　二維條碼加RFID識別技術
B　紅外線
C　影像識別技術
D　衛星定位
正確解答為：A　您的作答值為：A　答對了。TNO：O2020068
題目解析：
當物流業者從各方廠商收取到大量的貨物時，便在其中裝置RFID辨識標籤與二維條碼，並透過網際網路將貨物的儲存位置、產品數量、運送狀況上傳至雲端資料庫中，使企業在管理上更有效率及便利性。
9
導入物聯網技術到庫存管理應用上，主要是希望能夠應用在下列哪一項業務上？
A　導入e化與行動化，藉以提高貨物進出倉庫的流暢度，及更精確掌握庫存量
B　可以馬上得知顧客買的東西數量
C　顧客喜好可以反應回總公司
D　增加進出貨品的速度
正確解答為：A　您的作答值為：A　答對了。TNO：O2020069
題目解析：
庫存管理最重要的即是對產品進出倉庫的流暢度與貨物數量、內容、位置的相關管理。
10
目前物聯網的概念和技術已經普遍使用在物流業上，台北港的物流轉運點，主要是應用下列哪一種物聯網技術？
A　ZigBee
B　藍牙
C　紅外線
D　超音波
正確解答為：A　您的作答值為：A　答對了。TNO：O2020070
題目解析：
台北港貨櫃碼頭導入ZigBee技術，建立智慧港口園區，其內容為將ZigBee技術運用於碼頭作業車輛調派通訊、車輛位置感知、儲貨區報到系統、ARMG無人自動吊車及櫃場管制自動化等作業。
11
一家經營零售業的公司，必須在下列哪一類型的問題上導入物聯網的服務，才能達到永續經營的目標？
A　為了避免存貨不足導致缺貨問題
B　為了避免生產力不足的問題
C　為了避免原料不足的問題
D　為了避免客戶來源不明的問題
正確解答為：A　您的作答值為：A　答對了。TNO：O2020071
題目解析：
如何讓發貨人、收貨人可以即時獲得每個物流節點的貨物精確訊息；讓配銷、運送商可以更有效率進行貨物運送的監控、倉儲管理、運輸貨櫃管理等，都是物流產業結合物聯網可以達到的目標，可以使零售業的公司達到永續經營的目的。
12
若以全球最大零售商Walmart來說，其初期導入最主要的物聯網技術是下列哪一種？
A　RFID標籤
B　條碼
C　來客感知系統
D　線上售後服務系統
正確解答為：A　您的作答值為：A　答對了。TNO：O2020072
題目解析：
全球最大的量販超市Walmart運用了RFID技術，要求其全球前100大上游供應商在貨品的包裝或棧板上嵌入RFID標籤，方能上架，藉此來記錄商品在貨架上的數量，以便追蹤貨品在其供應鏈上的即時資訊，以減少缺貨率及掌握產品庫存資訊的透明度。
13
德國最大的零售商Metro利用RFID和無線區域網路所建構的物聯網服務中，不包含下列哪一項？
A　利用RFID讀出出貨資料
B　利用RFID確認商品被放置在正確的貨架上
C　利用RFID確認庫存的商品數量
D　利用RFID找出商品的瑕疵
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2020073
題目解析：
選項D的答案，其可利用RFID進行對商品的保存期限確認，但無法對商品實際的內容物品質與是否有瑕疵予以判斷。
14
下列哪一項不是未來商店利用物聯網所提供的服務？
A　確認商品的品質和瑕疵
B　在庫存盤點上，只要透過RFID就可即時並精確掌握商品的數量
C　商品上架之後，可以提供消費者商品履歷以及品牌故事等資訊
D　顧客結帳的時候，可利用自動讀取商品，進行快速結帳
正確解答為：A　您的作答值為：A　答對了。TNO：O2020074
題目解析：
透過讀取產品上的RFID標籤，僅可取得產品的相關資訊，例如產地、成分、保存期限等資訊，但無法對產品實際的內容物品質與是否有瑕疵予以判斷。
15
物聯網和能源監控的結合，是目前最受矚目的一項技術，該項技術所標榜的節能服務中，最常用的設備是下列哪一種？
A　智慧電表的電力監控
B　提供網路食譜的智慧冰箱
C　智慧空調
D　智慧洗衣機
正確解答為：A　您的作答值為：A　答對了。TNO：O2020075
題目解析：
欲實現智慧電網，智慧電表的普及為最主要的因素。家庭中各式電器的電量使用狀況皆能夠透過智慧電表進行上傳雲端進行分析與儲存，接著進一步的電量資訊可作為用電戶調節用電的參考，亦可作為電力公司生產電能與配置電量的依據。
