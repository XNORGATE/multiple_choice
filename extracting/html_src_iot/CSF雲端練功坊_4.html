下列哪一項不是智慧電網應用層所提供的服務？
A　提供電量消耗視覺化的服務
B　可用智慧型手機查看目前家電耗電的狀況
C　電量使用資訊可透過網際網路傳至配電者的資料蒐集中心
D　可以控制家電的開關，達到節能減碳的目的
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2020016
題目解析：
電量使用資訊可透過網際網路傳至配電者的資料蒐集中心，其是屬於網路層之範疇。
2
下列哪一項不屬於物聯網在智慧社區的應用？
A　隨時監控地下室的照明設備、通風設備、用水及用電等能源消耗，以達節能目的
B　監控緊急事件的發生，如瓦斯外洩、火災、漏電等事件，並可防範小偷闖入
C　交誼廳、健身房、三溫暖室、會議室及遊戲間等空間的使用，可透過身分辨識系統及預約系統而達到管理及自動收費的目的
D　管理員通知住戶郵件包裹的領取
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2020017
題目解析：
管理員通知住戶郵件包裹的領取，不屬於智慧社區所提供之自動化服務之範疇。
3
物聯網於智慧社區之應用不包含下列哪一項？
A　社區網路配置
B　社區安全監控
C　社區能源管理
D　社區事務管理
正確解答為：A　您的作答值為：A　答對了。TNO：O2020018
題目解析：
社區網路的配置是由電信業者負責，亦非屬於物聯網於智慧社區之應用。

下列哪一項不屬於物聯網技術於智慧家庭的應用？
A　由光感測器所偵測的資訊，調整室內窗簾
B　透過太陽能板儲存電力，提供晚上回來時所需要的電力
C　根據溫溼度感測器的資訊，調整冷氣
D　利用紅外線感測器和攝影裝備監控是否有人進入
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2020019
題目解析：
透過太陽能板儲存電力，提供晚上回來時所需要的電力是屬於智慧電網之範疇。

智慧型恆溫控制器如何利用物聯網的技術，下列敘述哪一項錯誤？
A　藉由使用者對溫度調整的紀錄資料，推算出使用者對空調溫度的喜好與設定習慣
B　可以與智能手環連結，進行資訊交流，更精確的針對個人化的睡眠習慣進行調節溫度
C　可與汽車及車庫大門進行連線，藉由判斷車子的進出，間接得知用戶是否在家，達到防盜的目的
D　智慧型恆溫控制器取代智慧電表，能夠在電力使用的離峰時期，開啟高耗電的家用電器，進而達到節能減碳的目的
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2020020
題目解析：
可與汽車及車庫大門進行連線，藉由判斷車子的進出，間接得知用戶是否在家，以節約能源。
6
下列哪一項不屬於物聯網於智慧電網之應用？
A　即時監控發電廠的發電量，並參考用電量與對環境的影響來自動調整發電量
B　即時監控電網中的電力流向，並對電網中的電力進行最佳化分配
C　在高壓傳輸線路的附近加裝感測器，偵測人類和其他動物，並在緊急狀況下停止電力傳輸，防止可能發生的觸電事件，用以避免人員的傷亡
D　在家中加裝智慧插座，並用手機控制使其可以自動斷電，減少電力消耗
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2020021
題目解析：
智慧電網的發展主要針對電能的管理、整合與配電，其中並不包含防止觸電事件發生。

智慧化家庭的控制系統中，下列哪一項不屬於人性化控制的體現？
A　利用光感測器控制窗簾或是照明系統
B　利用遙控器控制電視音量大小
C　利用溫溼度感測器控制室內空調系統
D　利用紅外線感測器與攝影機控制門禁系統
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2020022
題目解析：
遙控器仍須經由人為控制，並非智慧化家庭中所提供的自動化服務。

下列哪些屬於物聯網的應用？(複選題)
A　智慧家庭
B　智慧醫療
C　智慧交通
D　智慧電網
正確解答為：ABCD　您的作答值為：A　回答錯誤了。TNO：O2020023
題目解析：
智慧家庭將家中電器連結上網以感測器提供自動智能控制；智慧醫療將偵測儀器自動上傳避免人為疏失錯誤；智慧交通運用了物聯網的概念，將路聯網和車聯網的資訊透過網際網路上傳到雲端的資料庫中；智慧電網（Smart Grid）被視為未來電力系統的發展主流。

智慧電網應用層可以提供電量消耗的視覺化服務，下列敘述哪一項不是此服務所帶來的好處？
A　可有效降低電器每單位時間消耗的電量
B　使用者可以透過智慧型手機查看耗電狀況
C　使用者可以透過手機APP介面控制家電開關
D　透過圖表可以分析電器用電的詳細情形
正確解答為：A　您的作答值為：A　答對了。TNO：O2020024
題目解析：
智慧電網應用層可提供用電戶用電資訊查詢服務，諸如：用電高／低峰期、用電減價時段、電量計價方式，以及歷史用電紀錄等，藉以改變用電習慣以達到節能減碳之目的。但無法降低電器每單位時間消耗電量。

下列敘述哪些屬於智慧藥盒提醒系統之功能？(複選題)
A　提供多媒體觸控板的操作功能
B　自動量測血壓血糖的數據功能
C　讓使用者方便辨識什麼時候該吃什麼藥，避免吃錯
D　吃藥時間的提醒與設定功能
正確解答為：ACD　您的作答值為：A　回答錯誤了。TNO：O2020025
題目解析：
智慧藥盒主要作為藥物食用方面之智能管理控制，血壓血糖之測量非屬藥物食用之範疇。
11
下列敘述哪一項不屬於智慧物流之應用？
A　讓發貨人、收貨人可以及時獲得物流節點的精確訊息
B　提供消費者在消費平台上的優惠，以增加人氣
C　讓配銷、運送商可以更有效率地進行貨物運送的監控與管理
D　有效的管理倉儲當中的貨物與配送的時間
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2020026
題目解析：
物流管理系統的發展注重在使發貨人或收貨人可以即時獲得每個物流節點的貨物精確訊息；讓配銷、運送商可以更有效率進行貨物運送的監控、倉儲管理、運輸貨櫃管理。而對於產品優惠與否，與貨物流向無關，因而不屬於智慧物流之應用。

關於RFID讀取器（Reader）用於賣場物流管理，下列敘述哪些正確？(複選題)
A　可以透過裝置RFID標籤將貨品資訊傳至雲端資料庫
B　業者可以在盤點時，感應RFID標籤得知最新的貨物流動情形
C　賣場人員可以透過RFID讀取裝置，得知產品的銷售狀況
D　賣場人員可以透過RFID讀取裝置更改商品RFID Tag的內容
正確解答為：ABC　您的作答值為：A　回答錯誤了。TNO：O2020027

賣場人員需透過RFID讀寫器（Reader/Writer），才能夠對商品上的RFID Tag內容進行更改。

A　整合發電
B　輸送電量
C　分配電量
D　修正家中的電路
正確解答為：ABC　您的作答值為：A　回答錯誤了。TNO：O2020028

下列哪一項設備不適合用於物聯網室內定位？
A　於廁所門上裝設紅外線感測器
B　於天花板上裝設三軸加速度計
C　於地毯中嵌入壓力感測器
D　手機Wi-Fi無線訊號
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2020029
題目解析：
三軸加速度計主要針對物體的位移進行三維空間的加速度測量，若裝設於移動物件上可輔助定位，但若裝設於非移動物件（天花板）則對定位系統沒有任何作用。

下列哪些系統包含在物聯網的智慧家庭應用中？(複選題)
A　自動照明系統
B　火災警示系統
C　門禁系統
D　自動空調系統
正確解答為：ABCD　您的作答值為：A　回答錯誤了。TNO：O2020030
題目解析：
智慧家庭中存在著許多的自動化系統，例如：利用光感測器所偵測的資訊，調整窗簾的自動窗簾系統或燈光的自動照明系統；根據溫溼度感測器的資訊，調節冷氣的舒適度系統（自動空調系統）；利用紅外線感測器和攝影裝備監控是否有人進入的門禁系統；利用熱度感測器配合二氧化碳感測器監控房屋室內是否起火或冒煙的火災警示系統；利用磁簧開關偵測大門是否開啟的門禁系統等等。
