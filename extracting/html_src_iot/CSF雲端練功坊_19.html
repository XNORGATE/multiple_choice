有關ISM頻段敘述，下列哪一項錯誤？
A　不需特別申請，即可使用
B　可以自由免費的使用
C　各國共同的頻段為3.5GHz
D　此頻段專用於工業（Industrial）、科學（Scientific）及醫療（Medical）研究
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2050016
題目解析：
ISM頻段在各國的規定並不一定，在國際電聯無線電規則ITU-R中指定的ISM頻段共有一共有12個頻段，而在美國僅開放902~928 MHz、2.4 GHz~2.483GHz、5.725 GHz~5.875 GHz三個頻段，各國共同的ISM頻段為2.4GHz，因此Wi-Fi、Bluetooth、ZigBee等無線網絡，均可工作在2.4GHz頻段上。

物聯網系統開發時需整合多重異構問題，包括硬體和網路、作業系統和編譯語言的異構等，為了整合這些異構，需透過下列哪一項媒介？
A　整合軟體
B　中介軟體
C　應用軟體
D　系統軟體
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2050017
題目解析：
中介軟體（Middleware）泛指二個不同媒介中間應用於系統整合的軟體，如：服務組合、服務管理、物件抽象化等，中介軟體的存在便是為了解決物聯網中多重異構的問題，包含硬體和網路、作業系統和編譯語言的異構。

A　DIFS（DCF Inter-frame Space）
B　EIFS（Extended Inter-frame Space）
C　PIFS（PCF Inter-frame Space）
D　SIFS（Short Inter-frame Space）
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2050018
題目解析：
等待時間分別為(D)＜(C)＜(A)＜(B)。

雲端運算依照部署方式可分為四種，其中一種雲主要是在一些組織有共同特定的考量（例如：任務、安全要求、政策法規）共享基礎設施，可能由自己、第三方或是外部的供應商來管理，應為下列哪一種雲？
A　私有雲
B　公有雲
C　混合雲
D　社群雲
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2050019
題目解析：
社群雲主要是在一些組織有共同特定的考量（例如：任務、安全要求、政策法規）共享基礎設施，可能由自己、第三方或是外部的供應商來管理。所需花的費用比私有雲少比公有雲多。

D　使用MIMO提升傳送速度與範圍

關於ZigBee的特色，下列敘述哪一項錯誤？
A　安全性高
B　低耗電
C　能擴充大量的網路節點
D　與設備的相容性高
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2050022
題目解析：
Zigbee是一個短距離、低成本、低耗電、安全性高、且能擴充大量網路節點的網路架構。Zigbee在物聯網上的運用非常廣泛，無論在家庭自動化、家庭安全、工業與環境控制、個人醫療照護等，都是Zigbee主要的應用範疇。

藍牙（Bluetooth）網路內的裝置是主僕式（Master-Slave）架構，一個Master最多可以連結的Slave個數，應為下列哪一項？
A　5
B　6
C　7
D　8
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2050023
題目解析：
在藍牙環境中，裝置數目最多為八個且為主僕式（Master-Slave）架構，其中包含至少一個主裝置（Master device）與最多七個從屬裝置（Slave device）所組成的網路。
9
關於藍牙（Bluetooth）裝置形成的Piconet，下列敘述哪一項錯誤？
A　一個藍牙裝置不可以是多個Piconet中的成員
B　多個Piconet形成Scatternet
C　一個藍牙裝置扮演主人（Master）的角色，而其他的藍牙裝置則扮演僕人（Slave）的角色
D　由2至8個藍牙裝置所組成的主僕式（Master-Slave）架構
正確解答為：A　您的作答值為：A　答對了。TNO：O2050024
題目解析：
多個藍牙裝置可形成Piconet，而相鄰的Piconet可由同時扮演Master和Slave角色的Bridge來連結組成較大的Scatternet，一個Scatternet最多可由256個Piconet組成。
10
藍牙的Piconet連接成功後的四種選擇模式，下列哪一種模式下主裝置要定期傳送同步訊息來維持與其他裝置的同步關係？
A　保持模式
B　監聽模式
C　主動模式
D　待機模式
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2050025
題目解析：
主裝置和從屬裝置傳遞資料的一般模式可以用來傾聽、發送及接收封包，在這個模式下主裝置要定期傳送同步訊息來維持與其他裝置的同步關係。在主動模式下的從屬裝置都保有活動成員的位址，以接收主裝置傳送的資料。

關於物聯網中的異質性網路整合，下列敘述哪一項錯誤？
A　可以透過閘道器來解決異質性網路整合的問題
B　異質性網路整合最大的問題為不同的網路使用不同的通訊協定
C　閘道器可以處理不同網路或不同業者物聯網設備之溝通，但無法扮演使用者與物聯網設備的溝通橋梁
D　兩個不同電信業者間的設備可以透過閘道器來溝通
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2050027
題目解析：
閘道器不僅做為網路設備與網路設備之間的溝通橋樑，同時也是使用者與物聯網設備的溝通橋梁。

6LoWPAN RPL（Routing Protocol for LLNs）的路由協定，下列哪一項錯誤？
A　RPL可針對相同的應用設定不同的目標函數（Objective Function）
B　RPL可根據不同的應用建構出不同的DODAG（Destination-Oriented Directed Acyclic Graph）
C　RPL支援多點對點（Multipoint-to-Point）、點對多點（Point-to-Multipoint）及點對點（Point-to-Point）三種主要資料傳輸類型
D　RPL使用距離向量（Distance Vector）及來源路由（Source Routing）兩種運作概念
正確解答為：A　您的作答值為：A　答對了。TNO：O2050030
題目解析：
RPL可針對不同的應用設定不同的目標函數（Objective Function），基本行動台的集合，形成一個擴充服務集合。
