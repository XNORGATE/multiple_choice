下列哪一種長距離低功耗廣域網路（LPWAN）架構中，感測資料必須透過閘道器（Gateway）才能傳送到後端伺服器？
A　LTE-M
B　LoRa
C　NB-IoT
D　5G
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2050046
題目解析：
在LoRa網路架構中，感測資料必須先傳送到LoRa Gateway，再透過LoRa Gateway將資料傳送到後端伺服器（例如網路伺服器，或應用伺服器）。
2
下列哪些物聯網網路層技術使用免付費頻段？(複選題)
A　Bluetooth
B　LoRa
C　NB-IoT
D　ZigBee
正確解答為：ABD　您的作答值為：A　回答錯誤了。TNO：O2050047
題目解析：
Bluetooth、LoRa、ZigBee是使用非授權頻譜，即免付費頻段。NB-IoT使用付費頻段，是由電信業者提供服務，一般是以傳輸的資料量來計費。

下列哪些是物聯網應用層會使用到的通訊協定？(複選題)
A　MQTT
B　UDP
C　CoAP
D　TCP
正確解答為：AC　您的作答值為：A　回答錯誤了。TNO：O2050049
題目解析：
MQTT協定的訊息內容很精簡，適合用在資源有限且頻寬不高的物聯網裝置。CoAP協定使用RESTful架構，是為了機器間的通訊所設計，由於物聯網裝置的資源有限，而且一般傳輸的資料量不大，因此適合使用CoAP當作應用層的通訊協定。TCP和UDP均屬於物聯網網路層的通訊協定。
5
下列哪一種物聯網通訊協定是使用發布（publish）/訂閱（subscribe）機制？
A　CoAP
B　UDP
C　MQTT
D　HTTP
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2050050
題目解析：
MQTT是一種採用發布∕訂閱機制的通訊協定，架構中包含發布者（Publisher）、代理人（MQTT Broker）和訂閱者（Subscriber），Publisher負責發布特定主題（Topic）訊息給MQTT Agent，Subscriber向MQTT Agent訂閱自己有興趣的主題，當MQTT Agent收到Publisher傳送的訊息時，便會根據主題傳送給適當的Subscriber。以一個利用溫度監測的物聯網應用為例，我們可以把溫度感測器視為Publisher，接收感測器資料的裝置（例如：監控人員的手機）視為Subscriber，並可使用一台伺服器當作MQTT Agent。一旦收到溫度感測器所傳送的溫度數據時，伺服器便會將這些數據傳送給手機，因此監控人員便可隨時知道感測區域中的溫度狀況。
