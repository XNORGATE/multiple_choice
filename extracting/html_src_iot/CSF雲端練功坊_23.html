NFC的通訊原理與下列哪一項技術原理相似？
A　RFID
B　Bar Code
C　QR Code
D　Bluetooth
正確解答為：A　您的作答值為：A　答對了。TNO：O2060017
題目解析：
NFC（Near Field Communication）近距離無線通訊，是一種短距離的高頻無線通訊技術，其通訊原理與RFID的技術原理是十分相近的，其特性為適合一對一的通訊模式。

NFC的應用可以歸類成四種情境，下列哪一項較適合用於行動支付？
A　碰一下就好（Touch and Go）
B　碰一下再確認（Touch and Confirm）
C　碰一下再溝通（Touch and Connect）
D　碰一下就上網（Touch and Explore）
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2060018
題目解析：
碰一下再確認（Touch and Confirm）為非接觸式移動支付，用戶將設備靠近嵌有NFC模塊的POS機可進行支付，並確認交易。

5
NFC設備可以在主動或被動模式下交換數據，下列哪一項通訊模式啟動查詢設備以提供整個射頻場域（RF-field）？
A　被動模式
B　半被動模式
C　主動模式
D　半主動模式
正確解答為：A　您的作答值為：A　答對了。TNO：O2060020
題目解析：
在被動模式下，只有發起設備需要產生射頻場域，而目標設備只要使用負載調變（Load Modulation），即可以相同的速度將資料傳回給發起設備。含有NFC的行動載具主要以被動模式為主，除了可以提升節能的特性外，又能不耗電能，可以延長電池的壽命。

NFC近場通訊在主動模式下，最遠傳輸距離應為下列哪一項？
A　10公分
B　10英吋
C　20公分
D　20英吋
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2060021
題目解析：
在主動模式下，每台設備都要能產生自己的射頻場域，依照選定的傳輸速率傳送初始命令，其最大的資料傳輸量為424Kbps，且距離為20公分左右。

A　13.56MHz
B　125KHz
C　2.4GHz
D　5.8GHz

C　自然語言處理系統
D　雲端服務供應商所提供的服務
正確解答為：ABCD　您的作答值為：A　回答錯誤了。TNO：O2060026
