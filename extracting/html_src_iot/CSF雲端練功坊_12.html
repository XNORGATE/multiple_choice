1
物聯網自動化控制技術對行動不便的病患或獨居老人十分重要，因其可有效提升病患或老人於健康管理之舒適性，下列哪一項並非物聯網自動控制之行為？
A　使用者透過智慧型手機或是平板電腦遠端關閉電風扇
B　當家中光線太弱時，光感測器會傳送資料給控制系統，進而開啟燈光或是調整窗簾
C　當家中溫度過高時，溫度感測器會傳送資料給控制系統，進而開啟冷氣或是電風扇
D　當使用者踏入客廳時，紅外線感測器會傳送資料給控制系統，進而開起自動照明系統
正確解答為：A　您的作答值為：A　答對了。TNO：O2030046
題目解析：
物聯網自動控制乃藉由物體間根據使用者需求所事先制定之自動化行為，而「使用者透過智慧型手機或是平板電腦遠端關閉電風扇」屬於使用者手動控制物體開關之行為，不屬於自動控制的物聯網應用。
2
關於物聯網應用，下列敘述哪一項錯誤？
A　智慧醫療方面，使用自動化管理降低人為錯誤機率，以提升病患安全
B　智慧交通方面，災害回報與智慧導覽，可以有效提升交通安全與便利性
C　智慧電網方面，主要透過用戶用電統計，以增加電力公司徵收的電費
D　智慧商務方面，智慧冰箱與垃圾桶的應用，可透過RFID辨識技術紀錄家中食品的數量，也可透過聯網技術進行線上購物
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2030047
題目解析：
智慧電網的物聯網應用，可根據時間電價資訊（尖峰時間之用電適用較高電價，離峰時間之用電適用較低電價），自動將用電價格反應給用戶，公司用戶可參酌時間電價價格變動趨勢而調整生產製程，一般用戶亦可改變用電習慣移轉尖峰用電而充分利用離峰電力，將可大幅減輕電費負擔。相對地，因用戶用電之移轉可紓解系統尖峰負載供電壓力、減緩機組投資壓力及降低停限電機率，故實施時間電價可使用戶與電業互蒙其利。
3
下列哪一項不屬於物聯網於智慧安全的應用範疇？
A　市民使用市話撥打1999通知警方舉報道路損壞
B　當防盜警鈴響時，感測器自動發送警訊通知警方，並通知住戶
C　道路監視器自動追蹤肇事逃逸的車輛
D　發生車禍時，利用行車紀錄器紀錄事發經過，並自動連網通報最近的警務人員與救護中心
正確解答為：A　您的作答值為：A　答對了。TNO：O2030048
題目解析：
撥打一般市內電話為純人為操作，不具備物聯網感知層技術與特性，不屬於物聯網應用。
4
物聯網可應用於門禁管理上，將為生活帶來便利與安全並提升生活品質，下列哪一項不屬於物聯網門禁管理應用？
A　在社區大門上利用RFID技術辨別住戶
B　住戶由房間進入客廳，房間將自動關閉電燈與電腦，客廳也將自動開啟電視與空調，並根據客廳亮度自動調整窗簾的閉合程度
C　為了預防小偷闖入，在住家四周裝置監視器，並利用紅外線感測器進行監控
D　大門前安裝臉部辨識系統，比對此人是否為該住戶
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2030049
題目解析：
「住戶由房間進入客廳，房間將自動關閉電燈與電腦，客廳也將自動開啟電視與空調，並根據客廳亮度自動調整窗簾的閉合程度」屬於物聯網智慧生活之範疇。
5
在醫療產業常見到各種感測技術的物聯網應用，下列哪一項不在物聯網智慧醫療的範疇內？
A　利用RFID技術辨識與控管醫院之藥品或貴重儀器
B　病房裝設具影像辨識技術與連網功能的攝影機，可將即時影像傳遞至護理站
C　智慧手環的監控與通訊功能，可藉由智慧手機主動提醒病人用藥
D　親友自行攜帶水果禮盒與保健食品探望住院之病患
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2030050
題目解析：
攜帶水果禮盒與保健食品親自至醫院探視住院病患，未具備物聯網感知層與網路層之技術與特性，不屬於物聯網之應用。
6
以電信產業的核心行動通訊技術，然後串聯全國糖尿病照護中心，為糖尿病患者創造更多行動便利的照護服務，以上敘述屬於下列哪一種物聯網服務？
A　雲端血糖照護計畫
B　雲端老人照護計畫
C　物聯網居家系統
D　物聯網行動支付活動
正確解答為：A　您的作答值為：A　答對了。TNO：O2030051
題目解析：
糖尿病患者必須時時注意自身的血糖數值，因此雲端血糖照護計畫以電信產業的核心行動通訊技術，串聯全國糖尿病照護中心，為糖尿病患者創造更多行動便利的照護服務。

下列哪一項不是物聯網於健康照護服務中所必須提供的功能及服務方向？
A　關懷互動性
B　資訊即時性
C　自主管理性
D　行動娛樂性
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2030052
題目解析：
物聯網健康照護必須符合關懷互動性、資訊即時性以及自主管理性，即使是行動不便的病患或是獨居老人，都可享受此服務。

B　測量後的數據即時上傳到雲端健康管理通訊平台

A　監控血品的使用者滿意度
B　血品的到院入庫時間
C　血品使用時的交叉比對

下列那一項為使用RFID協助醫療器材管理的目的？
A　避免被偷
B　可即時掌握器材的所在位置及數量，可省去搜尋器材所耗費大量時間
C　避免錯誤使用
D　確保器材的乾淨程度

RFID無線射頻辨識技術，可運用於室內定位，可即時掌握器材的所在位置及數量，可省去搜尋器材所耗費大量時間。

在物聯網的健康照護新技術中，個人健康管理是非常重要的項目，其中有一項技術為非接觸式測溫鏡，主要的功能為下列哪一項？
A　不需要與人體互相直接接觸，就可以進行體表溫度測量
B　可以知道目前環境中的溫度
C　可以知道廚房中的食物溫度
D　可以監控物流的貨品溫度
正確解答為：A　您的作答值為：A　答對了。TNO：O2030056
題目解析：
測溫鏡利用投射在人體表面的LED光源（測量點）進行測量，使用者不需接觸物體，只要距離測溫鏡30公分內，即可進行體表溫度的檢測，完全杜絕病原體經由醫療器材對人體二次感染的機率。
12
在物聯網的個人智慧醫療所能提供的服務中，下列敘述哪一項錯誤？
A　在家裡測量的生理數值可以即時上傳到網路上進行後續分析與處理
B　家人與社工可以直接透過網路知道其健康狀態，並可對其進行即時的關照
C　可建立線上電子病歷與識別技術
D　可強迫病人吃藥
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2030057
題目解析：
物聯網技術主要透過感知技術、網路技術達成智慧應用，旨在輔助人類達到具備智慧化和便利性的理想生活，且不違反人類生理/心理之舒適度。
13
在物聯網的智慧醫療服務中，老年人可以在身上配戴感測器來了解其身上資料，透過下列哪一種感測器可以偵測老人跌倒的狀態？
A　三軸加速度計
B　聲波感測器
C　血壓感測器
D　氣體感測器
正確解答為：A　您的作答值為：A　答對了。TNO：O2030058
題目解析：
三軸加速度計為無線感測器技術之一，可感知配戴人/物在X、Y、Z三軸的加速度變化，因而可用來配戴於老年人身上，並了解其運動狀態或跌倒偵測。
14
在物聯網的醫療感測元件中，心跳感測器已經普遍運用在病人的醫療監測之中，關於心跳感測器，下列敘述哪一項錯誤？
A　配戴者需要將心跳感測器穿在胸前
B　可讓家庭和醫療專業人員監控配戴者體溫、心律與雜音
C　由於感測器的尺寸龐大且穿戴不便，所以配戴者的接受程度不高
D　配戴者可以隨時監控身體狀況，所以使用上已經非常普及
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2030059
題目解析：
微機電技術的進化，無線感測器的感測與處理器等元件已可做得相當微小，易於嵌入常見的穿戴裝置中，如手錶、手環、衣物中，配戴於人身上不會產生任何不便，目前已廣泛使用於各種應用之中。
15
使用物聯網技術來偵測老年人的生理/心理狀態的應用，我們稱之為智慧醫療服務，關於智慧醫療服務，下列敘述哪一項錯誤？
A　若有突發狀況發生，老年人僅能透過自行回報的方式將需求告知醫護人員
B　透過物聯網的傳輸機制，將老年人的突發狀況上傳到中央控管的伺服器
C　事件的嚴重性將快速地被醫院得知並判斷，而院方也能夠盡速派遣救護人員到場，或者也可以利用互聯網將此事件傳送至鄰近義工的通訊設備中
D　同一時間，可即時透過物聯網技術開啟因應的居家環境感測系統
正確解答為：A　您的作答值為：A　答對了。TNO：O2030060
題目解析：
若老年人有突發狀況產生，透過物聯網的感測技術，可觸發網路通訊之自動通報功能，無須老年人自行回報醫護人員。
