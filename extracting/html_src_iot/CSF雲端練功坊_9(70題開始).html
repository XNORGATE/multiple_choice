智慧型感知行車紀錄器結合了傳統的行車紀錄器、環境感知及網路功能，下列哪些物理參數被列入主要的環境監測紀錄內？(複選題)
A　溫溼度
B　二氧化碳感測器
C　三軸加速度
D　車門防盜開關
正確解答為：ABC　您的作答值為：A　回答錯誤了。TNO：O2030001
題目解析：
車主在車內使用智慧型感知行車紀錄器，可使用傳統的行車紀錄功能，並結合溫濕度與二氧化碳感測器，得知行車環境的空氣品質與舒適程度，再藉由三軸加速度計可測得道路平坦度，將相關資訊透過網路之間的分享，則可發展出具環境感知特性之行車導引服務。

A　血袋
B　手術器材
C　針筒
D　批價
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2030002
題目解析：
醫療器材管理雲中，RFID應用於血袋、手術器材、病人標籤、實驗器材、針筒、藥品等。

A　資訊流通
B　即時分析
C　即時醫療
D　即時照護

題目解析：
物聯網在醫療領域的應用被稱作智慧醫療（應用層），其中有兩項重要的特色，一為強調訊息的即時採集（感知層），二為資訊流通（網路層）。
4
依照應用場景的不同，下列哪一項不屬於智慧醫療的三大類型之一？
A　藥政管理
B　健康照護
C　醫療管理
D　醫療感測
正確解答為：A　您的作答值為：A　答對了。TNO：O2030004
題目解析：
智慧醫療的三大類型分別為居家的健康照護、以及醫院方的醫療感測與醫療管理。

A　醫療處置
B　人口統計資料
C　藥物使用

電子健康紀錄也能夠包括醫學相關的參照資料、醫療處置、藥物使用、人口統計資料、其他與非醫療的管理資料。

依歐洲電信標準協會（European Telecommunications Standards Institute, ETSI）將物聯網劃分成三個階層，政府為提倡節能減碳做環保並保障大眾搭乘公共運輸時的安全性，故建立先進大眾運輸服務系統，此系統應屬於下列哪一層？
A　應用層
B　傳輸層
C　網路層
D　感知層
正確解答為：A　您的作答值為：A　答對了。TNO：O2030007
題目解析：
政府為提倡節能減碳做環保並保障大眾搭乘公共運輸時的安全性，故建立先進大眾運輸服務系統，此系統應屬於物聯網的應用層，且為物聯網智慧運輸之應用。
8
於醫院內，若有缺失，可能造成病人服錯藥或輸血錯誤，甚至誤用手術器材與針筒導致急性過敏或敗血症，故將各項藥品、血袋、針筒或手術器材等物品加裝RFID，作為流程管理追蹤之用，並結合醫療廢棄物管理，以上敘述屬於下列哪一種智慧醫療應用？
A　健康照護
B　醫療管理
C　醫療感測
D　藥材管理
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2030008
題目解析：
於醫院的醫療管理中，包含藥品、血品、手術器材、針筒等，任何失誤都可能造成生命危害，因此上述醫療物品輔以RFID技術進行醫療管理，作為流程管理追蹤之用，並結合醫療廢棄物管理，可大幅降低人工失誤的機率。
9
智慧電網透過下列哪一項建設，讓使用者可以了解家中電量，且配電端亦可了解電量使用情況，進而針對發電與配電之間做良好的調度？
A　智慧型電表基礎建設（AMI）
B　智慧型電表（Smart meter）
C　網路電表（Net meter）
D　電力線通信（PLC）
正確解答為：A　您的作答值為：A　答對了。TNO：O2030009
題目解析：
在智慧型電表基礎建設（Advanced Metering Infrastructure, AMI）中，主要包含智慧型電表（Smart meter）、通信網路、電表資料管理系統（Meter Data Management System, MDMS）三大部分。智慧型電表（Smart meter）可自動產生用電資訊，讓使用者可以了解家中電量。若再透過 AMI 的通訊網路，將用戶端的用電資訊傳遞至電表資料管理系統（Meter Data Management System, MDMS），便可進行電力資料的整合與分析。
10
可提供追蹤能源消耗高峰和通知有設備忘記關閉，以上服務的能源消耗偵測是由智慧電網的哪一層所提供？
A　感知層
B　實體層
C　網路層
D　應用層
正確解答為：A　您的作答值為：A　答對了。TNO：O2030010
題目解析：
物聯網智慧電網的應用中，可藉由智慧電表追蹤能源消耗量，並統計用電高峰時段，透過其電力感知技術，也可感知用電設備的開關狀態，主要屬於物聯網感知層的範疇。
11
在智慧家庭的應用中，為了因應台灣現代工商社會，親子共處時間有限而導致親子共讀之困難，互動型自動說書機便油然而生。說書機一般而言可分兩部分，分別為自動說書娃娃與智慧書籍，娃娃可根據感測到的資訊述說不同的書籍內容。以上敘述屬於下列哪一項感測技術應用？
A　RFID
B　Bluetooth
C　IrDA
D　ZigBee
正確解答為：A　您的作答值為：A　答對了。TNO：O2030011
題目解析：
說書機一般而言可分兩部分，分別為自動說書娃娃與智慧書籍：在智慧書籍中鑲嵌RFID標籤，其中紀錄了書籍內容相對應的語音訊息；在自動說書娃娃中，則裝入了RFID讀取器與發聲喇叭，透過娃娃與書籍的近距離接觸，RFID讀取器便可從RFID標籤中讀取到語音訊息，並可將該訊息傳送至發聲喇叭，如此一來，說書娃娃便可模仿父母說書的角色，說故事給孩子聽了。
12
物聯網在抗旱防汛的應用上，可以使用無線環境感測器，感知土壤內的溫、溼度，並定期將資訊透過聯網閘道器回報給雲端進行運算，以便讓農地灌溉系統進行即時控管，其所使用的無線環境感測器之通訊技術以下列哪一項較合適？
A　ZigBee
B　NFC
C　IrDA
D　RFID
正確解答為：A　您的作答值為：A　答對了。TNO：O2030012
題目解析：
無線環境感測器是物聯網感知層的重要技術之一，除了負責感知物理環境資訊外，更可利用ZigBee通訊協定，其特殊的網狀網路拓樸可應用於大範圍的感測器布建，將資料逐步傳遞至聯網閘道器，以供其回報資訊到雲端。
13
智慧家庭為物聯網常見的應用之一，為了讓房子的設備更具人性化的控制能力，需裝設各種環境感測器。這些感測器所偵測的資訊，利用無線傳輸技術傳送到家中的中央調控系統並傳送至雲端。為連結各種不同無線感測器，下列哪一項配備不可或缺？
A　物聯網AP（Access Point）
B　衛星定位系統
C　區域網路集線器
D　寬頻網路路由器
正確解答為：A　您的作答值為：A　答對了。TNO：O2030013
題目解析：
物聯網AP（Access Point）為智慧家庭應用常見的設備之一，除了可連結各種不同無線感測器，讓房子的設備更具人性化的控制能力之外，更可將相關資訊連結至網際網路，讓使用者即使不在家中，也可透過遠端擷取家電資訊，乃至於遠端控制家電的開與關。
14
在醫療領域中會使用到各種不同的感測元件，以便隨時監控病人的情況，下列哪些項目屬於常見的醫療感測器？(複選題)
A　血壓感測器
B　心跳感測器
C　Google眼鏡
D　智慧網關
正確解答為：AB　您的作答值為：A　回答錯誤了。TNO：O2030014
題目解析：
「血壓感測器」與「心跳感測器」裝置在人體以便偵測血壓與心跳等生理資訊，以供醫療目的之用。而Google眼鏡與智慧網關則不具備感知人體生理資訊之醫療功能。
15
智慧家庭為物聯網常見的應用之一，下列哪些系統屬於智慧家庭的一部分？(複選題)
A　門禁系統
