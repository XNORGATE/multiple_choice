對於物聯網行動商務，下列敘述哪一項錯誤？
A　行動商務是使用者以行動化的終端裝置透過行動通訊網路來進行商業交易活動
B　行動支付O2O應用、智慧物流等是行動商務的創新應用已逐漸轉變為以企業為主角，由企業來決定產品的需求
C　物聯網與行動科技的快速發展下，許多行動商務的型態已日新月異
D　行動商務徹底轉變為全新的物聯網商業模式
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2040001
題目解析：
行動商務已逐漸轉變為以消費者為主角，由消費者來決定產品的需求與評價。

下列哪些是智慧型手機可能配備的通訊系統？(複選題)
A　5G
B　Wi-Fi
C　Ethernet
D　NFC
正確解答為：ABD　您的作答值為：A　回答錯誤了。TNO：O2040003
題目解析：
在網路技術中，智慧型手機已普遍具備了Bluetooth、Wi-Fi、NFC、4G、5G等通訊技術。而Ethernet接孔（如RJ45），因體積較大，無法滿足智慧型手機的輕薄需求，且有線連接大幅限制其移動性，因此不會出現在智慧型手機上。

下列哪一項不是目前可被手機接受的條碼技術？
A　QR Code
B　Bar Code
C　CRC Code
D　DataMatrix Code
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2040004
題目解析：
透過影像辨識技術，手機可掃描QR Code、Bar Code、DataMatrix Code等條碼。CRC Code是用來驗證資料是否錯誤的驗證碼，不算是條碼技術。

下列哪些是手機的作業系統？(複選題)
A　Android
B　iOS
C　Windows Phone
D　Inventor
正確解答為：ABC　您的作答值為：A　回答錯誤了。TNO：O2040005
題目解析：
智慧型手機主要的作業系統有Google所創立的Android、Apple公司針對行動裝置所開發的iOS以及Microsoft所推出的Windows Phone（Windows Mobile）。 Inventor則是一套3D機械設計軟體。

下列哪一項是智慧型手機尋找附近餐廳的主要技術？
A　三軸加速度計
B　衛星定位系統
C　陀螺儀
D　超音波
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2040006
題目解析：
尋找附近餐廳必須知道使用者目前所在位置，因此需透過GPS衛星定位技術得知所在位置資訊。三軸加速度計是偵測方向及速度與手機震動，超音波感測器是用來感測物體距離，陀螺儀則是用來感測與維持方向。

下列哪一項不是智慧型手機在NFC上的應用？
A　在行動支付中，縮短交易時間
B　在運輸中，取代實體通行證
C　在交通中，規劃路徑，縮短行車時間
D　在信用評價中，取代實體會員卡

關於行動支付的敘述，下列哪一項錯誤？
A　許多行動裝置搭配NFC、RFID、條碼辨識等技術作為身分辨識
B　行動支付是指以移動存取設備透過移動網路，所啟動的支付行為
C　目前行動支付僅支援NFC技術
D　常見的行動支付有台灣Pay、Apple Pay、LINE Pay、街口支付
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2040009
題目解析：
並非所有行動裝置皆支援NFC技術，因此目前行動支付皆支援條碼辨識，部分行動支付支援NFC技術。

下列哪些是達成行動支付的模式？(複選題)
A　APP軟體直接使用電子錢包
B　採用NFC技術加APP軟體，串聯各種金融卡片、電子錢包
C　使用ATM讀取信用卡資料，然後付款
D　使用電信商開設的金融平台
正確解答為：ABD　您的作答值為：A　回答錯誤了。TNO：O2040010
題目解析：
現階段行動支付有五種達成模式：第一種是行動裝置外接感應設備，例如手機殼裝有悠遊卡晶片，第二種是使用電信商開設的金融平台，第三種採用NFC技術加APP軟體，串連各種金融卡片、電子錢包。第四種是APP軟體利用QR Code條碼連接網路刷卡系統、金融卡轉帳系統，第五種是是APP軟體直接使用電子錢包。

對於工業4.0於食品生產之應用，下列哪一項錯誤？
A　食品安全需要保存與查驗很多過程檢驗結果，靠著智慧型自動化的資訊流可以快速檢驗
B　利用智慧型控制可以有效率的利用資訊並監督檢驗作業的進行
C　智慧型自動化可以提供食品產業少量多樣的生產
D　可以透過智慧型自動化進行線上檢測以建立食品安全
正確解答為：A　您的作答值為：A　答對了。TNO：O2040014
題目解析：
食品安全需要保存與查驗很多過程檢驗結果，靠著智慧型自動化的資訊流可以快速傳遞到主管機關與消費者。透過智慧型自動化進行線上檢測以建立食品安全，各種特殊指標成份的感測器是值得研究發展的方向也可利用智慧型自動化進行線上之檢測，但需建立於大量的實驗數據上，以減少實驗之誤差。
