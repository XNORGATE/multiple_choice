下列哪一項屬於物聯網於健康照護之應用？
A　小明在上班時，想起早上出門前未關閉家中的電腦，便使用手機遠端將家中的電腦關閉
B　小英收到一封簡訊，內容為：您今天已走了一萬步
C　小華使用手機拍下自己新買的衣服，並分享至社群網站
D　小景使用智慧眼鏡拍攝路邊美麗的風景
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2030016
題目解析：
由小英收到簡訊的內容，可推測小英身上配戴著感測器可偵測當日行動步數，累計至目標的一萬步後，便會透過網路功能，將相關資訊以簡訊方式傳給小英，如此日復一日，小英即可得知每日的運動目標是否達成。此即為物聯網於健康照護之應用之一。

物聯網於運動管理上有許多應用，下列哪一項錯誤？
A　穿戴智慧手錶，可紀錄慢跑時間與里程數
B　穿戴智慧鞋墊跑步，可測量慢跑時左右步伐之平衡性
C　穿戴智慧眼鏡於道路中，可規劃慢跑路線與導覽
D　穿戴智慧手環移動至房間，可透過NFC技術自動變換生活情境
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2030018
題目解析：
使用者穿戴智慧手環，可追蹤自身的體能狀況，例如：心率、動作、睡眠，甚至是平靜和興奮的時刻都可追蹤記錄，此即為物聯網於運動管理上的應用。但以NFC技術自動變換生活環境，則為智慧居家的應用場景。

12
高爾夫活動中，高爾夫球桿的數量多達十數支，若球員在廣大的球場中揹著球桿移動，該球員可能因額外勞動而超過一日熱量代謝的建議量，為了解決此問題，球場管理員在球桿車中加裝感應器和馬達，並在球桿內嵌入感應器、RFID識別等功能。此後，球桿車除了可自動跟隨在球員身邊以減少不必要的球員勞動量，更可收集各式球桿所回傳的紀錄，並針對球員的運動量進行統計、回報、分析與建議，以上敘述屬於下列哪一種物聯網應用？
A　物聯網於運動管理之應用
B　物聯網於物流管理之應用
C　物聯網於熱量管理之應用
D　物聯網於生理醫療之應用
正確解答為：A　您的作答值為：A　答對了。TNO：O2030027
題目解析：
如題所述之物聯網應用中，球桿車可藉由其內嵌的感應器與球桿中的感測器相互溝通，得知球員的運動相關資訊，如揮桿次數、力道、角度、加速度等資訊，進行後續的統計、回報、分析與建議，可供球員日後在運動上的管理與調整。

下列哪些是物聯網在食品控管中的應用？(複選題)
A　自助餐利用條碼管控菜色及份量，菜量不足時可立即補充
B　食品工廠利用配料管理系統，可對所有物料加裝條碼識別，再運送至生產工廠
C　食品包裝上以文字標記該產品的熱量、脂肪及成分
D　冷凍商品在配送中，若溫度太高，將會發出簡訊給相關管理人員
正確解答為：ABD　您的作答值為：A　回答錯誤了。TNO：O2030029
題目解析：
「自助餐利用條碼管控菜色及份量（感知層），得知菜量不足時（網路層）可立即補充」、「食品工廠利用配料管理系統，可對所有物料加裝條碼識別（感知層），再運送至生產工廠（感知層）」、「冷凍商品在配送中，若溫度太高（感知層），將會發出簡訊給相關管理人員（感知層）」皆運用了物聯網感知層或網路層技術；但「食品包裝上以文字標記該產品的熱量、脂肪及成分」未使用相關物聯網技術，不屬於物聯網在食品控管中應用之一。

物聯網技術落實了溫度管理的應用與驗證，可減少人類於金錢、物品與生命上的損失，下列敘述哪一項非溫度管理之功能？
A　發生火災時自動連線至近端警衛與遠端警消人員
B　監控和紀錄溫度感測器的溫度數據與當前位置
C　若發生多起電腦入侵事件，可啟用防火牆功能，並查看相關紀錄
D　使用者可透過遠端連線，檢視家中各項家電的資訊，除了開關狀態與用電資訊外，更可查看各家電的即時溫度與歷史溫度，若有異常溫度狀態產生，便可遠端關閉家電電源或設定自動關閉功能
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2030030
題目解析：
以防火牆防堵電腦入侵事件，屬於資訊安全之防護行為，與物聯網技術落實溫度管理的應用與驗證無關。
