在行動通訊技術標準中，下列哪一項為第四代行動通訊技術標準（4G LTE）才有的功能？
A　處理類比訊號，僅限語音的傳送
B　提供更高速及更低延遲的行動通訊系統
C　語音服務與現有網際網路的服務整合
D　處理數位訊號，僅可傳送語音及SMS文字簡訊
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2050031
題目解析：
1G（1st Generation），以類比（analog）聲音訊號為處理對象，僅限語音的傳送；2G（2nd Generation），主要處理數位（Digital）訊號，並改良第一代行動通訊的安全性及隱私性問題，可傳送語音及SMS文字簡訊；3G（3rd Generation），將語音服務與現有網際網路的服務整合，使電信網路與數據網路可合而為一；4G（4th Generation），可提供更高速及更低延遲的行動通訊系統。

關於服務導向架構（Service-Oriented Architecture, SOA）技術所提到的特性，下列敘述哪一項錯誤？
A　以物件為中心（Object centric）
B　為一種開放的標準（Open standard）
C　鬆散的關係（Loosely coupled）
D　為一種分散式架構（Distributed）
正確解答為：A　您的作答值為：A　答對了。TNO：O2050032
題目解析：
服務導向架構是將訊息集合起來避免資訊孤島的產生，並以流程為中心（Process centric），透過了解應用服務的流程要求，並將其包括輸入與輸出資料格式定義出來，如此其他的發展者就可以依據服務介面開發或選擇合適的元件來完成工作。

在一些物聯網的應用中，感測器需要將取得的資料直接傳送到數百公尺遠的資料收集裝置，下列那一項是比較適合的資料傳輸技術？
A　Wi-Fi
B　LoRa
C　RFID
D　NFC
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2050033
題目解析：
LoRa（Long Range）屬於長距離低功耗的物聯網通訊技術，最大傳輸距離可長達15~20公里。
4
物聯網發展面臨的問題之一是資料產生後，無法保證系統或是使用者能否分辨其資料的內容而善加運用，下列哪一項技術的作用是為了解決這樣的問題？
A　中介軟體
B　網頁服務
C　語意網
D　雲端運算
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2050034
題目解析：
語意物聯網是物聯網的提升，主要體現為由詞句比對（關鍵字比對）轉變為語意比對（意義比對）。具體的做法就是物體資訊的語意標注，用於語意標注的術語均取自應用主體。如此，應用主體可正確理解該資訊之語意而機器則能根據本體所指明的語意標記與其他資訊的語意標記之間的關係進行推理，進而從該資訊抵達與之相關的其他資訊。
5
關於語意網，下列敘述哪一項錯誤？
A　語意網的目標是希望讓機器與機器之間能夠溝通
B　語意網單獨透過XML來表達資料
C　語意網透過SPARQL協定與RDF查詢資料
D　語意網是一個高度強調開放式的標準
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：O2050035
題目解析：
語意網並非透過XML來表達資料，而是通過資源描述框架（RDF）以及其擴充架構（RDFS）與本體論語言（OWL）等，使得資料具有語意。將應用與資料分開，資料自帶語意，同時含有多維的表示。

下列哪一項不屬於中介軟體之一？
A　服務組合
B　服務管理
C　物件抽象化
D　系統標準化
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2050036
題目解析：
中介軟體（Middleware）泛指二個不同媒介中間應用於系統整合的軟體，如:服務組合、服務管理、物件抽象化，中介軟體的存在便是為了解決物聯網中多重異構的問題，包含硬體和網路、作業系統和編譯語言的異構。

下列哪一項不是5G的應用情境？
A　超可靠低延遲通訊（URLLC）
B　增強型行動寬頻通訊（eMBB）
C　高效能運算（High Performance Computing, HPC）
D　大規模機器型通訊（mMTC）
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2050037
題目解析：
國際行動通訊組織（IMT）提出的IMT-2020技術中提到5G的三個應用情境，分別為超可靠低延遲通訊（Ultra-reliable and Low Latency Communications, URLLC）、增強型行動寬頻通訊（Enhanced Mobile Broadband, eMBB）、大規模機器型通訊（Massive Machine Type Communications, mMTC）。

在語意網中，下列哪一項技術負責篩選出需由應用程式而不是由人來判斷的資訊？
A　SPARQL（SPARQL Protocol and RDF Query Language）
B　RIF（Rule Interchange Format）
C　RDFS（Resource Description Framework Schema）
D　OWL（Ontology Web Language）
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：O2050038
題目解析：
OWL（Ontology Web Language）彌補了RDFS的不足，用於描述那些需要由應用程式而不是由人來判斷的資訊。OWL可被用來明確表示辭彙表中術語的含義以及語句間的關係。

隨著Wi-Fi技術的持續發展，Wi-Fi聯盟推動了許多不同世代的Wi-Fi，其中第六代Wi-Fi（Wi-Fi 6）採用的是IEEE的哪一項標準？
A　IEEE 802.11aa
B　IEEE 802.11ax
C　IEEE 802.11ac
D　IEEE 802.11ay

題目解析：
Wi-Fi聯盟推動的第四、五、六代Wi-Fi分別採用IEEE 802.11n、IEEE 802.11ac、IEEE 802.11ax標準。

藍牙4.0技術，下列敘述哪一項錯誤？
A　分為單工與雙工模式
B　藍牙4.0擁有「低功耗藍牙」、「傳統藍牙」和「高速藍牙」三種模式
C　藍牙4.0的重點則在於主打Low Energy（LE）
D　在藍牙4.0中，三種模式是可以獨立但不可共同存在

12
隨著物聯網應用的普及，藍牙技術的發展也開始強調低功耗，下列哪一個版本的藍牙才開始強調低功耗的特性？
A　藍牙2.0
B　藍牙3.0
C　藍牙4.0
D　藍牙5.0
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2050042
題目解析：
藍牙2.0為傳統藍牙，藍牙3.0強調傳輸速度的提升，因此稱為高速藍牙，藍牙4.0開始則強調低功耗，因此稱為低功耗藍牙。

下列哪一個版本的藍牙開始支援網狀拓樸？
A　藍牙2.0
B　藍牙3.0
C　藍牙4.0
D　藍牙5.0
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：O2050043
題目解析：
早期藍牙網路屬於星狀拓樸，為了可以將資料傳送到更遠的藍牙裝置，藍牙網狀網路於2017年發表，並可運行在藍牙4.0及以上的低功耗藍牙架構中。

相較於藍牙4.0，下列哪些是藍牙5.0具備的特性？(複選題)
A　傳輸距離更遠
B　可支援網狀（Mesh）拓樸
C　傳輸速率更快
D　傳輸容量更大
正確解答為：ACD　您的作答值為：A　回答錯誤了。TNO：O2050044
題目解析：
藍牙5.0在2016年發布，有效傳輸距離約是藍牙4.0版的4倍以上，最大傳輸速度約為4.0版的2倍，傳輸容量比藍牙4.0版提升了8倍。網狀網路在藍牙4.0版本即可支援。
