下列哪些行為已經侵犯了個人的網路隱私權？(複選題)
A　在房客臥室安裝攝影機並且連上網路監看
B　在朋友家裡一起觀賞電腦上儲存的出國遊覽的影片
C　偷偷使用同學的電子郵件帳號密碼查看其私人郵件內容
D　在朋友的委託下使用其自然人憑證代為操作網路報稅
正確解答為：AC　您的作答值為：AC　答對了。TNO：U0070031
題目解析：
若是在私人空間裝設攝影機，有可能觸犯刑法第315-1條妨害秘密罪：「有下列行為之一者，處三年以下有期徒刑、拘役或三十萬元以下罰金： 一、無故利用工具或設備窺視、竊聽他人非公開之活動、言論、談話或身體隱私部位者。二、無故以錄音、照相、錄影或電磁紀錄竊錄他人非公開之活動、言論、談話或身體隱私部位者。」。而未經他人同意，登入其電子郵件帳號窺視，已觸犯刑法第358條妨害電腦使用罪：「無故輸入他人帳號密碼、破解使用電腦之保護措施或利用電腦系統之漏洞，而入侵他人之電腦或其相關設備者，處三年以下有期徒刑、拘役或科或併科三十萬元以下罰金。」

在網頁瀏覽程式中，下列哪一項功能的運用有可能對使用者的網路隱私權造成威脅？
A　刪除瀏覽紀錄
B　以Cookie來紀錄與使用者相關的上網動作與資訊
C　下載播放串流視訊
D　使用HTTPS的協定
正確解答為：B　您的作答值為：B　答對了。TNO：U0070032
題目解析：
因為HTTP協定是無狀態的，即伺服器不知道用戶上一次做了什麼，由其是在網路購物應用中，用戶瀏覽了商品頁面，買了一盒濾掛式咖啡和兩罐鮮奶，最後進入結帳頁面時，由於HTTP的無狀態性，伺服器並不知道用戶在上一個商品頁面買了什麼，此時Cookie就可以用來協助記錄使用者的動作與資訊，另一個常見的應用是在第一次登入網站後的自動登入功能。Cookie在某種程度上已經涉及使用者的隱私和安全，因為Cookie有可能記錄了一些敏感訊息，如使用者名稱、密碼、使用的瀏覽器和曾經存取的網站。

國內與資訊及電子商務相關的法律中，下列哪一項與保護個人網路隱私權的關係最為密切？
A　著作權法
B　電子簽章法
C　公平交易法
D　個人資料保護法
正確解答為：D　您的作答值為：D　答對了。TNO：U0070033
題目解析：
個人資料保護法（個資法），其保護客體不再以電腦處理之資料為限，人工蒐集、處理之個人資料亦適用之。公務機關及所有行業之企業、團體及個人等非公務機關均納入適用主體，其立法主要目的為：一、規範個人資料的蒐集、處理及利用。二、避免人格權受侵害。三、促進個人資料之合理利用。

關於個人資料保護法的規定，當事人就其個人資料有下列哪些權利？(複選題)
A　請求刪除
B　查詢或請求閱覽
C　請求以網路傳遞給第三者
D　請求停止蒐集、處理或利用
正確解答為：ABD　您的作答值為：AD　回答錯誤了。TNO：U0070034
題目解析：
依據個人資料保護法第3條規定，當事人就其個人資料依本法規定行使之下列權利：一、查詢或請求閱覽。二、請求製給複製本。三、請求補充或更正。四、請求停止蒐集、處理或利用。五、請求刪除。
5
對於公務機關蒐集個人資料而言，下列哪些情況下，公務機關可拒絕當事人就其保有之個人資料提出提供閱覽或複製本的請求？(複選題)
A　妨害公務機關執行法定職務
B　因公務機關電腦設備故障無法提供
C　妨害國家重大利益
D　妨害第三人之重大利益
正確解答為：ACD　您的作答值為：ACD　答對了。TNO：U0070035
題目解析：
依據個人資料保護法第10條規定，公務機關或非公務機關應依當事人之請求，就其蒐集之個人資料，答覆查詢、提供閱覽或製給複製本。但有下列情形之一者，不在此限：一、妨害國家安全、外交及軍事機密、整體經濟利益或其他國家重大利益。二、妨害公務機關執行法定職務。三、妨害該蒐集機關或第三人之重大利益。

下列哪些是屬於違法的行為？(複選題)
A　學生為了求學便利，可任意的複製軟體使用
B　為了上課影印課本
C　購買了版權軟體時，為了保存可以自己複製一份
D　使用無授權的軟體工作，並在完成工作後將無授權的軟體刪除

關於著作權，下列敘述哪一項正確？
A　在不知情的情形下，代為重製未經授權的他人著作物，不屬於違法的行為
B　在知情的情形下，代為重製未經授權的他人著作物，不屬於違法的行為
C　在不知情的情形下，代為重製未經授權的他人著作物，無法判定是否有違法的行為
D　在不知情的情形下，代為重製未經授權的他人著作物，是屬於違法的行為
正確解答為：D　您的作答值為：D　答對了。TNO：U0070039
題目解析：
依著作權法第87條第1項第7款：「未經著作財產權人同意或授權，意圖供公眾透過網路公開傳輸或重製他人著作，侵害著作財產權，對公眾提供可公開傳輸或重製著作之電腦程式或其他技術，而受有利益者。」另外依著作權法第88條：「因故意或過失不法侵害他人之著作財產權或製版權者，負損害賠償責任。數人共同不法侵害者，連帶負賠償責任。」因此在不知情的情形下，代為重製未經授權的他人著作物，仍屬於違法行為。

電腦高手阿強買回一套大富翁遊戲軟體，因為覺得非常好玩，於是從事了下列選項中的行為，下列哪些為合法的行為？(複選題)
A　重製一份作為備份之用
B　將備份的大富翁光碟借給同學玩
C　重製多片光碟，以低價賣給同學
D　為配合家中奇特的電腦環境，阿強修改作業系統設定，讓大富翁能執行順暢
正確解答為：AD　您的作答值為：D　回答錯誤了。TNO：U0070040
題目解析：
著作權法第59條規定：「合法電腦程式著作重製物之所有人得因配合其所使用機器之需要，修改其程式，或因備用存檔之需要重製其程式。但限於該所有人自行使用。前項所有人因滅失以外之事由，喪失原重製物之所有權者，除經著作財產權人同意外，應將其修改或重製之程式銷燬之。」若將其備份之電腦程式著作重製物借予他人或從事商業行為，已觸犯著作權法第91條規定：「擅自以重製之方法侵害他人之著作財產權者，處三年以下有期徒刑、拘役，或科或併科新臺幣七十五萬元以下罰金。意圖銷售或出租而擅自以重製之方法侵害他人之著作財產權者，處六月以上五年以下有期徒刑，得併科新臺幣二十萬元以上二百萬元以下罰金。以重製於光碟之方法犯前項之罪者，處六月以上五年以下有期徒刑，得併科新臺幣五十萬元以上五百萬元以下罰金。著作僅供個人參考或合理使用者，不構成著作權侵害。」

關於在電子郵件中，相關的規則及禮節，下列敘述哪些正確？(複選題)
A　英文書信最好全部用大寫
B　引用來函應力求精簡，且需註明哪些是引用的部分
C　引用他人文章或網頁內容最好標明出處或網頁連結
D　為力求精簡，在表示請求的書信中，當敘述完自己的目的後，即可結束
正確解答為：BC　您的作答值為：BC　答對了。TNO：U0070041
題目解析：
電子郵件的類型有相當多，如通知、徵詢、分享、請求、邀請等郵件種類，在撰寫郵件時，需注意禮貌與重點，在郵件中要清楚表明意圖或訴求，且在結尾時以用問候或感謝等禮貌性結束語。若要引用來函，應力求精簡並註明引用範圍，且應檢視是否有誤植；若是引用他人文章，亦需標明出處或該文章連結。英文全部大寫通常用於強調單詞或語氣，在字體排印學中的研究中已普遍表明全部大寫的可讀性和易讀性比小寫還差，另外，全部大寫也常被用來表達說話人的憤怒或吼叫語氣，若郵件中全部大寫可能會使收件者覺得有威嚇意味或厭惡。

關於MP3的使用，下列敘述哪一項是合法的行為？
A　將自己購買的CD轉換成MP3放在自己網站上供人下載
B　將自己購買的CD轉換成MP3放在自己的電腦裡聆聽
C　收到朋友寄來的MP3歌曲，再轉寄給別人
D　將MP3集結成光碟販售
正確解答為：B　您的作答值為：B　答對了。TNO：U0070042
題目解析：
著作權法第59條規定：「合法電腦程式著作重製物之所有人得因配合其所使用機器之需要，修改其程式，或因備用存檔之需要重製其程式。但限於該所有人自行使用。」因此可以將購買的CD轉換成MP3放在自己電腦僅供自己聆聽。但若將其備份之MP3重製物給予他人或從事商業行為，已觸犯著作權法第91條規定。

自己開設網路電台當DJ，播放流行音樂供大家聆聽，會產生下列哪一項違法行為？
A　複製
B　公開傳輸
C　重製
D　修改
正確解答為：B　您的作答值為：B　答對了。TNO：U0070045
題目解析：
依著作權法第3條第1項第10款之規定：「公開傳輸：指以有線電、無線電之網路或其他通訊方法，藉聲音或影像向公眾提供或傳達著作內容，包括使公眾得於其各自選定之時間或地點，以上述方法接收著作內容。」因此凡是網站上主動或被動播放的音樂、網路廣播電台、網路影片、網路廣告及其他通訊方法，如有公開傳輸音樂時，需申請公開傳輸之相關授權。
