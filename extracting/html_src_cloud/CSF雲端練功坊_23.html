關於單一伺服器，目前常見企業虛擬化效能提昇方案為下列哪一項？
A　使用多插槽的中央處理器（Central Processing Unit, CPU）
B　使用高速記憶體
C　使用固態硬碟（Solid State Disk, SSD）取代傳統轉盤硬碟
D　使用多片網路卡
正確解答為：C　您的作答值為：C　答對了。TNO：U0060046
題目解析：
影響虛擬化平台執行效率最顯著的是硬碟的速度，而傳統硬碟（HDD）僅有150 MB/s左右的讀寫速度，若要顯著提升虛擬化效能，最常見的方案是將傳統硬碟替換成固態硬碟（SSD），以Intel Optane 905P系列SSD為例，其讀取速度可達2,600 MB/s，寫入速度可達2,200 MB/s，延遲10us，另外Micron發表的X100 NVMe SSD，讀寫速度更超過9GB/s，延遲僅有8us，與傳統HDD的150 MB/s讀寫速度與平均延遲4.17ms相比，可大幅提昇虛擬化效能。
2
目前虛擬化解決方案提供商，其產品最完整的提供商為下列哪一項？
A　VMware
B　Microsoft
C　Oracle
D　IBM
正確解答為：A　您的作答值為：A　答對了。TNO：U0060047
題目解析：
VMware是戴爾科技旗下的軟體公司，成立於1998年，主要提供雲端運算和虛擬化技術，也是目前全球領先的虛擬化解決方案提供商，VMware的產品大致可分為三類：vSphere虛擬基礎架構管理，提高硬體機器的可用性、可靠性，讓不同類型的使用者來使用。vCenter虛擬基礎架構自動化，利用其平台來自動化相關服務，例如資源分配和安全控管。ESXi虛擬化平台，能夠運行多種作業系統的虛擬化平台。
3
下列哪一項不是VMware寄居架構產品？
A　VMware Workstation
B　VMware Fusion
C　VMware Player
D　VMware vSphere
正確解答為：D　您的作答值為：D　答對了。TNO：U0060048
題目解析：
VMware vSphere、VMware ESX、VMware ESXi屬於原生架構虛擬化產品。VMware Workstation、VMware Player、VMware Fusion（專為Mac系統開發）、VMware Server（已在2011年6月30日結束支援）都是寄居架構虛擬化產品。

下列哪一項不是原生架構（Native）產品？
A　Microsoft Hyper-V
B　VMware vSphere
C　KVM
D　Oracle VirtualBox
正確解答為：D　您的作答值為：D　答對了。TNO：U0060049
題目解析：
原生架構的虛擬化軟體有VMware ESX、VMware vSphere、Xen、KVM、Adeos、CP/CMS、Hyper-V、LynxSecure、SIMMON、XtratuM、z/VM等，Oracle VirtualBox屬於寄居架構虛擬化產品。

VMware下的虛擬硬碟格式為下列哪一項？
A　VMDK
B　VMX
C　VHD
D　VMEM
正確解答為：A　您的作答值為：D　回答錯誤了。TNO：U0060050
題目解析：
VMDK是一種檔案格式，描述虛擬磁碟在VMware Workstation和VirtualBox等虛擬機中使用的內容，該格式最初是由VMware為其虛擬設備產品所開發。
