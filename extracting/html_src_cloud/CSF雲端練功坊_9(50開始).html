A　Facebook
B　Microsoft Office

2
關於網路社群服務使用，下列敘述哪一項正確？
A　社群服務皆可設定朋友的瀏覽權限，所以可放心在已經設定權限的網站放上私密的資料與相片
B　規劃安排一趟國內旅遊，可到各縣市政府觀光局網站、部落格瞭解各種旅遊資訊及配套方案
C　很多網站都是使用暱稱不是本人的真實姓名，所以可在上面為所欲為或謾罵
D　學校附近的便當店漲價或用餐不愉快，可在留言板上罵對方是奸商或黑店
正確解答為：B　您的作答值為：B　答對了。TNO：U0030002
題目解析：
以Facebook為例，當你使用Facebook時，即同意允許所有人觀看你Facebook上所有公開的資料，從設立帳號必須以真實姓名才能註冊，到個人自我介紹上的就讀學校、工作經歷、家人及感情狀況、公開發文、上傳照片、回覆文章等，都是將個人隱私揭露。任何放在網路上的消息、資訊，皆有機會洩漏自己的隱私，另外若在網路上任意謾罵或散佈假消息，皆可能會觸犯相關法律，因此放在網路上的資訊、發文或回應，皆須相當注意。

正確解答為：C　您的作答值為：C　答對了。TNO：U0030003
題目解析：
任何放在網路上的消息、資訊，皆有機會洩漏自己的隱私，因此放在網路上的資訊、發文或回應，皆須相當注意。尤其是身分證件、銀行帳戶資料等資料，絕對不能放在公開的網路上，以免被人竊取或盜用。

網路社群又稱為虛擬社群（Virtual Community），其特色為下列哪些？(複選題)
A　表達的自由
B　多對多的傳播
C　成員出自於自願的行為
D　集中式的管理
正確解答為：ABC　您的作答值為：ABCD　回答錯誤了。TNO：U0030004
題目解析：
Virtual Community虛擬社群一詞最先在Howard Rheingold於1993年出版的「虛擬社群」一書中被提出，在該書中指出虛擬社群是藉由網路連結所建構而成的一種虛擬空間與一種社會集合體。Rheingold認為虛擬社群是一種新型的社會組織，並有以下四種特質：1.表達的自由；2.多對多的傳播；3.成員出於自願的行為；4.缺乏集中的控制。

A　TelIP

D　TelNet
正確解答為：C　您的作答值為：C　答對了。TNO：U0030008

關於即時通訊軟體的使用，下列敘述哪一項正確？
A　即時通訊軟體登入會有紀錄，一定找得到是誰在使用電腦登入
B　即時通訊軟體的帳號密碼只要保管好，絕對沒有外洩的問題
C　通常即時通訊軟體已規範使用人的居住地，所以上網的地點一定是居住的所屬國家
D　即時通訊軟體中不要傳輸或溝通帳號、密碼或信用等個人資料，以維護資訊安全
正確解答為：D　您的作答值為：D　答對了。TNO：U0030009
題目解析：
A.即時通訊軟體僅需要帳號密碼就能夠登入，因此無法確定登入者是否本人，雖然目前有部分即時通訊軟體可以透過簡訊驗證方式綁定手機門號，但仍然無法保證是誰登入。B.不論是即時通訊軟體或是其他網路服務，都有帳號密碼外洩的風險。C.即時通訊軟體並不會規範使用人的居住地，且即時通訊軟體或其他網路服務，不論使用者身在何處，只要有帳號與密碼接能夠登入使用。

網路行銷及網拍產品的特質為下列哪些？(複選題)
A　網站環境具有親和力
B　可靠度高
C　產品數位化
D　產品價位較低
正確解答為：AD　您的作答值為：ACD　回答錯誤了。TNO：U0030010
題目解析：
網路行銷及網拍產品為了提升使用者的使用經驗與依賴度，其網站環境通常會具有較高的親和力，便於使用者使用。而且由於是透過網路進行交易，賣家省去了實體店面的營運成本，能夠將其省下的成本反映在產品售價上，因此網拍產品通常具有較低的價位。但相反的，由於購買者無法確保商品的進貨來源與實際狀況，不同賣場的商品其可靠度也有較大的落差，因此網路購物通常會有「收到商品與照片不符」、「詐騙」、「個資外洩」等擔憂。

在一般的社交網路中，大多數的用戶扮演的角色為下列哪一項？
A　閱讀者
B　創作者
C　分享者
D　管理者
正確解答為：A　您的作答值為：A　答對了。TNO：U0030013
題目解析：
根據Forrester Research研究機構對網路社群角色的分類，依據網路資訊分享的行為，區分成七種網路社交行為角色。分別是創作者（creators）、談論者（conversationalists）、評論者（critics）、蒐集者（collectors）、分享者（quoter）、參與者（joiners）以及閱讀者（spectators）。其中屬於閱讀者的比例超過50%，居於多數。

強調互動的Web2.0社交網路最主要的目標為下列哪一項？
A　增加3C產品的販售量
B　培養相關人才
C　增加創作與分享的人數
D　降低網路使用門檻
正確解答為：C　您的作答值為：C　答對了。TNO：U0030014
題目解析：
Web 2.0是一種新的網際網路方式，透過網路應用促進網路上人與人間的資訊交換和協同合作，其模式更加以用戶為中心，廣泛應用在網路社區、社群網站、部落格、Wiki等等。和傳統網站相比，Web 2.0社交網路能夠增加使用者間的互動性，提高使用者創作與分享的便利性，也能因此達到增加創作與分享人數的主要目標。
