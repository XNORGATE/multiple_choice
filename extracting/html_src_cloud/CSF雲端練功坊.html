下列哪一項為上網瀏覽網頁一定需要的軟體？
A　文書處理
B　試算表
C　簡報軟體
D　瀏覽器
正確解答為：D　您的作答值為：D　答對了。TNO：U0010001
題目解析：
Web Browser，網頁瀏覽器，簡稱瀏覽器，是用於檢索並顯示網頁的應用程式，讓使用者方便地瀏覽網頁、圖片、影音或其他多媒體內容。

瀏覽器主要解析的檔案格式為下列哪一項？
A　HTML檔案
B　EXE檔案
C　DOC檔案
D　RAR檔案
正確解答為：A　您的作答值為：A　答對了。TNO：U0010002
題目解析：
Web Browser主要解析HTML檔案，將其彩現成視覺化網頁。HTML，全名為HyperText Markup Language，是一種用於建立網頁的標準標記語言。

目前Microsoft最新的瀏覽器為何？
A　Internet Explorer
B　Edge
C　Chrome
D　Firefox
正確解答為：B　您的作答值為：B　答對了。TNO：U0010003
題目解析：
2015年4月29日，微軟在Build 2015開發者大會中，發布新瀏覽器Microsoft Edge，Microsoft Edge為Windows 10的預設瀏覽器,且不再開發Internet Explorer新版本。2016年1月6日，微軟宣布將會停止發布Internet Explorer 11之前版本的安全性更新，自此Internet Explorer已慢慢走入歷史，而由Microsoft Edge取代成為Windows預設瀏覽器。

14
在Windows環境下使用Chrome、Firefox、Edge或IE時，按下鍵盤上的[F5]鍵會發生下列哪一項情況？
A　網頁中的所有內容，均直接向伺服器端發送請求，並重新載入目前所在的網頁
B　重新在本機端再載入目前所在的網頁
C　關閉目前的視窗
D　清除目前瀏覽器的頁面
正確解答為：B　您的作答值為：B　答對了。TNO：U0010014
題目解析：
當使用者透過瀏覽器來存取Web伺服器時，會下載如HTML網頁、圖片、影片等一系列的內容，每種資訊的存取都會消耗伺服器資源與網路頻寬，為了減少網路與伺服器的負擔，瀏覽器提供了快取機制，讓內容暫存在瀏覽器內，避免相同的內容重複傳輸影響效率與伺服器資源。對於大部分的瀏覽器，一般有下列兩種更新網頁的方法：[F5]與[Ctrl]+[F5]。[F5]鍵是一般更新，使用瀏覽器的快取資源，重新在本機端再載入目前所在的網頁。而[Ctrl]+[F5]屬於強制更新，網頁中所有內容均向伺服器重新發送請求，不使用瀏覽器快取，可讓網頁取得所有元件的最新內容。

A　按下鍵盤上的[F5]鍵
B　按下瀏覽器上的「重新整理」按鈕
C　按下鍵盤上的[Ctrl]+[F5]鍵
D　按下「上一頁」按鈕，再按下「下一頁」按鈕
正確解答為：C　您的作答值為：C　答對了。TNO：U0010015
題目解析：
對於大部分的瀏覽器，一般有下列兩種更新網頁的方法：[F5]與[Ctrl]+[F5]。[F5]鍵是一般更新，使用瀏覽器的快取資源，重新在本機端再載入目前所在的網頁，與按下瀏覽器上的「重新整理」按鈕有相同作用。而[Ctrl]+[F5]屬於強制更新，網頁中所有內容均向伺服器重新發送請求，不使用瀏覽器快取，可讓網頁取得所有元件的最新內容。
