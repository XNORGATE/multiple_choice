小明開設色情貼圖網站，以招募會員的方式，對付費的會員提供色情圖片供其觀賞、拷貝。相關法律問題，下列敘述哪些正確？(複選題)
A　小明觸犯了刑法第235條
B　小明可能處三年以下有期徒刑
C　色情貼圖中如含有未滿二十歲以下之人為姦淫或猥褻行為之圖畫，則小明同時觸犯了兒童及少年性交易防制條例第28條
D　小華到小明開設的色情貼圖網站下載了圖片自行觀賞，並未構成刑法第235條之罪
正確解答為：AD　您的作答值為：D　回答錯誤了。TNO：U0070003
題目解析：
刑法第235條：「散布、播送或販賣猥褻之文字、圖畫、聲音、影像或其他物品，或公然陳列，或以他法供人觀覽、聽聞者，處二年以下有期徒刑、拘役或科或併科九萬元以下罰金。意圖散布、播送、販賣而製造、持有前項文字、圖畫、聲音、影像及其附著物或其他物品者，亦同。前二項之文字、圖畫、聲音或影像之附著物及物品，不問屬於犯人與否，沒收之。」色情貼圖中如含有未滿二十歲以下之人為姦淫或猥褻行為之圖畫，則小明同時觸犯了兒童及少年性交易防制條例第38條：「散布、播送或販賣兒童或少年為性交、猥褻行為之圖畫、照片、影片、影帶、光碟、電子訊號或其他物品，或公然陳列，或以他法供人觀覽、聽聞者，處三年以下有期徒刑，得併科新臺幣五百萬元以下罰金。意圖散布、播送、販賣或公然陳列而持有前項物品者，處二年以下有期徒刑，得併科新臺幣二百萬元以下罰金。查獲之前二項物品，不問屬於犯罪行為人與否，沒收之。」小華到小明開設的色情貼圖網站下載了圖片自行觀賞，並未散布、播送或販賣，因此並未構成刑法第235條之罪。

小明在購物網站購買電視，使用後發現不滿意想要退貨，下列敘述哪一項正確？
A　無論何時皆可退貨
B　退貨僅能取回一半價錢
C　七天內是可以退貨的
D　無論何時都不能夠退貨
正確解答為：C　您的作答值為：D　回答錯誤了。TNO：U0070004
題目解析：
依消保法第19條規定：「通訊交易或訪問交易之消費者，得於收受商品或接受服務後七日內，以退回商品或書面通知方式解除契約，無須說明理由及負擔任何費用或對價。但通訊交易有合理例外情事者，不在此限。」亦即網路購物、電視購物、電話行銷、路邊訪問推銷、賣場問卷調查推銷皆適用此條法規。但若在實體店面購物，不適用此七天鑑賞期的規定，除非有品質瑕疵或可歸責於業者的事由。另外依照「通訊交易解除權合理例外情事適用準則」規定，只要通訊交易的商品或服務有下列情形，並經企業經營者告知消費者，便可以排除消費者保護法第19條解除權之適用：一、易於腐敗、保存期限較短或解約時即將逾期。二、依消費者要求所為之客製化給付。三、報紙、期刊或雜誌。四、經消費者拆封之影音商品或電腦軟體。五、非以有形媒介提供之數位內容或一經提供即為完成之線上服務，經消費者事先同意始提供。六、已拆封之個人衛生用品。七、國際航空客運服務。

小英從網路上引用了一篇具有創用CC授權標註的文章，下列敘述哪一項正確？
A　使用時必須按照著作人或授權人所指定的方式，表彰其姓名
B　可以使用這篇文章於商業目的之用途
C　可以修改這篇文章的內容
D　不須標註這篇文章的原始出處
正確解答為：A　您的作答值為：D　回答錯誤了。TNO：U0070005
題目解析：
第一個BY標註表示使用時必須按照作者或者授權人指定的方式對作品進行姓名標示。第二個NC標註表示不得為商業目的而使用本作品。第三個ND標註表示不得改變、轉變或更改本作品。

7
在網站收集他人撰寫的文章，若未取得作者同意，可能會侵犯作者的下列哪一項權利？
A　著作權
B　專利權
C　肖像權
D　隱私權
正確解答為：A　您的作答值為：D　回答錯誤了。TNO：U0070007
題目解析：
依照著作權法第3條第1項第1款規定：「著作：指屬於文學、科學、藝術或其他學術範圍之創作。」另依著作權法第10條規定；「著作人於著作完成時享有著作權。」因此當他人撰寫文章後，就已受著作權所保護，若未取得作者同意，可能會侵犯作者的著作權。
8
小華破解了他人的帳號密碼並侵入其電腦中。相關法律問題，下列敘述哪些正確？(複選題)
A　小華觸犯了刑法第358條
B　小華將被處以三年以下有期徒刑
C　小華所犯之罪為非告訴乃論
D　小華在入侵後又在電腦內植入電腦病毒，則觸犯了刑法第361條
正確解答為：AB　您的作答值為：B　回答錯誤了。TNO：U0070008
題目解析：
刑法第358條規定：「無故輸入他人帳號密碼、破解使用電腦之保護措施或利用電腦系統之漏洞，而入侵他人之電腦或其相關設備者，處三年以下有期徒刑、拘役或科或併科三十萬元以下罰金。」刑法第361條規定：「對於公務機關之電腦或其相關設備犯前三條之罪者，加重其刑至二分之一。」小華在入侵後又在電腦內植入電腦病毒，是觸犯刑法第362條：「製作專供犯本章之罪之電腦程式，而供自己或他人犯本章之罪，致生損害於公眾或他人者，處五年以下有期徒刑、拘役或科或併科六十萬元以下罰金。」刑法第363條規定：「第三百五十八條至第三百六十條之罪，須告訴乃論。」，因此小華所犯之罪屬於告訴乃論。
9
在我國於網路上搶先佔用網域名稱的行為，可能觸犯下列哪一項法律？
A　專利法
B　著作權法
C　商標法
D　公平交易法
正確解答為：C　您的作答值為：D　回答錯誤了。TNO：U0070009
題目解析：
這種行為稱為域名搶注（Cybersquatting），指搶先一步去登記含有著名企業或名人商標的網際網路域名後，再高價賣給該企業或名人以賺取暴利的行為。實施域名搶注的人亦被稱爲網路蟑螂。商標法第70條規定：「未得商標權人同意，有下列情形之一，視為侵害商標權：一、明知為他人著名之註冊商標，而使用相同或近似之商標，有致減損該商標之識別性或信譽之虞者。二、明知為他人著名之註冊商標，而以該著名商標中之文字作為自己公司、商號、團體、網域或其他表彰營業主體之名稱，有致相關消費者混淆誤認之虞或減損該商標之識別性或信譽之虞者。三、明知有第六十八條侵害商標權之虞，而製造、持有、陳列、販賣、輸出或輸入尚未與商品或服務結合之標籤、吊牌、包裝容器或與服務有關之物品。」因此網路蟑螂之搶先註冊他人商標為網域名稱之行為，已可能違反商標法第70條之規定。

下列哪一項為智慧財產權所保護的內容？
A　一般人知的權利
B　人類腦力辛勤創作的結晶
C　國家尊嚴
D　消費者消費的樂趣
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：U0070010
題目解析：
依據「世界智慧財產權組織公約」的規定，智慧財產權包括：一、文學、藝術及科學之著作。二、演藝人員之演出、錄音物以及廣播。三、人類之任何發明。四、科學上之發現。五、產業上之新型及新式樣。六、製造標章、商業標章及服務標章，以及商業名稱與營業標記。七、不公平競爭之制止。八、其他在產業、科學、文學及藝術領域中，由精神活動所產生之權利。因此智慧財產權是各國法律為了保護人類腦力辛勤創作的結晶，而創設各種權益或保護規定的統稱。

下列哪些不是在網路上發表文章時應有的態度？(複選題)
A　就事論事，避免人身攻擊
B　雖然是未經證實的消息，也要先讓大家知道
C　可以全盤引用別人的文章，自己意見加不加無所謂
D　轉載別人的文章時，註明出處
正確解答為：BC　您的作答值為：B　回答錯誤了。TNO：U0070011
題目解析：
在網路上發表文章時，應先證實其消息是否正確再發表，否則若是散布假訊息，可能觸犯社會秩序維護法第63條第1項第5款規定：「散布謠言，足以影響公共之安寧者」，另外如果散布的假訊息涉及侵害他人的名譽或信用，可能觸犯刑法第310條的「誹謗罪」或刑法第313條的「妨害信用罪」。另外若引用別人的文章，需要先徵求對方同意，若未取得作者同意，可能會侵犯作者的著作權。

下列哪一項行為是違反法律的？
A　在自己的隨身碟上儲存一張自拍的裸照
B　在網路留言板上辱罵老師
C　在家用電腦連上色情網站
D　在傳給好朋友的電子郵件中抱怨知名牛肉麵店的麵食不乾淨
正確解答為：B　您的作答值為：D　回答錯誤了。TNO：U0070013
題目解析：
在網路留言板上辱罵老師或其他人，可能涉及刑法第309條的「公然侮辱罪」或刑法第310條的「誹謗罪」，另外被害人也可依民法第18條、184條、195條請求民事賠償。因此不管是在公共場合或是網路上，自己的言行舉止都要相當謹慎。刑法第309條：「公然侮辱人者，處拘役或九千元以下罰金。以強暴犯前項之罪者，處一年以下有期徒刑、拘役或一萬五千元以下罰金。」刑法第310條：「意圖散布於眾，而指摘或傳述足以毀損他人名譽之事者，為誹謗罪，處一年以下有期徒刑、拘役或一萬五千元以下罰金。散布文字、圖畫犯前項之罪者，處二年以下有期徒刑、拘役或三萬元以下罰金。對於所誹謗之事，能證明其為真實者，不罰。但涉於私德而與公共利益無關者，不在此限。」
14
在西元1998年撰寫CIH病毒的台灣學生沒有受到法律制裁，主要因素為下列哪些？(複選題)
A　沒有造成損害
B　當時的法律沒有明確的規定與法條
C　沒有人檢舉
D　該學生沒有犯意或故意散播病毒
正確解答為：BD　您的作答值為：B　回答錯誤了。TNO：U0070014
題目解析：
在1998年時，大同大學資訊工程系的學生陳盈豪，為驗證防毒軟體號稱百分百防毒是不實廣告，而寫出了一隻電腦病毒，稱為CIH病毒，能夠躲避防毒軟體的偵測，最終感染了全球數百萬台電腦，在某些電腦上，CIH病毒不僅會破壞硬碟，甚至會感染電腦的BIOS，成為當時世界知名的事件。1998年陳盈豪寫出病毒後，在他不知情的狀況下，他的同學使用了實驗用電腦，而將此病毒攜出，進而在網際網路擴散開來，在1999年4月26日，CIH病毒在全球各地大規模爆發，但由於當時台灣沒有任何對於電腦病毒的刑事法令，因此陳盈豪獲不起訴處分。我國於民國92年6月25日增定刑法第三十六章「妨害電腦使用罪」，即是為了有效嚇阻類似的犯罪行為，其中第360條之「干擾他人電腦使用罪」，便是明確處罰散佈電腦病毒之規定。
15
小明寫了一個病毒程式，其行為是否違反法律，下列敘述哪一項正確？
A　若自己沒有散佈就沒有違法
B　屬於個人創作的自由，沒有違法
C　若造成公眾的損害就違法
D　造成的損害超過法定的規定才算違法
正確解答為：C　您的作答值為：D　回答錯誤了。TNO：U0070015
題目解析：
撰寫病毒程式只要造成公眾或他人的損害就違法，可能觸犯刑法第358條至363條，其中又以第360條與第362條最為相關。刑法第360條：「無故以電腦程式或其他電磁方式干擾他人電腦或其相關設備，致生損害於公眾或他人者，處三年以下有期徒刑、拘役或科或併科三十萬元以下罰金。」刑法第362條：「製作專供犯本章之罪之電腦程式，而供自己或他人犯本章之罪，致生損害於公眾或他人者，處五年以下有期徒刑、拘役或科或併科六十萬元以下罰金。」。
