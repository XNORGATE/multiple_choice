Google提供的Google App Engine（GAE）屬於下列哪一項雲端服務？
A　SaaS（Software as a Service）
B　IaaS（Infrastructure as a Service）
C　CaaS（Content as a Service）
D　PaaS（Platform as a Service）
正確解答為：D　您的作答值為：D　答對了。TNO：U0040046
題目解析：

進行MapReduce運算時，MapReduce程式會將輸入資料分割為多個block，Hadoop 2.x預設的分割block size大小為下列哪一項？
A　16MB
B　32MB
C　64MB
D　128MB
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：U0040047
題目解析：
Hadoop在不同版本有不同的預設block size，Hadoop 2.x預設為128 MB，Hadoop 1.x預設則為64 MB。
3
Google的MapReduce雲端運算所採用的儲存檔案系統為下列哪一項？
A　GFS
B　HDFS
C　SQL
D　SQLite
正確解答為：A　您的作答值為：C　回答錯誤了。TNO：U0040048
題目解析：
GFS（Google File System，Google檔案系統）是由Google公司開發的一種分散式檔案系統，也是Google的MapReduce雲端運算採用的儲存檔案系統。HDFS（Hadoop Distributed File System）是Yahoo提出的Hadoop內部使用的分散式檔案系統。SQL與SQLite則是關聯式資料庫管理系統。

Yahoo!的MapReduce雲端運算所採用的儲存檔案系統為下列哪一項？
A　GFS
B　HDFS
C　SQL
D　SQLite
正確解答為：B　您的作答值為：C　回答錯誤了。TNO：U0040049
題目解析：
Yahoo!的MapReduce雲端運算採用的儲存檔案系統為HDFS（Hadoop Distributed File System），HDFS是Yahoo!提出的Hadoop內部使用的分散式檔案系統。

A　Windows Server 2005
