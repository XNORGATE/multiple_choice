A　%
B　&
C　*
D　#
正確解答為：D　您的作答值為：D　答對了。TNO：U0030038
題目解析：
#這個符號在網際網路中稱為Hashtag（主題標籤），又稱為話題標記，是網際網路的元資料標籤類型之一，由一個井號加上一個詞、單字，或沒有空格的一句話所構成。通常使用在社群網站的貼文中，表示貼文的話題標記，如此一來使用者便可以藉著各種題標連結到平台內標記有相同題標的貼文。
