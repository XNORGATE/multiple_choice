下列哪些網路技術可用來作為企業內部網路（Intranet）連線使用？(複選題)
A　非對稱數位用戶線路（ADSL）
B　纜線數據機（Cable Modem）
C　乙太網路（Ethernet）
D　租用專線（Leased Line）
正確解答為：CD　您的作答值為：AD　回答錯誤了。TNO：U0040016
題目解析：
Intranet其實就是將Internet的網路型態運用於公司或企業內部的網路資訊作業，企業內部網路連線需要考量到連線的安全性，因此可以透過Ethernet自行架設，或是透過租用專線（Leased Line）的方式，提供連接兩個地區的高速線路。ADSL（Asymmetric Digital Subscriber Line）是利用一般電話線來進行上網的技術。Cable Modem是利用有線電視系統的同軸電纜線來進行作為上網的設備。ADSL與Cable Modem皆不適合用於企業內部網路連線使用。

下列哪些網路應用類型是Web2.0的代表性服務？(複選題)
A　電子郵件（Email）
B　部落格（Blog）
C　維基百科（Wikipedia）
D　電子布告欄（Bulletin Board System, BBS）
正確解答為：BC　您的作答值為：ABD　回答錯誤了。TNO：U0040017
題目解析：
Web 2.0網站允許用戶作為虛擬社群中用戶生成內容的建立者，並通過對話進行互動和協同運作，與Web 1.0人們只能被動地瀏覽內容的網站形成了對比。Web 2.0功能的範例，包括：社群網站、部落格、Wikipedia、影片分享網站、代管服務等服務。

下列哪一項標準協定讓電子郵件，可以傳輸圖像及聲音等非文字的訊息？
A　POP3（Post Office Protocol 3）
B　IMAP（Internet Message Access Protocol）
C　MIME（Multipurpose Internet Mail Extension）
D　SMTP（Simple Mail Transfer Protocol）
正確解答為：C　您的作答值為：B　回答錯誤了。TNO：U0040018
題目解析：
MIME（Multipurpose Internet Mail Extentions）解決傳輸多種型態訊息的困難，並且強化壓縮及加密的能力，各種媒體型態的訊息透過支援MIME協定的電子郵件軟體，就可以完成更豐富的電子郵件功能。POP3（Post Office Protocol, version 3）為收信主機專屬之協定。IMAP（Internet Mail Access Protocol）協定允許透過遠端讀取信件的方式收發投遞。SMTP（Simple Mail Transfer Protocol）是一個在網際網路上傳輸電子郵件的標準。

B　IMAP（Internet Message Access Protocol）
C　HTTPS（Hypertext Transfer Protocol Secure）

下列哪些通訊協定為用來加密電子郵件？(複選題)
A　SMTP（Simple Mail Transfer Protocol）
B　HTTPS（Hypertext Transfer Protocol Secure）
C　S/MIME（Secure/Multipurpose Internet Mail Extension）
D　PGP（Pretty Good Privacy）
正確解答為：CD　您的作答值為：C　回答錯誤了。TNO：U0040020
題目解析：
S/MIME（Secure Multipurpose Internet Mail Extensions）是一種Internet標準，在安全方面對MIME協議進行了擴展，可以將MIME實體封裝成安全文件，為電子郵件應用增添了消息真實性、完整性和保密性服務，S/MIME不局限於電子郵件，也可以被其他支援MIME的傳輸機制使用。PGP（Pretty Good Privacy）是一套用於訊息加密、驗證的機制，可用於傳送機密訊息。SMTP（Simple Mail Transfer Protocol）是一個在網際網路上傳輸電子郵件的標準。HTTPS（HyperText Transfer Protocol Secure，超文本傳輸安全協定）是一種透過網際網路進行安全通訊的傳輸協定。

下列哪一項通訊協定主要功能是自動分配IP位址與相關參數？
A　DNS（Domain Name System）
B　DHCP（Dynamic Host Configuration Protocol）
C　ARP（Address Resolution Protocol）
D　SNMP（Simple Network Management Protocol）
正確解答為：B　您的作答值為：B　答對了。TNO：U0040021
題目解析：
DHCP（Dynamic Host Configuration Protocol，動態主機設定協定）是一個用於IP網路的網路協定，位於OSI模型的應用層，使用UDP協定工作，主要用途為內部網路或網路服務供應商自動分配IP位址給用戶以及內部網路管理員對所有電腦作中央管理。DNS（Domain Name System，網域名稱系統）是網際網路的一項服務，是將網域名稱和IP位址相互對映的一個分散式資料庫，使人更方便地存取網際網路。ARP（Address Resolution Protocol，位址解析協定）是一個通過解析IP Address來找尋MAC Address的網路傳輸協定。SNMP（Simple Network Management Protocol，簡單網路管理協定）該協定能夠支援網路管理系統，用以監測連接到網路上的裝置是否有任何狀況。

利用DNS的反查服務來確認郵件伺服器是否為合法的主機，此為垃圾郵件（SPAM Mail）的防護機制之一，其使用下列哪一項紀錄類型？
A　A主機紀錄
B　CNAME別名紀錄
C　MX郵件交換紀錄
D　PTR指標紀錄
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：U0040022
題目解析：
DNS系統中，常見的資源記錄類型有：主機記錄（A記錄）：A記錄是用於名稱解析的重要記錄，它將特定的主機名對映到對應主機的IP位址上。別名記錄（CNAME記錄）：CNAME記錄用於將某個別名指向到某個A記錄上，這樣就不需要再為某個新名字另外建立一條新的A記錄。IPv6主機記錄（AAAA記錄）：與A記錄對應，用於將特定的主機名對映到一個主機的IPv6位址。服務位置記錄（SRV記錄）：用於定義提供特定服務的伺服器的位置，如hostname、port number等。PTR記錄：提供了正規表示式方式去對映一個域名，定義某個IP對應的domain name，即將IP位址轉換成主機的FQDN。郵件交換記錄(MX記錄)：是用於指定負責處理發往收件人域名的郵件伺服器。

BT（BitTorrent）是目前著名的一種P2P（Peer-to-Peer）檔案傳輸模式，下列哪一項功能是BT的主要特性？
A　快速尋找檔案
B　保證檔案的頻寬
C　加強檔案傳輸的安全
D　透過分享加快檔案傳輸的速度
正確解答為：D　您的作答值為：D　答對了。TNO：U0040023
題目解析：
BitTorrent協定（簡稱BT）是用在對等網路中檔案分享的網路協定程式，它是P2P（peer-to-peer）傳輸模式，使用的用戶越多、下載同一檔案的人越多，下載該檔案的速度就越快。

A　PPPoE（Point-to-Point Protocol over Ethernet）
B　VoIP（Voice over Internet Protocol）
C　IPSec（Internet Protocol Security）
D　VPN（Virtual Private Network）
正確解答為：B　您的作答值為：B　答對了。TNO：U0040026
題目解析：
