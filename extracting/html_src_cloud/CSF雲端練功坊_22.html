1
寄居架構（Hosted）需要一個作業系統作為底層，其底層稱之為下列哪一項？
A　主機作業系統（Host OS）
B　虛擬機作業系統（Guest OS）
C　虛擬機監視器（Hypervisor）
D　虛擬機（Virtual Machine, VM）
正確解答為：A　您的作答值為：A　答對了。TNO：U0060031
題目解析：
在寄居架構中，直接安裝在主機硬體上的作業系統稱為Host OS，安裝在虛擬機中的作業系統稱為Guest OS。
2
不論是寄居架構（Hosted）或原生架構（Native），所有虛擬化產品都需要下列哪一項功能而達成虛擬化？
A　主機作業系統（Host OS）
B　虛擬機作業系統（Guest OS）
C　虛擬機監視器（Hypervisor）
D　虛擬機（Virtual Machine, VM）
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：U0060032
題目解析：
不論原生架構或寄居架構，所有虛擬化產品都需要有虛擬機監視器（Hypervisor，或稱Virtual Machine Monitor），用來建立與執行虛擬機的軟體、韌體或硬體，以提供虛擬的平台來執行虛擬機，也負責管理其他虛擬機的執行。
3
下列哪一項不是Hypervisor的功能？
A　接管實體硬體，模擬虛擬硬體
B　在虛擬機（Virtual Machine, VM）之間分配實體硬體資源
C　提供一個管理介面給使用者
D　提供作業系統的桌面
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：U0060033
題目解析：
Hypervisor主要功能是建立與執行虛擬機的軟體、韌體或硬體，以提供虛擬的平台來執行Guest OS，因此作業系統的桌面是由Guest OS而非Hypervisor提供。
4
當寄居架構的主機作業系統（Host OS）當機時，下列敘述哪一項錯誤？
A　Hypervisor一定無法繼續運作
B　虛擬機（Virtual Machine, VM）一定無法繼續運作
C　虛擬機作業系統（Guest OS）一定無法運作
D　實體硬體一定無法運作
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：U0060034
題目解析：
寄居架構是在原作業系統上運行一層Hypervisor，做Guest OS與Host OS的溝通，以取得伺服器硬體資源，因此當Host OS當機時，依賴宿主作業系統的Hypervisor、虛擬機、Guest OS會隨之無法運行，若Host OS當機並非硬體損壞導致，則實體硬體仍可正常運作。
5
一個執行虛擬化環境的硬體伺服器，可同時執行多少個開機狀態的作業系統（Operating System, OS），下列敘述哪一項正確？
A　無限多個
B　視硬體容量而定
C　視中央處理器（Central Processing Unit, CPU）、虛擬機作業系統（Guest OS）的負載及記憶體的容量而定
D　視網路卡頻寬而定
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：U0060035
題目解析：
一台伺服器上能同時執行的虛擬機數量，需要同時考量伺服器所配備的CPU數量、CPU頻率、RAM大小，以及執行的作業系統其負載與RAM使用量等多種因素。
6
下列哪一項不是虛擬化最大的好處之一？
A　可增加中央處理器（Central Processing Unit, CPU）的使用率
B　可快速、彈性、自動生成作業系統
C　可合理分配硬體資源
D　可降低記憶體的使用量
正確解答為：D　您的作答值為：B　回答錯誤了。TNO：U0060036
題目解析：
所有虛擬化產品都需要有Hypervisor提供虛擬的平台來執行虛擬機，每個虛擬機除了會根據其設定大小消耗記憶體外，亦會消耗額外的記憶體用於虛擬化，因此虛擬化只會增加記憶體的使用率，並無法降低記憶體的使用量。
7
關於虛擬化，下列敘述哪一項錯誤？
A　就是一台硬體上交換啟動多個作業系統（Operating System, OS）
B　可同時間在一台硬體上執行多個OS
C　可同時間讓多個OS之間用實體機（如：網路連接）的方式連接
D　可同時讓實體硬碟有多個虛擬硬碟
正確解答為：A　您的作答值為：B　回答錯誤了。TNO：U0060037
題目解析：
虛擬化能讓一台實體伺服器能同時執行多個虛擬機與作業系統，而非交換啟動。
8
虛擬化上的虛擬機作業系統（Guest OS）以及虛擬硬體合稱為一個虛擬機（Virtual Machine, VM），關於VM，下列敘述哪一項錯誤？
A　VM的硬體是由實體硬體分配出來的
B　VM在建立時就開始使用實體硬體
C　VM可使用實體網路卡
D　VM可使用實體硬碟
正確解答為：B　您的作答值為：C　回答錯誤了。TNO：U0060038
題目解析：
所有虛擬化產品都需要有Hypervisor提供虛擬的平台來執行虛擬機，因此虛擬機在建立時，是使用Hypervisor提供的虛擬化硬體，而非直接使用實體硬體。
9
關於虛擬化的技術，下列敘述哪一項錯誤？
A　虛擬機（Virtual Machine, VM）是由檔案所組成
B　VM可複製到其它的實體機開機
C　VM在複製到其它同硬體設備的實體機時，虛擬機作業系統（Guest OS）需要重新安裝驅動程式
D　VM在其它實體機啟動時，行為和原來的實體機一樣
正確解答為：C　您的作答值為：D　回答錯誤了。TNO：U0060039
題目解析：
VM在複製到其它同硬體設備的實體機時，只要使用相同的Hypervisor，即可直接啟動原本的Guest OS，而不需要重新安裝驅動程式。以VMware Workstation為例，如下圖所示，虛擬機是由一組虛擬磁碟和組態檔組成，如.vmx虛擬機組態檔、.vmxf其他虛擬機組態檔、.vmdk虛擬磁碟特性、-flat.vmdk虛擬磁碟資料檔、.nvram虛擬機BIOS或EFI組態檔、.vmsd為虛擬機快照說明檔、.vmsn為虛擬機快照資料檔、.vswp為虛擬機分頁檔、.log為目前虛擬機紀錄檔等檔案。
10
關於虛擬化技術，下列敘述哪一項錯誤？
A　新一代的虛擬化技術為純軟體技術
B　不同硬體上安裝的相同Hypervisor會因為硬體種類而產生相容性問題
C　各大硬體廠商已推出專門針對虛擬化技術的產品
D　不同硬體上能執行虛擬化的技術不同
正確解答為：A　您的作答值為：C　回答錯誤了。TNO：U0060040
題目解析：
隨著技術的發展演進，已經有許多過去透過軟體來執行的工作，轉由硬體直接處理，使系統所需執行的任務能夠以更高效能的方式處理；現在不僅CPU，包括Memory、Chipset、網路、I/O處理等，都成為提升虛擬化效能的關鍵角色，因此在虛擬化技術上，硬體與軟體兩者是相輔相成的，透過軟硬體的配合，才能夠提高虛擬化效能。
11
Hadoop雲端運算由下列哪一項團隊開發？
A　Google
B　Yahoo!
C　Apache
D　Facebook
正確解答為：C　您的作答值為：C　答對了。TNO：U0060041
題目解析：
Apache Hadoop是一款支援資料密集型分散式應用程式，並以Apache 2.0許可協定發布的開源軟體框架。Hadoop是根據Google發表的MapReduce和GFS（Google File System）的論文實作而成。
12
Hadoop在Apache基金會網站中有其他的開放原始碼套件，組成一個Hadoop生態體系（Hadoop Ecosystem），其具備下列哪些能力？(複選題)
A　HBase
B　Hive
C　Pig
D　Mahout
正確解答為：ABCD　您的作答值為：C　回答錯誤了。TNO：U0060042
題目解析：
Hadoop的定位是用來處理與保存大量資料的雲端運算平台，在Hadoop中包含了分散式檔案系統（HDFS）、MapReduce框架、儲存系統（HBase）等元件外，亦有許多元素構成：Hadoop Common：Hadoop體系最底層的一個模組，爲Hadoop各子項目提供各種工具，如：配置文件和日誌操作等。Core：用於分散式檔案系統和一般性I/O之用的元件和介面。HBase：基於Hadoop Distributed File System，基於列存儲模型的可擴展的分散式資料庫，支援儲存大型的結構化數據。Avro：主要負責數據的序列化，提供高效能、跨語言以及可保存資料的RPC資料序列化系統。Hive：分散式資料倉儲，透過Hive可管理存放於HDFS的資料，並提供根據SQL發展的查詢語言來查詢資料。Pig：超大資料集的資料流語言以及執行環境，可在HDFS和MapReduce叢集環境中執行。ZooKeeper：Google Chubby的一種實現，提供分散式且高可用性的協調服務，可為建置分散式系統提供分散式鎖定等原始鎖定功能。Chukwa：分散式資料收集和分析系統，其會執行收集器以便在HDFS中儲存資料，且會使用MapReduce來產生報表。Mahout：機器學習函式庫，包含許多建立叢集和分類與推薦的工具。
13
Hadoop雲端運算檔案系統為下列哪一項？
A　GFS
B　HDFS
C　HBase
D　BigTable
正確解答為：B　您的作答值為：D　回答錯誤了。TNO：U0060043
題目解析：
HDFS（Hadoop Distributed File System）是Yahoo提出的Hadoop內部使用的分散式檔案系統。GFS（Google File System，Google檔案系統）是由Google公司開發的一種分散式檔案系統，也是Google的MapReduce雲端運算採用的儲存檔案系統。HBase是基於HDFS的可擴展的分散式資料庫。BigTable是建立在GFS和MapReduce之上管理結構化數據的分布式儲存系統。

如果想要讓Windows 7和Windows 10共存並交換資料，最省成本的方案為下列哪一項？
A　雙重啟動Windows 7和Windows 10
B　找兩台電腦各安裝一套系統
C　使用VMware Workstation建立兩個虛擬機（Virtual Machine, VM）
D　使用Windows 10為主機作業系統（Host OS），並使用內建的Windows Virtual PC安裝一個Windows 7的VM
正確解答為：D　您的作答值為：C　回答錯誤了。TNO：U0060044
題目解析：
要讓兩個作業系統共存並交換資料，最簡單的方案為透過寄居架構將其中一個作業系統做為Host OS，並透過Hypervisor（Windows Virtual PC、VMware Workstation等軟體）安裝另一個作業系統為Guest OS，再透過Hypervisor提供的工具程式，或是在Guest OS設定共用資料夾後在Host OS中掛載網路磁碟機等方式，即可直接交換兩個作業系統間的資料。
15
影響一個虛擬化平台執行效率的最顯著原因為下列哪一項？
A　中央處理器（Central Processing Unit, CPU）的數量或頻率
B　記憶體的數量
C　硬碟的速度
D　網路的速度
正確解答為：C　您的作答值為：C　答對了。TNO：U0060045
題目解析：
影響虛擬化平台執行效率最顯著的是硬碟的速度，這是由於所有的資料都是存放在硬碟中，處理前需要先由硬碟讀取出來暫放在記憶體中等待CPU處理，或是處理完的資料亦需要儲存回硬碟中，相較於CPU與記憶體的速度，硬碟的讀取速度相當緩慢，因此硬碟是影響虛擬化平台執行效率最顯著的原因。
