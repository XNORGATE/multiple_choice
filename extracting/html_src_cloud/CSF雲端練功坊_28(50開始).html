常見的網路攻擊型態主要有下列哪些？(複選題)
A　修改IP封包，以假造的IP位址來扮演其他使用者
B　利用猜密碼程式破解帳號密碼
C　從網路上竊取IP封包，讀取通訊內容
D　切斷伺服器的網路連線
正確解答為：ABC　您的作答值為：ABC　答對了。TNO：U0080001
題目解析：
網路攻擊類型大致上可分為主動式攻擊與被動式攻擊，主動式攻擊是指入侵者針對檔案或通訊內容進行偽造或修改，大致上可分為四種方法：偽裝（Masquerade）、重播（Replay）、訊息竄改（Message Modification）、服務阻絕（Denial of Service）。被動式攻擊則是指入侵者非法取得資訊資產的存取權限，但是並未對其內容進行竄改，大致上有兩種方法：竊聽（Eavesdropping）、通訊分析（Traffic Analysis）。而目前常見的網路攻擊型態有：TCP SYN、Smurf Attack、Ping of Death、LAND Attack、Teardrop Attack、電腦病毒、蠕蟲、木馬程式、連接埠掃瞄、密碼攻擊、連線截奪、重送攻擊、緩衝區溢位、竊聽、社交工程、資料隱碼等方法。

關於加密法，下列敘述哪些錯誤？(複選題)
A　數位簽章使用對稱式加密演算法來運作
B　一般使用對稱式加密演算法來傳遞非對稱式加密演算法的金鑰
C　一般來說對稱式加密演算法較非對稱式加密演算法之加密速度快
D　對稱式加密演算法中加密者與解密者是使用不一樣的金鑰
正確解答為：ABD　您的作答值為：CD　回答錯誤了。TNO：U0080002
題目解析：
數位簽章使用的是非對稱加密技術，用私鑰加密，用公鑰解密。為了顧及金鑰安全與加解密效能，通常都是用非對稱式加密演算法來傳遞對稱式加密演算法的金鑰。對稱式加密演算法中加密與解密使用的是同一把金鑰。

關於非對稱式加密演算法，下列敘述哪些正確？(複選題)
A　具有一把公鑰以及一把私鑰
B　SHA-1為非對稱式加密演算法的代表
C　加解密速度極快，很適合大量資料
D　加解密都使用同一對金鑰
正確解答為：AD　您的作答值為：CD　回答錯誤了。TNO：U0080003
題目解析：
非對稱式加密演算法的代表為RSA演算法，SHA-1與MD5都是屬於一種雜湊函數。非對稱式加密演算法會有一對金鑰：公鑰（Public key）及私鑰（Private key）。

關於傳遞加密文件，下列敘述哪些錯誤？(複選題)
A　為了安全，每次傳遞文件主體前都會產生一組對稱式交談金鑰（Session Key），僅供該次傳遞資料使用
B　在傳遞資料主體前，先使用交談金鑰將文件主體進行加密
C　寄件者利用自己的公開金鑰（Public Key），將交談金鑰加密後送給收件者
D　收件者收到加密文件後，使用自己的私密金鑰（Private Key）將文件主體解密閱讀
正確解答為：CD　您的作答值為：B　回答錯誤了。TNO：U0080004
題目解析：
寄件者需利用收件者的公開金鑰（Public Key）將交談金鑰（Session Key）進行加密後送給收件者，收件者收到之後，再用自己的私密金鑰（Private Key）進行解密取得交談金鑰。在雙方交換交談金鑰完成後，寄件者會使用交談金鑰與對稱式加密演算法對加密文件主體進行加密後傳遞，收件者收到加密文件後，透過先前交換的交談金鑰進行解密。

關於數位簽章，下列敘述哪些正確？(複選題)
A　數位簽章就是把自己的簽名或印章掃描成圖檔，附在電子文件上
B　經過數位簽章的文件是容易被驗證的
C　數位簽章具有不可否認性
D　數位簽章就是利用筆式輸入系統在電子文件上簽名
正確解答為：BC　您的作答值為：B　回答錯誤了。TNO：U0080005
題目解析：
數位簽章使用的是非對稱加密技術，每個人都有一對私鑰與公鑰，簽名的時候用私鑰，驗證簽名的時候用公鑰。公鑰必須向身份認證機構註冊，註冊後身份認證機構會給使用者一個數位證書，使用者把此數位證書連同文件及簽章一起發給收件者，收件者可向身份認證機構求證其簽章是否真的為該使用者，因此數位簽章具有不可否認性。

關於自然人憑證，下列敘述哪一項錯誤？
A　包含了數位簽章與公開金鑰
B　私密金鑰存在IC卡中
C　由內政部憑證管理中心所簽發
D　自然人憑證並無有效期限的限制
正確解答為：D　您的作答值為：B　回答錯誤了。TNO：U0080009

關於SSL（Secure Sockets Layer），下列敘述哪一項錯誤？
A　SSL是一種安全通訊協定
B　網址如果是https://開頭，表示受到SSL/TLS保護
C　目前SSL仍是安全的保密協定
D　可以防止資料在傳輸過程中被第三者竊取而了解內容
正確解答為：C　您的作答值為：B　回答錯誤了。TNO：U0080010
題目解析：
SSL（Secure Sockets Layer，安全通訊協定）目的是為網際網路通訊提供安全及資料完整性保障，不具備客戶端身分辨識功能，且2014年10月，Google發布在SSL 3.0中存在的設計缺陷，建議禁用此一協定，並且在2015年，RFC 7568標準棄用了SSL 3.0，目前較安全的協定為TLS 1.3。

在網路的線上交易環境中，目前常被應用的安全機制為下列哪一項？
A　CSS
B　XML
C　GFS
D　SET
正確解答為：D　您的作答值為：B　回答錯誤了。TNO：U0080011
題目解析：
SET（Secure Electronic Transactions，安全電子交易協議）是由Visa和Master Card聯合Netscape，Microsoft等公司，於1997年推出的一種電子支付模型，是用來確保開放網路上使用信用卡進行線上交易的安全性。CSS（Cascading Style Sheets，階層式樣式表）是一種用來為結構化文件（如HTML）添加樣式的電腦語言。XML（Extensible Markup Language，可延伸標記式語言）是一種標記式語言，透過電腦所能理解的資訊符號（標記），讓電腦之間可以處理包含各種資訊的資料。GFS（Google File System，Google檔案系統）是由Google公司開發的一種分散式檔案系統，也是Google的MapReduce雲端運算採用的儲存檔案系統。

關於IPSec，下列敘述哪一項正確？
A　IPSec不具有認證功能
B　建立區域網路內部的安全通訊並非IPSec的功能之一
C　IPSec僅能運作於IPv6環境下
D　IPSec具有保密功能
正確解答為：D　您的作答值為：B　回答錯誤了。TNO：U0080012
題目解析：
IPSec（Internet Protocol Security，網際網路安全協定）是一個透過對IP封包進行加密和認證，藉此來保護IP協定的網路傳輸協定組。IPsec主要由以下協定組成：1.AH（Authentication Header），保證被傳輸封包的完整性和可靠性，此外，它還保護資料不受Replay Attack的攻擊。2.ESP（Encapsulating Security Payload），對封包提供了源可靠性、完整性和保密性的支援。3.SA（Security Association），定義如何協商，還有要使用哪些策略和參數。

在密碼學中，替代（Substitution）是基本的加密技術，最簡單的凱撒加密法是將明文中的所有字母都在字母表上向後或向前按照一個固定數目進行偏移後被替換成密文，假設以凱撒加密法來加密明文「APPLE」，下列哪一項不是凱撒加密法加密後的密文？
A　BQQMF
B　LEAPP
C　DSSOH
D　HWWSL
正確解答為：B　您的作答值為：B　答對了。TNO：U0080013
題目解析：
替換式密碼（取代加密法）是按規律將文字加密的一種方式，替換式密碼亦有許多不同類型，如簡易替換加密、單表加密、多表加密等。簡易替換加密是一種以特定方式改變字母表上字母順序，並以此順序書寫的加密方式，最簡單的凱薩加密法，是將明文中的所有字母都在字母表上向後或向前按照一個固定數目進行偏移後被替換成密文。如下圖所示，選項A「BQQMF」是往右Shift 1的加密，選項C「DSSOH」是往右Shift 3的加密，選項D「HWWSL」是往右Shift 7的加密，選項B「LEAPP」則是將明文順序交換位置，不屬於凱薩加密法。
