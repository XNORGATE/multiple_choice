家裡使用電信公司的ADSL寬頻上網時，來自電話網路的局線、ADSL數據機、電話及電腦的網路線之間的連接方式，下列敘述哪一項錯誤？
A　電話要跟寬頻分享器直接以電話線相連
B　來自電腦的網路線要連接到ADSL數據機的網路接孔
C　來自電話的電話線要連接到ADSL數據機的電話線接孔
D　來自電話網路的局線要連接ADSL數據機
正確解答為：A　您的作答值為：A　答對了。TNO：U0050031
題目解析：
ADSL連接方式如下圖所示，ADSL數據機基本上只有「電源」、「電話線」、「網路線」三種接孔，來自電話網路的電話線（局線）要連接到ADSL數據機的電話線接孔，電腦的網路線要與ADSL數據機的網路線接孔連接，若要讓多台電腦能同時上網，可以將Hub、Switch、Router或無線寬頻分享器連接ADSL數據機提供的網路接孔，來分享網路連線。

關於動態主機設定協定（Dynamic Host Configuration Protocol, DHCP），下列敘述哪一項錯誤？
A　屬於OSI模型中網路層（Network Layer）的協定
B　可以讓網路上的電腦自動取得IP位址與相關的參數
C　可動態地分配IP位址，減少網路管理人員的工作負擔
D　DHCP伺服器能接受網路節點的請求，動態地分配IP位址
正確解答為：A　您的作答值為：A　答對了。TNO：U0050032
題目解析：
DHCP（Dynamic Host Configuration Protocol，動態主機設定協定）是一個用於IP網路的網路協定，使用UDP協定工作，屬於OSI模型的應用層，主要用於內部網路或網路服務供應商自動分配IP位址與相關的參數給用戶，讓DHCP伺服器能接受網路節點的請求，自動且動態地分配IP位址給網路上的電腦，藉此減少網路管理人員的工作負擔。
3
關於動態主機設定協定（Dynamic Host Configuration Protocol, DHCP）的優點，下列敘述哪一項錯誤？
A　可以讓一組IP位址有彈性地重複分配使用
B　可以讓IP位址動態地分配給DHCP用戶端
C　可以提昇網路線路的數據傳輸效率
D　方便網路IP位址的管理與維護
正確解答為：C　您的作答值為：D　回答錯誤了。TNO：U0050033
題目解析：
DHCP（Dynamic Host Configuration Protocol，動態主機設定協定）主要用於內部網路或網路服務供應商，自動且動態地分配IP位址給網路上的電腦，讓IP位址有彈性地重複分配使用，藉此減少網路管理人員的工作負擔，與方便網路IP位址的管理與維護，並無法提升網路的傳輸效率。

關於網際網路（Internet），下列敘述哪些正確？(複選題)
A　是一個廣域的網路
B　是一個區域性的網路
C　約在西元1920年開始發展
D　ARPANET可說是Internet的始祖
正確解答為：AD　您的作答值為：B　回答錯誤了。TNO：U0050034
題目解析：
網際網路是一個全球性的廣域網路，起源可以追溯到1960年代美國聯邦政府委託進行的一項研究，目的是建立具備容錯的電腦通訊技術，美國國防部高等研究計劃署（ARPA）建立的ARPANET成為網際網路發展的始祖，WWW則直到1989年才被提出。

關於TANet，下列敘述哪一項正確？
A　政府機關所使用的內部網路
B　提供給教育與研究單位使用的學術網路
C　民間公司所建立的商業網路
D　政府提供給所有人民自由使用的公用網路
正確解答為：B　您的作答值為：D　回答錯誤了。TNO：U0050036
題目解析：
TANet（Taiwan Academic Network）為臺灣學術網路，為中華民國教育部與臺灣各學術單位共同建立提供給教育與研究單位使用的學術網路。TANet的管理組織分為三個階層：「台灣學術網路管理委員會」、「區域網路中心推動小組」以及「縣市教育網路中心推動小組」。TANet網路架構分為國家骨幹網路、地區性網路、校園網路三個階層：主節點：TANet在中央研究院、臺北、新竹、臺中、臺南，全台五個點設定主節點，主要收攏各縣市所在之區域網路中心，提供各區域網路中心之間骨幹網路交換。區網中心：在全國各區域選出一個院校單位，成立區域網路中心，收攏鄰近大學大專技職院校及較近區域之高中職與國中小，協助該區域內相關學校及學術研究單位共享網際網路資源。縣市網中心：各縣市建立收攏各高中職、國中小及幼稚園等教育單位。

著名的全球資訊網（World Wide Web, WWW）所採用的主要協定為下列哪一項？
A　FTP（File Transfer Protocol）
B　HTTP（Hypertext Transfer Protocol）
C　SMTP（Simple Mail Transfer Protocol）
D　RTP（Real-time Transport Protocol）
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：U0050038
題目解析：
HTTP（HyperText Transfer Protocol）是全球資訊網（WWW）的資料通訊基礎。設計HTTP最初的目的是為了提供一種發布和接收HTML頁面的方法，透過HTTP或HTTPS協定請求的資源由URL（Uniform Resource Identifiers，統一資源識別碼）來標識。FTP（File Transfer Protocol）是用於電腦網路上在客戶端和伺服器之間進行檔案傳輸的應用層協定。SMTP（Simple Mail Transfer Protocol，簡單郵件傳輸協定）是在網際網路上傳輸電子郵件的標準。RTP（Real-time Transport Protocol，即時傳輸協定）是網路傳輸協定，主要用在串流媒體系統於網際網路上傳遞音訊和視訊。

智慧型手機可以讓我們講電話及連上網際網路，在使用時如要降低手機電磁波可能產生的生物效應，下列敘述哪些正確？(複選題)
A　不要讓耳朵或頭部太靠近手機
B　選擇使用電磁波能量比吸收率（Specific Absorption Rate, SAR）的值比較低的手機
C　調低手機的音量
D　選擇使用體積比較小的手機
正確解答為：AB　您的作答值為：AB　答對了。TNO：U0050039
題目解析：
在使用智慧型手機時，如要降低手機電磁波可能產生的生物效應，除了減少使用時間、不要讓耳朵或頭部太靠近手機、選擇收訊良好的地方通話之外，可以選擇SAR（Specific Absorption Rate，電磁波能量比吸收率）數值較低的手機，SAR是國際手機電磁波輻射量，代表生物體每單位公斤在單位時間內所吸收的功率，以W/kg為單位。SAR值越低，輻射被吸收的量越少，台灣NCC規範的手機SAR值最大不得超過2.0 W/kg。

關於網域名稱系統（Domain Name System, DNS），下列敘述哪些正確？(複選題)
A　DNS伺服器可以透過網域名稱的解析得到對應的MAC位址
B　每一台DNS伺服器都記錄了網際網路上所有主機名稱與IP位址之間的對應
C　假如世界上所有的DNS伺服器都當機，則網際網路的運作將完全癱瘓
D　DNS採用分散式以及階層式的網域名稱管理架構，DNS伺服器查不到的主機IP資訊可向其上層的DNS伺服器請求查詢
正確解答為：CD　您的作答值為：BD　回答錯誤了。TNO：U0050040
題目解析：
網域名稱系統（Domain Name System, DNS）是網際網路的一項服務，是將網域名稱和IP位址相互對映的一個分散式資料庫，使人們更方便地存取網際網路，讓使用者在瀏覽網站時，能夠輸入 www.google.com，而不用記住這台Server的IP是多少，因此假如世界上所有的DNS伺服器都當機，則網際網路的運作將完全癱瘓。DNS系統基本是採樹狀階層式架構，其中根域名伺服器（Root DNS Server）記錄所有TLD（Top Level Domain）並提供頂級域伺服器的對映，從Root DNS server開始，其他的DNS命名空間被委託給其他的DNS伺服器，如各個國家的頂級域伺服器，以及各組織類別的權威域名伺服器，這些伺服器提供DNS名稱空間中的特定部分，DNS伺服器查不到的主機IP資訊可向其上層的DNS伺服器請求查詢。

關於網域名稱「www.tku.edu.tw」，下列敘述哪些正確？(複選題)
A　www.tku.edu.tw中的「tw」代表頂級網域名稱
B　www.tku.edu.tw中的「www」代表連線使用的協定名稱
C　www.tku.edu.tw中的「edu」代表該網址屬於教育機構
D　www.tku.edu.tw中的「tku」代表網域所在地理區域的名稱
正確解答為：AC　您的作答值為：BC　回答錯誤了。TNO：U0050042
題目解析：
網域名稱的解讀以右到左為其階層，以「www.tku.edu.tw」為例：「.tw」是代表國家及地區的頂級網域名稱（Top-level domains，TLD），臺灣為tw，日本為jp，香港為hk，中國為cn，韓國為kr等。「.edu」是代表組織類型的通用頂級網域（General Top-level domains，gTLD），.edu表示教育機構，.gov表示政府及其屬下機構，.com表示商業機構，.mil表示軍事機構，.org表示非營利組織，.net表示網路服務供應商，.int表示國際組織等。「.tku」是代表該網址所屬機構，「www」則表示該機構內的主機名稱。
13
192.168.1.1屬於下列哪一項等級的IP位址？
A　Class A
B　Class B
C　Class C
D　Class D
正確解答為：C　您的作答值為：C　答對了。TNO：U0050043
題目解析：
IPv4位址可分為A、B、C、D、E五大類：A類IPv4位址網路標誌位為0，IP位址範圍 1.0.0.0~127.255.255.255；B類IPv4位址網路標誌位為10，IP位址範圍 128.0.0.0~191.255.255.255；C類IPv4位址網路標誌位為110，IP位址範圍 192.0.0.0~223.255.255.255；D類IPv4位址網路標誌位為1110，IP位址範圍 224.0.0.0~239.255.255.255；E類IPv4位址網路標誌位為11110，IP位址範圍 240.0.0.0~247.255.255.255。「192.168.1.1」（192轉為二進位為11000000），屬於Class C等級的IP位址（網路標誌位為110）。

關於IP位址，下列敘述哪些正確？(複選題)
A　現行的IPv4中每一組IP以四組用英文句點分開的數字表示
B　沒有IP位址的電腦無法連上網際網路

URL可用來表示網際網路中資源所在之處，當輸入「ftp://192.168.3.253:7」的URL資訊時，其中指定的連結埠號（Port）為下列哪一項？
A　192
B　3
C　253
D　7
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：U0050045
題目解析：
URL的標準格式為「[協定類型]://[伺服器位址]:[埠號]/[資源層級UNIX檔案路徑][檔名]?[查詢]#[片段ID]」，因此「ftp://192.168.3.253:7」中，協定類型為ftp、伺服器位址為192.168.3.253，埠號（Port）為7。
