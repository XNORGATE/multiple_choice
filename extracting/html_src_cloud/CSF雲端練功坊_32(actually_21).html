1
關於雲端運算的認知，下列敘述哪一項錯誤？
A　雲端運算是資訊科技數十年的發展演化而成的
B　雲端運算是高空作業的簡稱
C　雲端運算已普遍用在網路應用
D　雲端運算已成為電腦技術的主流
正確解答為：B　您的作答值為：C　回答錯誤了。TNO：U0060001
題目解析：
雲端運算（Cloud Computing）不是高空作業的簡稱，而是一種基於網際網路的運算方式，共享的軟硬體資源和資訊可以依據使用者的需求提供給各種終端裝置，雲端運算是資訊科技歷經數十年的發展，基於許多關鍵技術演化而成的，如網際網路、多核心處理器、網格運算、虛擬化等技術，已成為電腦技術的主流，更是普遍用在網路應用，如Gmail、Google Maps、Facebook、Twitter、Amazon EC2等應用。
2
關於雲端運算，下列敘述哪一項錯誤？
A　使用者需要了解雲端專業知識才能使用
B　Gmail跟Google Maps都是雲端運算的應用
C　又稱「雲計算」
D　智慧型手機也可以使用這個技術
正確解答為：A　您的作答值為：C　回答錯誤了。TNO：U0060002
題目解析：
雲端運算將運算資源當成公共服務提供給使用者使用，使用者不需了解雲端中基礎設施的細節，也不必具有相應的專業知識，亦無需直接進行控制，即可使用雲端服務商提供的資源和應用服務。
3
下列哪一項不是廣義的雲端運算應用？
A　捷運
B　餐廳
C　旅館
D　沒有上網的個人電腦
正確解答為：D　您的作答值為：C　回答錯誤了。TNO：U0060003
題目解析：
雲端運算是一種基於網際網路的運算方式，因此使用雲端運算最基本的要求就是設備需要能夠連上網際網路。
4
下列哪一項不是雲端運算的必要基礎？
A　網路
B　儲存
C　運算資源
D　終端設備
正確解答為：D　您的作答值為：C　回答錯誤了。TNO：U0060004
題目解析：
在某些雲端運算的應用情境中，是透過網路將多部伺服器組成龐大運算系統，並將龐大的運算內容自動分拆成無數個較小的運算內容，再由運算分析之後將處理結果回傳給使用者，它可分析DNA結構、基因圖譜定序、解析癌症細胞等極為複雜的運算，在這些特定運算應用中，終端設備就不是必要的基礎。
5
下列哪一項不是雲端運算的基礎技術？
A　網格運算（Grid Computing）
B　公用運算（Utility Computing）
C　集中式運算（Centralized Computing）
D　分散式運算（Distributed Computing）
正確解答為：C　您的作答值為：C　答對了。TNO：U0060005
題目解析：
雲端運算的基礎技術有叢集運算（Cluster Computing）、網格運算（Grid Computing）、公用運算（Utility Computing）、分散式運算（Distributed Computing）等技術。叢集運算是將一組鬆散或緊密耦合的電腦連接在一起，並將每個節點設定為執行相同的任務，通常情況下，所有節點使用相同的硬體和相同的作業系統。網格運算是由一群鬆散耦合的電腦組成的，利用大量異構電腦的未用資源，組成一個超級虛擬電腦，常用來解決大規模的計算問題，網格計算的焦點放在支援跨管理域計算的能力。公用運算是一種服務供應模型，服務供應商根據客戶需要，向客戶提供運算資源和基礎架構管理，並按特定用途而非統一費率向客戶收費。分散式運算主要研究分散式系統如何進行運算，組件之間彼此進行互動以實現共同目標，分散式運算把需要進行大量計算的資料分割成小塊，分派給多台電腦分別計算，再將運算結果上傳，將結果統一合併得出資料結論。

雲端運算首次出現在商業應用，是下列哪一項產品？
A　VMware vSphere
B　Microsoft Azure
C　Oracle VirtualBox
D　Amazon EC2
正確解答為：D　您的作答值為：C　回答錯誤了。TNO：U0060006
題目解析：
VMware vSphere最初版本為2009年4月發布，Microsoft Azure於2010年2月推出，Oracle VirtualBox最初版本於2007年1月發布，Amazon EC2於2006年8月推出，因此選項中最早出現的商業應用為Amazon EC2，除此之外，VMware vSphere與Oracle VirtualBox皆屬於虛擬機器軟體，並非雲端運算的商業應用。
7
下列哪一項不是雲端運算的服務概念？
A　電力公司根據使用者用電的度數收取電費
B　自來水公司根據用水的公升數收取水費
C　電信公司包月制的手機話費
D　自己在陽台上種植水果食用
正確解答為：D　您的作答值為：C　回答錯誤了。TNO：U0060007
題目解析：
雲端運算將運算資源當成公共服務提供給使用者使用，使用者不需了解雲端中基礎設施的細節，也不必具有相應的專業知識，亦無需直接進行控制，即可使用雲端服務商提供的資源和應用服務。選項(D)「自己在陽台上種植水果食用」，因為需要了解種植水果的各項細節與知識，因此不符合雲端運算的服務概念。
8
下列哪一項不是雲端運算的重要概念？
A　已量化後的多個運算資源
B　透過網路為媒介傳送至特定介面上的使用者
C　以服務的方式，依需求的不同付費
D　軟體與資料都儲存在使用者終端裝置上
正確解答為：D　您的作答值為：C　回答錯誤了。TNO：U0060008
題目解析：
雲端運算可以依據使用者的需求，將共享的軟硬體資源和資訊提供給使用者的終端裝置，軟體、介面、程式邏輯、資料大部分都分散存放在雲端許多的伺服器中。

下列哪一項為雲端運算用「雲」這個字的目的？
A　表示此技術深不可測
B　表示此技術非常高級
C　表示此技術使用者不需要了解內容，只要會使用即可
D　表示此技術非常不穩定
正確解答為：C　您的作答值為：C　答對了。TNO：U0060009
題目解析：
雲端運算的使用者不需了解雲端中的技術細節，只需要知道自己的需求以及如何使用雲端服務商提供的資源和應用服務。
10
下列哪一項為雲端運算在電腦技術上，最大的目的？
A　將資料集中方便管理
B　強化某一台主機的運算能力
C　打破電腦技術中每一個層次的相依性，用網路邏輯上互連
D　讓介面、資料及邏輯都包裝在同一個應用中
正確解答為：C　您的作答值為：C　答對了。TNO：U0060010
題目解析：
雲端運算在電腦技術上最大的目的是打破電腦技術中每一個層次的相依性，將各層次的電腦技術中引入抽象的層次，並用網路邏輯上互連，使各層的技術得以各自改良發展，並在單一層次技術改變時，不至影響到其他層的技術。
11
將運算資源以服務的方式發佈，下列哪一項不是雲端運算的定義？
A　基礎設施即服務（Infrastructure as a Service, IaaS）
B　平台即服務（Platform as a Service, PaaS）
C　軟體即服務（Software as a Service, SaaS）
D　維護即服務（Maintenance as a Service, MaaS）
正確解答為：D　您的作答值為：C　回答錯誤了。TNO：U0060011
題目解析：
美國國家標準與技術研究院（NIST）提出的雲端運算定義中，雲端運算有三種服務模式：SaaS（Software as a Service）、PaaS（Platform as a Service）、IaaS（Infrastructure as a Service）。
12
下列哪一項不是軟體即服務（Software as a Service, SaaS）的優點？
A　不再有軟體版本的困擾
B　使用者可依需求購買軟體的功能而非全部買單
C　不需要部署即可使用
D　可利用USB安裝，不再需要光碟
正確解答為：D　您的作答值為：C　回答錯誤了。TNO：U0060012
題目解析：
SaaS（Software as a Service）向使用者提供應用程式或軟體的雲端服務，在這種服務模式中，使用者僅需透過網際網路，不需經過傳統的安裝步驟即可使用服務，如Gmail、Google Calendar、Facebook等，使用者不再有軟體版本的困擾，且可依需求購買軟體的功能而非全部買單，只需要網際網路與瀏覽器而不需要部署即可使用。
13
雲端運算中，資料不會儲存在下列哪一項設備中？
A　網路硬碟
B　雲端伺服器
C　本機設備
D　伺服器所在區域網路之儲存設備
正確解答為：C　您的作答值為：C　答對了。TNO：U0060013
題目解析：
雲端運算所有資料皆儲存在服務提供者的伺服器之上，不會存在本機設備裡。
14
下列哪一項應用不是平台即服務（Platform as a Service, PaaS）的範圍？
A　資料庫（Database）伺服器
B　網頁（Web）伺服器
C　實體（Physical）伺服器
D　應用程式（Application）伺服器
正確解答為：C　您的作答值為：C　答對了。TNO：U0060014
題目解析：
PaaS（Platform as a Service）通常是提供應用程式基礎架構，如Google App Engine，消費者掌控運作應用程式的環境，但並不掌控作業系統、硬體或運作的網路基礎架構。PaaS的應用範圍有資料庫伺服器、網頁伺服器、應用程式伺服器等，實體伺服器屬於IaaS的應用範圍。
15
下列哪一個廠商因為提供IaaS（Infrastructure as a Service）成為該領域的霸主？
A　Oracle
B　Microsoft
C　Google
D　Amazon
正確解答為：D　您的作答值為：C　回答錯誤了。TNO：U0060015
題目解析：
Gartner 2019年的統計資料，Amazon Web Services（AWS）雲端運算服務去年營收154.95億美元，佔全球雲端基礎架構及服務（IaaS）市佔率47.8%，排名第一。Microsoft Azure營收為50.38億美元，市佔率15.5%排名第二。阿里雲營收24.99億美元，7.7%市佔率排名第三。Google Cloud營收13.14億美元，4%市佔率排名第四。IBM營收5.77億美元，1.8%市佔率排名第五。
