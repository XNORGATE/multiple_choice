A　會話群組
B　篩選器

您可以透過下列哪些存取協定，將郵件從Gmail伺服器下載至您的電腦，讓您在離線狀態下也能使用Microsoft Outlook或Thunderbird等程式來存取郵件？(複選題)
A　POP
B　SMTP
C　IMAP
D　HTTP
正確解答為：AC　您的作答值為：AB　回答錯誤了。TNO：U0020018
題目解析：
POP（Post Office Protocol）和IMAP（Internet Message Access Protocol）是存取電子郵件的兩種方法，屬於「Pull」的協定。POP的運作方式是與您的電子郵件服務連線，並下載電子郵件服務中的所有新郵件，一旦郵件下載後，電子郵件服務就會將它們刪除。IMAP是直接連接到郵件伺服器上，直接對郵件進行操作，並將郵件保留在伺服器上​​。

日積月累之下，您的Gmail收件匣已累積有多年的電子郵件，透過下列哪一項功能可讓您將郵件從「收件匣」中消失，但在需要重新檢視時，仍可搜尋得到郵件？
A　刪除
B　封存
C　標籤
D　篩選器
正確解答為：B　您的作答值為：B　答對了。TNO：U0020026
題目解析：
如果想清理收件匣中的信件，但並不想真的把電子郵件刪掉，只要「封存」或「略過」這些郵件即可，這些電子郵件會移到名為「所有郵件」的標籤中。如果選擇「封存郵件」，日後有人回覆時，該郵件會再次出現在您的收件匣中。如果選擇「略過郵件」，日後相關的回覆郵件都不會出現在您的收件匣中，若想再找出該郵件，需在會話群組中進行搜尋。

小昌的Gmail郵件地址為chinsanchen@gmail.com，可採用的合法郵件地址別名有下列哪些？(複選題)
A　chinsanchen+school@gmail.com
B　chinsan.chen@gmail.com
C　chinsan+chen@gmail.com
D　chinsanchen.school@gmail.com
正確解答為：AB　您的作答值為：BD　回答錯誤了。TNO：U0020028
題目解析：
原則上email帳號只能有一個，但是使用地址別名功能，就能使用這一個email帳號，創造出許多的email地址，在Gmail中可以採用的兩種主要的做法為：「<使用者名稱>+<任何別名>@gmail.com」或「<使用者名稱中加上”.”>@gmail.com」。第二種做法主要是因為Gmail會自動忽略使用者名稱中的「.」，所以可以用這種方法創造不同的郵件地址別名。
