把圖片檔案拖曳到「Google圖片」的「以圖搜尋方框」內時，會啟動下列哪一項動作？
A　僅上傳圖片
B　僅顯示圖片
C　僅下載圖片
D　搜尋網路上和該圖片類似或相同的其它圖片
正確解答為：D　您的作答值為：D　答對了。TNO：U0010046
題目解析：
在Google圖片網頁下（https://www.google.com.tw/imghp），將圖片拖曳至網頁中，會啟動「以圖搜尋」功能，即搜尋網路上和該圖片類似或相同的其它圖片。

在Mozilla Firefox中的擴充功能等同於下列哪一項？
A　EXE可執行檔
B　特殊格式的執行檔
C　Java執行檔
D　JavaScript
正確解答為：D　您的作答值為：D　答對了。TNO：U0010047
題目解析：
Mozilla Firefox的擴充功能，可以用下列技術開發，JavaScript（Mozilla瀏覽器的主要語言）、CSS、XUL（定義使用者介面和與用戶互動的元素）、DOM（即時更改XUL代碼和編輯被載入的HTML內容）、XPCOM。

下列哪一項瀏覽器以載入速度較快而聞名？
A　Microsoft IE
B　Google Chrome
C　Mozilla Firefox
D　Apple Safari
正確解答為：B　您的作答值為：C　回答錯誤了。TNO：U0010048
題目解析：
Google Chrome起始於2008年，已在2020年1月達到將近70%的全球桌面瀏覽器占有率，為當前使用率最高的網頁瀏覽器，Google Chrome最出名的功能，是載入網頁的速度（解析JavaScript的速度）特別快，且介面簡潔，2009年加入的擴充功能創造了瀏覽器的多樣性，因此深受使用者的喜愛。

下列哪一項瀏覽器在不使用附加元件或擴充功能之下，可以直接檢視頁面中各種原始資料（如HTML或JavaScript）？
A　Microsoft IE
B　Google Chrome
C　Mozilla Firefox
D　所有主流瀏覽器都可以
正確解答為：D　您的作答值為：D　答對了。TNO：U0010049
題目解析：
目前主流瀏覽器都可以檢視瀏覽頁面中的原始資料，在Windows環境下，可以使用[Ctrl]+[U]鍵、[F12]鍵、右鍵選擇「檢視網頁原始碼」皆能夠檢視網頁的HTML程式碼。
5
下列哪些不是網頁使用CSS的主要目的？(複選題)
A　套用樣式
B　相容於各種瀏覽裝置
C　可快速變更頁面外觀
D　加快網頁載入速度
正確解答為：BD　您的作答值為：D　回答錯誤了。TNO：U0010050
題目解析：
CSS(Cascading Style Sheets)，是一種用來為結構化文件（如HTML網頁）添加頁面樣式與設計，CSS必須與HTML或XML一同運作，其中HTML負責確定網頁中有哪些內容，CSS確定以何種外觀（大小、粗細、顏色、字型、間距、對齊和位置）展現這些元素，其主要目的在於方便的套用網頁樣式，以及快速變更頁面的外觀。
