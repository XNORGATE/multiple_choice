創用CC授權條款中，標誌代表的意思，是下列何者？
A　姓名標示（Attribution, BY）
B　相同方式分享（Share Alike, SA）
C　非商業性（NonCommercial, NC）
D　禁止改作（No Derivatives, ND）
正確解答為：D　您的作答值為：B　回答錯誤了。TNO：U0070046
題目解析：
表示禁止改作（No Derivative Works，ND），使用時不得改變、轉變或更改本作品。姓名標示（BY）標誌為。相同方式分享（SA）標誌為。非商業性使用（NC）標誌為。
2
偷看不屬於自己的電子郵件，其行為是否已觸犯法律，下列敘述哪些錯誤？(複選題)
A　是，已觸犯著作權法
B　否，因為並沒有將電子郵件轉寄散發
C　否，因為電子郵件只能算是電磁記錄
D　是，已觸刑法的妨害書信罪
正確解答為：ABC　您的作答值為：ABC　答對了。TNO：U0070047
題目解析：
刑法第315條妨害書信罪：「無故開拆或隱匿他人之封緘信函、文書或圖畫者，處拘役或九千元以下罰金。無故以開拆以外之方法，窺視其內容者，亦同。」對於一般文書之定義於刑法第220條：「在紙上或物品上之文字、符號、圖畫、照像，依習慣或特約，足以為表示其用意之證明者，關於本章及本章以外各罪，以文書論。錄音、錄影或電磁紀錄，藉機器或電腦之處理所顯示之聲音、影像或符號，足以為表示其用意之證明者，亦同。」因此偷看不屬於自己的電子郵件已觸刑法第315條的妨害書信罪。
3
在網路上發表文章時，下列哪些行為是不適當的？(複選題)
A　使用大量的注音符號，讓文章更生動
B　理性討論，避免謾罵
C　轉載文章註明作者出處
D　公布軟體註冊破解碼供大家一同分享
正確解答為：AD　您的作答值為：AD　答對了。TNO：U0070048
題目解析：
文章內容使用注音文或火星文除了會造成閱讀困難與文章內容難以理解外，甚至可能產生誤解，例如「ㄍㄉㄆㄊㄌㄔ」可解讀成「古道葡萄綠茶」或「軌道砲台列車」等多種不同字詞。而散布軟體的破解碼、註冊碼、破解程式等行為已觸犯著作權法第80-2條：「著作權人所採取禁止或限制他人擅自進入著作之防盜拷措施，未經合法授權不得予以破解、破壞或以其他方法規避之。破解、破壞或規避防盜拷措施之設備、器材、零件、技術或資訊，未經合法授權不得製造、輸入、提供公眾使用或為公眾提供服務。」
4
下列哪些情況是違法的？(複選題)
A　使用信用卡號進行網路購物
B　幫討厭的朋友徵網路援交
C　把歌手的MP3下載回自己電腦再燒錄分送好友
D　把版權軟體放上網路與朋友分享
正確解答為：BCD　您的作答值為：BCD　答對了。TNO：U0070049
題目解析：
冒用他人名義註冊帳戶或張貼文章，已觸犯刑法第210條偽造文書罪：「偽造、變造私文書，足以生損害於公眾或他人者，處五年以下有期徒刑。」把歌手的MP3下載回自己電腦並將其備份之MP3重製物給予他人或從事商業行為，已涉及侵害著作財產權人之「重製權」。任意將受著作權法保護的電腦程式放上網路供他人下載，已涉及侵害著作財產權人之「重製權」及「公開傳輸權」。
5
下列哪些情況是違法的？(複選題)
A　設立色情網站
B　在網路上販賣從日本帶回來的藥品
C　散佈自己拍攝的照片
D　將木馬程式植入網站，竊取別人電腦中的資料
正確解答為：ABD　您的作答值為：BD　回答錯誤了。TNO：U0070050
題目解析：
設立色情網站，利用網際網路傳送猥褻圖文，已觸犯刑法第235條：「散布、播送或販賣猥褻之文字、圖畫、聲音、影像或其他物品，或公然陳列，或以他法供人觀覽、聽聞者，處二年以下有期徒刑、拘役或科或併科九萬元以下罰金。」。依規定非藥商不得販售藥物，若想在網路販售藥物，除了須有實體通路營業處所外，還須向當地衛生局申請，因此一般民眾不可在網路上販賣藥物，否則就會觸犯藥事法第27條、藥事法第65條、藥事法第69條等法規。將木馬程式植入網站，竊取別人電腦中的資料已觸犯刑法第358條「入侵電腦罪」、刑法第359條「破壞電磁記錄罪」、刑法第360條「干擾電腦罪」、刑法第362條「製作電腦犯罪程式罪」等法規。
