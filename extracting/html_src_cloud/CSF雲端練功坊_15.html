雲端運算的技術主要包含下列哪些層面？(複選題)
A　硬體層
B　雲端布建和管理
C　虛擬化環境
D　軟體層
正確解答為：ABCD　您的作答值為：ABCD　答對了。TNO：U0040031
題目解析：
雲端運算的技術架構可以大致分為硬體層（Physical Hardware）、虛擬化環境（Virtualization Environment）、雲端布建和管理（Cloud Provisioning and Management）、軟體層（Software)。
2
雲端運算目前的特色不包含下列哪一項？
A　高可擴充性
B　高安全性
C　隨需服務
D　高延展性
正確解答為：B　您的作答值為：D　回答錯誤了。TNO：U0040032
題目解析：
雲端運算有隨需服務、超大規模、高通用性、高可擴充性、快速佈署靈活度、虛擬化、高可靠性、共享資源池、可量測與計價等特性。但安全性則是雲端運算一直以來持續面對的威脅，例如不安全的介面、惡意的雲端服務供應商內部員工、共享資源池的資源安全性、資料外洩、帳號安全等安全性議題。
3
雲端運算具備下列哪些特徵？(複選題)
A　隨需所選自助式服務
B　共用資源池
C　服務可測量
D　存取方式多樣化
正確解答為：ABCD　您的作答值為：ABCD　答對了。TNO：U0040033
題目解析：
美國國家標準與技術研究院（NIST）提出的雲端運算定義中，雲端運算有五種特徵：自助式隨需服務(On-demand self-service)、廣泛網路接取(Broad network access)、共享資源池(Resource pooling)、快速的彈性(Rapid elasticity)、可量測的服務(Measured service)。
4
雲端運算的應用通常被區分為下列哪些？(複選題)
A　CaaS（Content as a Service）
B　SaaS（Software as a Service）
C　PaaS（Platform as a Service）
D　IaaS（Infrastructure as a Service）
正確解答為：BCD　您的作答值為：CD　回答錯誤了。TNO：U0040034
題目解析：
美國國家標準與技術研究院（NIST）提出的雲端運算定義中，雲端運算有三種服務模式：SaaS（Software as a Service）、PaaS（Platform as a Service）、IaaS（Infrastructure as a Service）。
5
下列哪一項比較適合導入雲端運算服務？
A　資料庫
B　交易型業務
C　促進協作的應用
D　ERP
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：U0040035
題目解析：
已往不同使用者要進行協作時，只能各自於電腦上編輯後，再互相交換資料，進行差異性比對，若導入雲端運算服務，可直接資料放上雲端，開啟資料權限後，任何允許編輯的使用者，皆可同時對該文件進行協作，例如Google協作平台、Google Docs、Github都是協作應用的一種。
6
下列哪一項為雲端軟體即服務（Software as a Service, SaaS）？
A　Microsoft Azure
B　Google App Engine
C　iCloud
D　Amazon EC2
正確解答為：C　您的作答值為：C　答對了。TNO：U0040036
題目解析：
iCloud是Apple所提供的線上同步儲存服務和雲端計算服務，屬於應用層級的SaaS。Microsoft Azure提供IaaS與PaaS服務，Google App Engine提供PaaS服務，Amazon EC2提供IaaS服務。

雲端軟體即服務（Software as a Service, SaaS）的計費方式，可能有下列哪幾種？(複選題)
A　以記憶體速度計費
B　以使用者或使用量的方式計費
C　以計算能力計費
D　以儲存空間容量計費
正確解答為：BCD　您的作答值為：BCD　答對了。TNO：U0040037
題目解析：
常見的SaaS計費方式有：按使用時間、按使用者人數、按使用量、按服務內容、按儲存空間容量等。
8
下列哪些為雲端平台即服務（Platform as a Service, PaaS）？(複選題)
A　Facebook
B　Google App Engine
C　Amazon EC2
D　Microsoft Azure
正確解答為：BCD　您的作答值為：AD　回答錯誤了。TNO：U0040038
題目解析：
Google App Engine提供PaaS服務，Amazon EC2提供IaaS與PaaS服務，Microsoft Azure提供IaaS與PaaS服務。Facebook則屬於SaaS服務。
9
雲端平台即服務（Platform as a Service, PaaS）的計費方式有許多種可能，若以Google App Engine的標準環境為例，計費方式可能有下列哪幾種？(複選題)
A　以服務的內容計費
B　以運算能力、儲存空間的多寡計費
C　以使用電量計費
D　以執行時間計費
正確解答為：BD　您的作答值為：ABD　回答錯誤了。TNO：U0040039
題目解析：
PaaS計費方式通常以運算能力、記憶體、儲存空間等實體資源的多寡作為其費率，並依其執行時間作為計費方式，以Google App Engine為例，每個執行個體依其運算能力、記憶體、儲存空間的多寡，每小時的費用約在0.05~0.4美金。

下列哪些為雲端基礎設施即服務（Infrastructure as a Service, IaaS）？(複選題)
A　Google App Engine
B　Amazon EC2
C　HiCloud
D　FlexiScale

Amazon EC2、HiCloud、FlexiScale皆提供IaaS服務。Google App Engine則提供PaaS服務。
11
雲端基礎設施即服務（Infrastructure as a Service, IaaS）的計費方式是採取下列哪些？(複選題)
A　以服務的內容計費
B　以執行時間計費
C　以運算能力、儲存空間的多寡計費
D　以使用人數計費
正確解答為：BC　您的作答值為：ABCD　回答錯誤了。TNO：U0040041
題目解析：
IaaS計費方式通常以運算能力、記憶體、儲存空間等實體資源的多寡作為其費率，並依其使用時間作為計費方式，以Amazon EC2專用主機為例，a1規格每個月約207美金，m5規格每個月約2,364美金。HiCloud提供的服務，b.1c1g規格每小時0.5台幣，b.64c256g規格每小時125.66台幣。
12
雲端運算在技術層面，下列敘述哪些正確？A.晶片和硬體技術的飛速發展、B.虛擬化技術成熟、C.雲端運算可以輕鬆實現不同設備間的資料共享、D.雲端運算使用特定的終端裝置，以保證安全、E.雲端運算不限制使用地點
A　ABCD
B　ABDE
C　ABCE
D　BCDE
正確解答為：C　您的作答值為：D　回答錯誤了。TNO：U0040042
題目解析：
NIST定義的雲端運算五項特徵其中一項為Broad Network Access（廣泛網路接取），意指服務是經由網路提供，因此能讓不同平台的客戶端都可以使用，因此D.為錯誤，雲端運算不會限定存取的終端裝置，且雲端運算的安全是需要結合多方協力才能達成，包含了網路裝置安全、網路資訊安全、網路軟體安全等議題，且最重要的是使用者使用網路時的習慣。

雲端運算的隱憂包含下列哪些選項？A.資訊安全、B.服務轉移彈性、C.隱私的保全、D.節能減碳、E.系統相容度
A　ABCD
B　BCDE
C　ABCE
D　ABDE
正確解答為：C　您的作答值為：C　答對了。TNO：U0040043
題目解析：
雲端運算的隱憂有：資訊安全、個人隱私、服務轉移彈性、系統相容度、智慧財產權與軟體授權問題、資料存取的穩定性等。

SLA（Service Level Agreement）是意旨下列哪一項協定？
A　軟體層級協定
B　服務連結協定
C　軟體連結協定
D　服務層級協定
正確解答為：D　您的作答值為：D　答對了。TNO：U0040044
題目解析：
SLA（Service-Level Agreement，服務級別協定）是服務提供商與客戶之間定義的服務指標正式承諾，SLA通常定義有平均故障間隔時間（MTBF）、平均故障時間（MTTF）或平均修復時間（MTTR）、哪一方負責報告錯誤與支付費用、吞吐量等可量化細節。
