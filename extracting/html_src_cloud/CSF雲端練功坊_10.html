視訊直播是近來興起的社交應用，其常見應用為下列哪些？(複選題)
A　線上即時教學課程
B　分享才藝表演
C　轉播各類精彩賽事
D　分享遊戲過程
正確解答為：ABCD　您的作答值為：ABCD　答對了。TNO：U0030016
題目解析：
隨著寬頻網路日趨普及，電腦運算能力增加，原本只有電子媒體和企業才可進行的直播，擴展到一般網際網路的使用者、家庭皆有辦法進行網路視訊直播。常見的視訊直播平台有Twitch、YouTube、Facebook、鬥魚TV等，許多藝人與網路素人開始利用直播平台直接與觀眾產生互動，而直播的內容除了一開始的各類賽事轉播與線上即時教學課程外，目前也常用於轉播遊戲過程、展現各種才藝或是其他表演，並與觀眾即時性互動。

在Web2.0網路社群，主要的營利方式為下列哪一項？
A　廣告
B　會員費
C　創作費
D　管理費
正確解答為：A　您的作答值為：A　答對了。TNO：U0030017
題目解析：
社群網路服務的商業模式，大致上可以區分為「廣告收入」、「向使用者收費」、「第三方網站誘導」、「遊戲內建購買」。其中以廣告收入為網路社群主要的營利方式，主要方式為透過用戶的登入習慣、發言內容、發言頻率，加上大數據資料探勘，針對不同用戶投放不同的廣告內容，增加廣告投放效益。

網路社群與即時通訊軟體的主要差異為下列哪些？(複選題)
A　即時通訊軟體較為私密，不知道雙方帳戶較難產生聯繫
B　網路社群可以作為電話之外即時的聯絡工具
C　網路社群較為公開，只要搜尋到對方資訊，都可以加為朋友
D　社群提供粉絲們關注名人的管道
正確解答為：ACD　您的作答值為：ACD　答對了。TNO：U0030018
題目解析：
「社群網路」主要目的為資訊的交流與分享，可以做為社交拓展的一種方式，較為公開，只要透過關鍵字或朋友間的關係就能搜尋到對方，也提供了使用者與其粉絲方便交流的管道，但由於社群網路並非屬於即時通訊模式，因此不適合做為電話之外的即時聯絡工具。「即時通訊軟體」則是一種透過網路進行實時通訊的系統，允許多人透過網路進行即時的文字訊息、檔案、語音或視訊交流，較為私密，若不知道對方帳戶，較難聯繫到對方。

視訊會議是一般即時通訊軟體進階應用，與傳統的即時通訊的差別為下列哪些？(複選題)
A　廣播式的資料傳輸
B　著重於多方通話
C　更嚴格的品質考驗
D　即時的簡報影像傳輸
正確解答為：ABCD　您的作答值為：ABCD　答對了。TNO：U0030022
題目解析：
視訊會議為了達到與現實會議相仿的目的，會著重於多方通話，且對於影像與語音的時延及通訊品質會更加要求，而視訊會議通常都會是多個位於不同地點的使用者同時參與，為了降低時延差異，會採用廣播式的資料傳輸，亦即一方說話，會同時將語音資料發送給會議的所有參與者，以降低不同參與者收到資料的時間差。知名的視訊會議軟體有Zoom、LINE、Hangouts、Teams等。

網路社群對人類溝通結構造成很大的衝擊，主要的負面影響為下列哪些？(複選題)
A　更容易感到焦慮和沮喪
B　人與人之間的交情越來越粗淺
C　人們花費越來越多的時間整理來自網路的垃圾資訊
D　網路霸凌更容易發生
正確解答為：ABCD　您的作答值為：ACD　回答錯誤了。TNO：U0030023
題目解析：
據科技網站TechCrunch報導，美國Michigan State University研究人員發表在Journal of Behavioral Addictions中的論文指出，對社群網站沉癮的網路使用者，其行為特徵與吸食古柯鹼或海洛因的人們相似。且根據英國皇家公共衛生學會（RSPH）的調查，部分的社群網站有著許多負面影響，例如人與人之間的交情越來越粗淺與表面；使用社群媒體的人中，約17%的使用者曾感到焦慮和沮喪，睡眠品質下降，導致疲勞影響日常生活，產生自卑感、憂慮或壓力；70%的使用者表示曾遭受到網路霸凌；再來最嚴重的是社群網路中充斥著各類的垃圾資訊、置入性行銷及為了賺取點擊率而大量生產的八卦文章、假消息，這些大量的垃圾資訊，反而使人們要花更多時間來整理與查證，或可能被不實或片段的消息誤導。
