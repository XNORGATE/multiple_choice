B　直接寄信給擁有者申請共用
C　Google文件預設即為公開狀態，不須另外設定
D　取得共用連結，知道連結的人就可以共用此文件
正確解答為：AD　您的作答值為：BD　回答錯誤了。TNO：U0020040

A　透過軟體市集（Google Play、App Store）下載安裝
B　透過手機廠商提供的同步管理軟體（iTunes、HTC Sync、Samsung Kies）由電腦安裝至手機
C　Android手機可藉由AppsInstaller或ES文件管理器，安裝記憶卡內的APK檔
D　透過相機鏡頭掃描二維條碼下載安裝程式
正確解答為：ABCD　您的作答值為：AC　回答錯誤了。TNO：U0020044
題目解析：
上述四種方法皆可為智慧型手機安裝應用程式，但最安全的做法還是從官方提供的APP市集（iOS的App Store以及Android的Google Play）下載安裝，因為官方提供的APP市集基本都會針對上架的APP進行審核與安全檢查。
