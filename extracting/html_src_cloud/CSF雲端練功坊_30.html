勒索軟體可使用下列哪些方式來防範？(複選題)
A　定期資料備份
B　使用隱私模式瀏覽網頁
C　安裝防毒軟體
D　安裝防火牆

題目解析：

關於密碼設定方式，下列敘述哪些正確？(複選題)
A　避免使用帳號、生日及名字等相關資訊來建立密碼
B　使用帳號倒敘或生日來建立密碼，容易記憶，亦可避免被猜到
C　結合多個字詞來建立超過15個字的長密碼
D　避免使用現成的單一字詞來建立密碼
正確解答為：ACD　您的作答值為：ACD　答對了。TNO：U0080034
題目解析：
依據國家標準技術研究所（NIST）於2017年6月發布對於密碼的建議，與其使用一些複雜、看似亂碼的短密碼，選擇幾組自己熟悉的單字或數字，再湊成一個長密碼，除了讓自己容易記憶外，更能有效抵擋駭客攻擊，暴力破解會需要更久的時間。另外設定密碼時，不要用和帳號名稱相同或倒敘的密碼、不要用身份證字號、不要用生日、不要用在字典中查得到的單一字詞、不要用序列數等原則作為密碼。

關於防火牆的應用，下列敘述哪一項正確？
A　使用過濾性防火牆（Filter）將內部與外部網路完全區隔開來，比使用代理性防火牆（Proxy）更加安全
B　使用過濾性防火牆（Filter）能夠提高網路的存取速度，因為它將資料儲存在快取區（Cache）裡，在需要相同資料的情況下，無需從外部網路重複傳遞
C　使用防火牆管理封包是否放行，同時也將通過的封包送至指定的目的地位址
D　防火牆依照封包的來源、目的地位址、協定及通訊埠口（Port）等因素進行判斷並透過一系列的規則來決定封包的處理方式
正確解答為：D　您的作答值為：C　回答錯誤了。TNO：U0080035
題目解析：
從安全角度來說，代理性防火牆（Proxy）因為將內部與外部網路完全隔離，要建立連線只能透過Proxy幫忙連線代理，因此比起過濾性防火牆（Filter）更加安全。將資料儲存在快取區（Cache）藉此提高網路存取速度，屬於代理性防火牆（Proxy）的特色。防火牆只負責管理封包是否放行，封包的路由則是由路由（Routing）負責。

下列哪些為非軍事區（Demilitarized Zone, DMZ）裡，外部網路與內部網路間的訪問策略？(複選題)
A　DMZ可以訪問內部網路
B　內部網路可以訪問DMZ的主機
C　外部網路可以訪問DMZ的主機
D　外部網路不可以訪問內部網路
正確解答為：BCD　您的作答值為：AD　回答錯誤了。TNO：U0080036
題目解析：
DMZ（Demilitarized Zone）是一種在外部網路和內部網路外，建立一個對外部網路的物理或邏輯子網路，將部分用於提供對外服務的伺服器主機劃分到DMZ子網路內，在DMZ內的主機能與外部網路通訊，而DMZ內的主機與內部網路的通訊會被限制，使DMZ內的主機能被內部網路和外部網路訪問，但內部網路能避免被外部網路訪問。

下列哪些為使用防火牆的缺點？(複選題)
A　可能造成網路交通的瓶頸
B　無法針對服務類型（如WWW或FTP）進行過濾
C　無法防範電腦蠕蟲或木馬程式的快速蔓延
D　無法排除來自內部的攻擊
正確解答為：AD　您的作答值為：ACD　回答錯誤了。TNO：U0080037
題目解析：
防火牆通過將網路劃分成不同的區域，制定出不同區域之間的存取控制策略來控制不同信任程度區域間傳送的資料，除了能夠過濾不同服務類型的封包來監控網路傳輸外，還能夠更進一步檢查封包資料流的內容與行為，防範電腦蠕蟲或木馬程式的快速蔓延。由於所有網際網路的封包都會經過防火牆的過濾，將造成網路速度的瓶頸，且防火牆無法防範來自內部的攻擊，另外防火牆也無法防範繞過防火牆、新的與未知的、利用標準網絡協議中的缺陷、利用伺服器或作業系統的漏洞等攻擊。

常見的無線網路犯罪及攻擊方法為下列哪些？(複選題)
A　利用不需經過認證的無線網路進行違法行為，造成架設者背黑鍋
B　偽裝需要認證之無線網路識別碼（Service Set Identifier, SSID）騙取被害者的帳號密碼
C　對通訊內容進行監控，利用竊聽內容來獲取帳號及密碼
D　利用阻斷服務攻擊（Denial of Service, DoS）造成使用者與管理者無法取得系統資源及服務
正確解答為：ABCD　您的作答值為：ABCD　答對了。TNO：U0080038
題目解析：
常見的無線網路犯罪及攻擊方法有：違法使用無線網路：使用不需經過認證的無線網路，或完全沒有更改出廠設定無線網路存取點，就很容易被用來作違法的事情，造成架設者背黑鍋。竊聽：入侵者能夠針對無線網路通訊內容進行竊聽，以獲取被害人的個人資料，如帳號、密碼等資料。分析：針對無線網路通訊的流量、內容、以及行為等資訊進行分析，以獲得目標網路資料。偽裝：攻擊者藉由偽裝為需要認證之無線網路識別碼（SSID），騙取被害人的登入帳號與密碼。重送：攻擊者藉由將收集到的合法使用者認證資訊，重新發送給認證伺服器做驗證，取得合法權限。訊息竄改：攻擊者將收到的訊息做修改之後，再傳送出去，藉此改變或擾亂原本正常的通訊。阻斷服務攻擊：攻擊者透過各種可能的方法，如ICMP flooding或UDP flooding等方式，讓使用者與管理者無法取得系統資源及服務，讓被害人無法使用無線網路。中間人攻擊：攻擊者扮演中間人的角色，讓兩端的通訊經由他傳遞，並在通訊兩端不知情的情況下讀取、插入、更改傳遞的訊息。

下列哪些設定方式可增加無線基地台的安全性？(複選題)
A　不允許無線網路基地台的名稱或無線網路識別碼（Service Set Identifier, SSID）被廣播
B　使用WEP（Wired Equivalent Privacy）而不使用WPA（Wi-Fi Protected Access）加密
C　關閉DHCP（Dynamic Host Configuration Protocol）功能
D　建立MAC位址清單將已知的裝置列入，以進行過濾
正確解答為：ACD　您的作答值為：ABD　回答錯誤了。TNO：U0080039
題目解析：
WEP、WPA（包含WPA2與WPA3）是用來保障無線網路連線安全的加密方式。WEP的安全性相對較低的，因此建議使用WPA，目前WPA最新版本為WPA3，於2018年6月正式發布，也是目前最安全的Wi-Fi存取保護機制，其次為WPA2。

關於雲端服務可能對隱私造成的影響為下列哪些？(複選題)
A　雲端服務能蒐集龐大的資料
B　雲端資料中心仍有可能因被攻擊或侵入而造成資料外洩
C　就如我們安心把錢放在銀行一樣，個人隱私資料一樣可以安心放在雲端
D　使企業在短時間內即可提供服務，不需增加固定硬體設備與資訊人員
正確解答為：AB　您的作答值為：AB　答對了。TNO：U0080043
題目解析：
雲端運算的服務往往將使用者的資料集中在雲端資料中心，使資料中心更容易成為有心人士攻擊與侵入的對象，因此雲端資料中心有可能因被攻擊或侵入而造成資料外洩，甚至是因為個人的不良使用習慣，而造成個人的帳號密碼被他人取得，進而使得存放於雲端資料中心的資料外洩，因此若要將帶有個人隱私的資料，建議先加密後再上傳雲端較為安全。

關於雲端安全技術的處理流程，下列敘述哪一項錯誤？
A　透過網路進行大量且即時的樣本收集，並將樣本資料送到後端資料中心
B　將樣本資料進行關聯性分析
C　將樣本資料進行各種分析掃描
D　將樣本資料直接傳回給使用者作為判斷的依據
正確解答為：D　您的作答值為：D　答對了。TNO：U0080044
題目解析：
為了因應快速變化的各式威脅，雲端安全技術通常透過網路進行大量且即時的樣本收集，由於這些資料極為龐大，資料探勘與大數據分析需要耗用大量的運算資源，因此需要將樣本資料送到後端資料中心，藉由雲端運算的強大運算能力來處理這些資料，經由資料中心對這些樣本資料進行關聯性的分析（如IP與各種行為之關聯性分析），或對這些樣本資料進行各種分析掃描（如是否為惡意程式行為），並將分析的結果放入資料庫中做為後續判斷的依據。
