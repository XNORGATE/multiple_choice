在Facebook中啟用帳號保安的安全瀏覽功能後，Facebook會盡可能於瀏覽資訊時使用下列哪一項協定連線？
A　FTP
B　HTTPS
C　RMI
D　SSH
正確解答為：B　您的作答值為：B　答對了。TNO：U0030046
題目解析：
HTTPS（HyperText Transfer Protocol Secure，超文本傳輸安全協定）是一種透過計算機網路進行安全通訊的傳輸協定，HTTPS經由HTTP進行通訊，但利用SSL/TLS來加密封包，HTTPS開發的主要目的，是提供對網站伺服器的身分認證，保護交換資料的隱私與完整性。FTP（File Transfer Protocol，檔案傳輸協定）是一種用於在電腦網路上在客戶端和伺服器之間進行檔案傳輸的應用層協定。RMI（Remote Method Invocation），為Java遠端方法呼叫，是Java程式語言裡用於實現遠端過程呼叫的程式設計介面。SSH（Secure Shell，安全外殼協定），是一種加密的網路傳輸協定，可在不安全的網路中為網路服務提供安全的傳輸環境。

瀏覽社群網站或使用即時通訊軟體過程中，下列哪一項使用方式可能造成電腦中毒？
A　點擊連結
B　分享相片
C　打卡
D　更新個人資訊
正確解答為：A　您的作答值為：A　答對了。TNO：U0030048
題目解析：
分享照片、打卡、更新個人資訊都是單純的資訊分享行為，不會造成電腦中毒；而網路上出現的陌生連結，可能經過偽裝的惡意連結，因此看到可疑或是陌生的網址千萬別點下去。

許多民眾LINE訊息曾收到詐騙毒訊息，如「免費電影線上看」、「LINE免費貼圖」等分享連結，下列哪些方式可以避免LINE詐騙？(複選題)
A　開啟「隱私設定」中「阻擋訊息」的功能，可避免接收到非好友傳送的訊息
B　做好帳號保護，避免多個帳號設定同組密碼
C　LINE好友傳訊息時，一定值得信任，訊息中的連結可以直接點擊
D　取消「我的帳號」中「允許自其他裝置登入」，避免駭客取得帳密後從電腦登入
正確解答為：ABD　您的作答值為：ABD　答對了。TNO：U0030049
題目解析：
即使是LINE好友傳訊息，也無法保證好友帳號是否被盜用，或是朋友未驗證訊息就直接轉發，因此好友傳的訊息中若有連結，亦可能經過偽裝的惡意連結，因此建議可先向發送訊息的朋友查證。
