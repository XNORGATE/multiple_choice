下列哪一項技術或工具不支援動態網頁的功能？
A　HTML5
B　HTML3
C　Java
D　Flash
正確解答為：B　您的作答值為：B　答對了。TNO：U0010016
題目解析：
HTML5是HTML最新的修訂版本，由全球資訊網協會（W3C）於2014年10月完成標準制定，與Java、Flash皆支援動態網頁設計。HTML3僅支援靜態網頁設計。

關於Google Chrome中「書籤」的主要功能，為下列哪一項？
A　搜尋網路上的資料
B　快速移到常去的網站
C　執行網頁應用程式
D　執行伺服器應用程式
正確解答為：B　您的作答值為：B　答對了。TNO：U0010017
題目解析：
Google Chrome的「書籤」功能類似IE或Edge的「我的最愛」，主要提供使用者能夠快速的前往常去的網站。

關於Microsoft Edge中「我的最愛」，下列敘述哪一項錯誤？
A　在Google Chrome或Mozilla Firefox中稱之為「書籤」
B　可以是單一網頁，也可用資料夾方式整理
C　某一網頁的網址中如果有變數，無法存入「我的最愛」
D　大部分的主流瀏覽器都可備份回復「我的最愛」
正確解答為：C　您的作答值為：C　答對了。TNO：U0010018
題目解析：
網頁中的變數通常會夾帶在網址後方，因此亦能存入「我的最愛」。
4
下列哪一項不是網頁中執行的應用程式？
A　ActiveX
B　JavaScript
C　Java
D　Servlet
正確解答為：D　您的作答值為：C　回答錯誤了。TNO：U0010019
題目解析：
Servlet（Server Applet），是用Java編寫的「伺服器端」程式，並執行於支援Java的應用伺服器中，因此不屬於網頁中執行的應用程式。

Microsoft Edge支援iOS、Android、Windows、macOS等作業系統。

若想檢視網頁的HTML檔案，在Google Chrome中，下列哪一項方法無效？
A　[Ctrl]+[U]鍵
B　在頁面上按右鍵，選擇「檢視網頁原始碼」
C　[F12]鍵
D　將網址直接拖曳至記事本
正確解答為：D　您的作答值為：D　答對了。TNO：U0010021
題目解析：
[Ctrl]+[U]鍵、[F12]鍵、右鍵選擇「檢視網頁原始碼」皆能夠檢視網頁的HTML程式碼。

想在網頁瀏覽器正確顯示不同國家的文字，最好選擇下列哪一項編碼？
A　Big5
B　ASCII
C　GBK
D　Unicode（UTF-8）
正確解答為：D　您的作答值為：D　答對了。TNO：U0010022
題目解析：
ASCII編碼使用7 bits表示一個字元，僅能表示128個字，GBK收錄21,886個漢字和圖形符號，Big5碼的中文字集僅有13,051個中文字，皆無法完整支援不同國家的文字。UTF-8是一種針對Unicode的可變長度字元編碼，能夠收錄與正確顯示不同國家的文字。

關於Google Chrome搜尋瀏覽網頁中的文字，下列敘述哪一項錯誤？
A　按下鍵盤上的[Ctrl]+[F]鍵，可直接尋找
B　按下右上角的功能鈕，選擇「尋找」
C　使用Google搜尋引擎
D　搜索到的資料會以黃色標記，並顯示找到的數量
正確解答為：C　您的作答值為：C　答對了。TNO：U0010023
題目解析：
使用Google搜尋引擎是搜尋有關鍵字的網頁，並非搜尋目前瀏覽網頁中的文字。

A　Internet Explorer
B　Edge
C　Firefox
D　Chrome
正確解答為：B　您的作答值為：B　答對了。TNO：U0010024
題目解析：
Microsoft Edge為Windows 10的預設瀏覽器。2016年1月6日，微軟宣布將會停止發布Internet Explorer 11之前版本的安全性更新，自此Internet Explorer已慢慢走入歷史，而由Microsoft Edge取代成為Windows預設瀏覽器。

下列哪些瀏覽器可以透過帳號登入的方式，在其他電腦上同步顯示原電腦上設定的「書籤」或「我的最愛」？(複選題)
A　Microsoft IE
B　Mozilla Firefox
C　Apple Safari
D　Opera

下列哪一項技術讓瀏覽器的使用操作，更接近桌面的使用操作，如放大縮小及拖曳？
A　Java
B　CSS
C　PHP
D　JavaScript + AJAX
正確解答為：D　您的作答值為：D　答對了。TNO：U0010029
題目解析：
AJAX(Asynchronous JavaScript and XML），是一套綜合了多項技術的瀏覽器端網頁開發技術，能夠做出視覺效果與互動行為， 在視覺效果上，能改變透明度、大小、位置，使網頁產生更多的變化，讓引導使用者與程式互動更加簡易。而在互動介面上，提供了拖曳元件、水平、垂直拉動捲軸、元件排序等，讓使用者的使用上更加便利與流暢。
15
在瀏覽器中，最常見決定頁面樣式的檔案格式為下列哪一項？
A　HTML
B　XML
C　CSS
D　Java
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：U0010030
題目解析：
CSS（Cascading Style Sheets），是一種用來為結構化文件（如HTML網頁）添加頁面樣式與設計，CSS必須與HTML或XML一同運作，其中HTML負責確定網頁中有哪些內容，CSS確定以何種外觀(大小、粗細、顏色、字型、間距、對齊和位置)展現這些元素。
