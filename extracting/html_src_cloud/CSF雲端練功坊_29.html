網路中有許多可能暴露個人隱私的資料，下列哪一項並非其中之一？
A　瀏覽器的Cookie
B　個人數位憑證
C　網站伺服器上的紀錄
D　瀏覽器的瀏覽紀錄
正確解答為：B　您的作答值為：B　答對了。TNO：U0080016
題目解析：
數位憑證是在網際網路上唯一識別人員和資源的電子檔案。憑證就好像一張身分證，用來識別持有者並提供其他重要資訊，憑證由稱為憑證授權單位（CA）的可信任機構發行，CA將驗證憑證持有者的身份並對憑證進行簽署，使他人無法偽造或竄改憑證，CA對憑證進行簽署之後，持有者可以提供該憑證做為身份證明並建立加密的機密通訊，透過這種資料交換模式，擁有者既可證實自己的身分，亦不用過度披露個人資料，對電腦服務存取雙方皆有保障。

關於瀏覽器Cookie的敘述，哪些正確？(複選題)
A　Cookie可用於儲存各種資訊，包含個人識別資訊
B　瀏覽器關閉後，Cookie會自動清除
C　Cookie可能會洩漏用戶的隱私資料
D　Cookie屬於瀏覽器重要功能，沒有辦法關閉
正確解答為：AC　您的作答值為：AC　答對了。TNO：U0080017
題目解析：
Cookie是使用者造訪過的網站所建立的檔案，可儲存使用者的瀏覽資訊，提供使用者更流暢與方便的使用體驗。儲存在客戶端中的Cookie，按照儲存位置，可分為暫時性Cookie和持續性Cookie，暫時性Cookie會再瀏覽器關閉或階段性工作完成後就消失，持續性Cookie儲存在硬碟裡，除非用戶手動清理或到了過期時間，否則Cookie不會被刪除。主流的瀏覽器都可以選擇關閉或開啟Cookie，並且可讓使用者選擇性刪除Cookie。
3
防火牆（Firewall）不具有下列哪一項功能？
A　阻擋由內部向外部網路的連線
B　阻擋由外部向內部網路的連線
C　封鎖某一個連接埠
D　網路頻寬管理
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：U0080018
題目解析：
防火牆最基本的功能就是隔離網路，通過將網路劃分成不同的區域，制定出不同區域之間的存取控制策略來控制不同信任程度區域間傳送的資料。網路頻寬管理較屬於路由器的功能，而非防火牆的功能。
4
關於防火牆（Firewall）在網路安全上的應用，下列敘述哪些正確？(複選題)
A　從內部網路向網際網路的連線，都需經過防火牆
B　從網際網路向內部網路的連線，都需經過防火牆
C　符合防火牆政策的連線才可以通過防火牆
D　伺服器需要大量資料傳輸時，可放置在防火牆外面靠網際網路這一側，並區分內外部資源，以降低防火牆負荷
正確解答為：ABCD　您的作答值為：BC　回答錯誤了。TNO：U0080019
題目解析：
防火牆是位於兩個或多個網路之間，實行網路間存取控制，是目前最重要的一種網路防護裝置，通過將網路劃分成不同的區域，制定出不同區域之間的存取控制策略來控制不同信任程度區域間傳送的資料。內部網路向外部網路，或是外部網路向內部網路的連線，都須經過防火牆，且需符合防火牆的設定規則，才能通過。由於所有網際網路的封包都會經過防火牆的過濾，將造成網路速度的瓶頸，因此當伺服器需要大量的資料傳輸時，可將其放置在防火牆外面靠網際網路這一側，並區分內外部資源，以降低防火牆負荷。

B　偵測巨集病毒
C　阻止通訊埠掃描
D　攔截夾有電腦病毒檔案的郵件
正確解答為：ABD　您的作答值為：ABD　答對了。TNO：U0080020
題目解析：
大多數防火牆都能夠監測埠掃描（Port Scan）活動，藉此阻止通訊埠掃描攻擊。雖然應用層防火牆可藉由監測不符合規則的封包，防範電腦蠕蟲或木馬程式的攻擊，但木馬程式或病毒的偵測與清除，僅能依靠防毒軟體來處理。

關於密碼的設定，下列敘述哪些錯誤？(複選題)
A　密碼長度設定的短一點，只有4位數是很安全的
B　密碼設定的跟帳號一樣即可
C　使用越長的密碼越好
D　利用自己的生日作為密碼
正確解答為：ABD　您的作答值為：ABCD　回答錯誤了。TNO：U0080022
題目解析：
2017年6月，國家標準技術研究所（NIST）發布新的「800-63」指引第三版，改正對於密碼的舊建議，不再強調使用難以記憶的英文、數字、特殊符號的無意義組合，而且亦不再建議密碼需每90天汰換。與其使用一些複雜、看似亂碼的短密碼，選擇幾組自己熟悉的單字或數字，再湊成一個長密碼，除了讓自己容易記憶外，又能有足夠的安全性的重要原則，只要密碼夠長，拼合多個看似無意義但便於記憶的字詞作為密碼，更能有效抵擋駭客攻擊，暴力破解會需要更久的時間。

關於網路釣魚（Phishing），下列敘述哪些正確？(複選題)
A　網路釣魚（Phishing）是種誘騙使用者透過電子郵件或網頁提供個人資訊的手段
B　網路釣魚（Phishing）信件看來就像真的通知信一樣，誘引使用者到不正確的詐騙網站上
C　Phishing一詞，是Fishing和Phone的綜合體，由於駭客始祖起初是以電話作案，所以用Ph來取代F，創造了Phishing
D　在網路釣魚（Phishing）的詐騙網站中，通常會要求使用者提供帳號密碼以便進行下一步的盜用
正確解答為：ABCD　您的作答值為：CD　回答錯誤了。TNO：U0080023
題目解析：
網路釣魚（Phishing）是一種企圖從電子通訊中，透過將電子郵件或網頁偽裝成可信的組織單位，讓電子郵件就像真的通知信一樣，誘引使用者到不正確的詐騙網站上，以獲得使用者資料，如使用者名稱、密碼、身分證字號、生日、信用卡資料等個人資訊的詐騙過程。

關於曾經衝擊全球的「Melissa」及「ILOVEYOU」病毒，下列敘述哪一項錯誤？
A　藉由Email主動散播
B　透過微軟的電子郵件軟體Outlook散播
C　透過社交工程誘使使用者開啟附件

流傳廣泛的KAVO、TAVO等病毒主要以下列哪一項方式傳遞？
A　電子郵件
B　安裝檔案
C　外接儲存裝置

下列哪些行為可能會讓電腦感染電腦病毒？(複選題)
A　閱讀來歷不明的電子郵件
B　使用隨身碟等外接儲存裝置
C　隨意下載網路上的檔案
D　安裝盜版軟體
正確解答為：ABCD　您的作答值為：ABCD　答對了。TNO：U0080026
題目解析：
雖然安裝防毒軟體可預防大多數的電腦病毒，但由於電腦病毒的種類日新月異，這些新型電腦病毒的散播、潛伏、發現到防毒軟體公司更新病毒定義檔，或作業系統的安全更新，都會有一段的空窗期時間，在這段空窗期內，感染這些新型電腦病毒的風險相對很高，所以除了安裝防毒軟體外，使用者亦需有良好的使用習慣，避免電腦受到病毒感染。如不要開啟或閱讀來歷不明的電子郵件、不要隨意下載網路上的檔案、不要安裝來路不明的盜版軟體等，在使用隨身碟等外接儲存裝置時也建議先用防毒軟體掃描，並且關閉自動執行功能。

近年來勒索病毒已成為全球關注的網路安全焦點之一，關於勒索病毒的敘述，哪些正確？(複選題)
A　勒索病毒可能透過鎖住電腦或將資料加密，要求受害者繳交贖金
B　攻擊目標僅有在個人電腦上
C　大部分的勒索病毒感染途徑是釣魚電子郵件
D　瀏覽網頁不會感染勒索病毒
正確解答為：AC　您的作答值為：AC　答對了。TNO：U0080027
題目解析：
勒索病毒是一種惡意程式，其散播途徑與一般惡意程式或電腦病毒一樣，常透過假冒成普通電子郵件等社交工程方法欺騙受害者點擊連結下載、將惡意程式植入網路廣告誘使受害者點擊、利用系統漏洞在聯網的電腦間傳播。一旦感染，它會鎖住受害者的電腦，或者將受害者電腦的硬碟資料加密，進一步要求受害者繳交贖金後，才有機會取回對電腦的控制權或者將硬碟加密解開。近年來勒索病毒攻擊目標逐漸從電腦轉向智慧型手機或其他行動裝置，勒索病毒可能偽裝成合法Apps被使用者下載與安裝，進而取得手機的控制權。要避免電腦受到勒索病毒威脅，與防範任何惡意程式的方式一樣，平常使用電腦網路的時候應謹慎，不要開啟或閱讀來歷不明的電子郵件、不要隨意下載網路上的檔案、不要安裝來路不明的盜版軟體等。另外最重要的是，定期備份資料在另外一個獨立裝置，萬一被勒索病毒威脅，還可以找回重要資料。

A　利用防火牆阻擋外部網路對內部電腦的不正常連線
B　安裝防毒軟體
C　阻止程式主動對外部網路連線
D　關閉電腦的自動執行（AutoRun）功能
