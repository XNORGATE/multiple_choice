1
使用Gmail或Hotmail收發電子郵件，屬於下列哪一項類型雲端服務？
A　SaaS（Software as a Service）
B　PaaS（Platform as a Service）
C　IaaS（Infrastructure as a Service）
D　STaaS（Storage as a Service）
正確解答為：A　您的作答值為：A　答對了。TNO：U0040001
題目解析：
SaaS（Software as a Service）向使用者提供應用程式或軟體的雲端服務，在這種服務模式中，使用者僅需透過網際網路，不需經過傳統的安裝步驟即可使用服務，常見範例為電子郵件（Gmail、Hotmail、Yahoo Mail等）、行事曆（Google Calendar）以及Office工具（Google Docs、Microsoft Office 365）。
2
想要選用雲端服務供應商（Cloud Service Provider, CSP或稱Cloud Service Vendor, CSV）的虛擬主機，建立專屬的應用伺服器主機時，可選擇下列哪些雲端服務？(複選題)
A　Amazon EC2
B　Google App Engine
C　Microsoft Azure
D　Dropbox
正確解答為：AC　您的作答值為：AC　答對了。TNO：U0040002
題目解析：
Amazon EC2是Amazon提供的虛擬機器服務，是AWS（Amazon Web Services）眾多雲端服務中的一種。Microsoft Azure是微軟的公用雲端服務，服務包含了Virtual Machine、RemoteApp、Cloud Service、Service Fabric、Azure Kubernetes Service、Azure Application Services等服務。Google App Engine主要提供開發和代管網路應用程式的平台。Dropbox主要提供雲端儲存服務。
3
由於網路的蓬勃發展，出現越來越多種不同型態的終端設備，下列哪一項裝置為本身無需實體儲存空間與應用程式，而是讓雲端伺服器進行所有數據處理，再由用戶端顯示資訊？
A　智慧型手機（Smartphone）
B　平板電腦（Tablet PC）
C　精簡型用戶端（Thin Client）
D　複雜型用戶端（Thick Client）
正確解答為：C　您的作答值為：C　答對了。TNO：U0040003
題目解析：
Thin Client（精簡版客戶端）指的是在Client-Server網絡體系中一個無需實體儲存設備與應用程式的終端設備，Thin Client將其滑鼠、鍵盤等輸入傳送到伺服器處理，伺服器再把處理結果回傳至Thin Client顯示。

下列哪一項不是美國國家標準與技術研究院（NIST）所定義雲端運算的五項基本特性？
A　隨選的自助服務（On-demand Self-service）
B　安全的網路存取（Secure Network Access）
C　資源池（Resource Pooling）
D　迅速重新部署的靈活度（Rapid Elasticity）
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：U0040004
題目解析：
NIST定義的雲端運算五項特徵為：On-demand Self-service（隨需選取的自助式服務）：客戶可以依其需求索取雲端資源。Broad Network Access（廣泛網路接取）：服務是經由網路提供，因此能讓不同平台的客戶端都可以使用。Resource Pooling（共享資源池）：服務者所提供的運算資源如同為一個大水池，能隨時依使用者需求重新分配。Rapid Elasticity（迅速重新部署的靈活性）：運算資源可以快速且有彈性地被提供或釋放，且對使用者而言，資源是依據需求索取的。Measured service（可量測的服務）：使用者與服務提供者雙方都可隨時且透明的監控資源使用情況。

D　STaaS（Storage as a Service）

著名的網路書店亞馬遜（Amazon）也是虛擬化服務的主要供應商，其Amazon EC2（Amazon Elastic Compute Cloud）提供下列哪一項類型的虛擬化服務？
A　SaaS（Software as a Service）
B　PaaS（Platform as a Service）
C　IaaS（Infrastructure as a Service）
D　STaaS（Storage as a Service）
正確解答為：C　您的作答值為：C　答對了。TNO：U0040007
題目解析：
Amazon AWS(EC2、S3)，屬於IaaS產品，EC2提供虛擬伺服器，讓使用者透過網頁租用，在幾分鐘內即可取得如同實體伺服器一樣的運算資源，操作與管理上也與實體伺服器相同。Amazon AWS(Cloud9、CodeStar)較屬於PaaS服務。

在電子郵件傳遞過程中，下列哪一項DNS中的資源紀錄類型的功能是標示郵件伺服器主機？
A　A主機紀錄
B　CNAME別名紀錄
C　MX郵件交換紀錄
D　NAPTR指標紀錄
正確解答為：C　您的作答值為：C　答對了。TNO：U0040008
題目解析：
DNS系統中，常見的資源記錄類型有：主機記錄（A記錄）：A記錄是用於名稱解析的重要記錄，它將特定的主機名對映到對應主機的IP位址上。別名記錄（CNAME記錄）：CNAME記錄用於將某個別名指向到某個A記錄上，這樣就不需要再為某個新名字另外建立一條新的A記錄。IPv6主機記錄（AAAA記錄）：與A記錄對應，用於將特定的主機名對映到一個主機的IPv6位址。服務位置記錄（SRV記錄）：用於定義提供特定服務的伺服器的位置，如hostname、port number等。PTR記錄：提供了正規表示式方式去對映一個域名，定義某個IP對應的domain name，即將IP位址轉換成主機的FQDN。郵件交換記錄(MX記錄)：是用於指定負責處理發往收件人域名的郵件伺服器。

在雲端運算中，任何事物都可以被視為一種服務（Anything as a Service, XaaS或稱Everything as a Service, XaaS）時，最重要的基礎為下列哪一項技術？
A　高可用性
B　異地備援
C　虛擬化
D　資料加密
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：U0040009
題目解析：
虛擬化技術透過軟體模擬硬體功能，進而建立虛擬的電腦系統，虛擬化技術提供不同應用軟體或虛擬機器能使用或共享實體伺服器的資源，讓不同的軟體和虛擬機器在同一個伺服器上面運行，以達到充分利用資源與迅速重新部署的靈活性。

當使用HTTPS進行網站連線時，可獲得下列哪些好處？(複選題)
A　網站的身分正確性
B　流量傳輸的品質
C　資料傳輸的安全
D　預防電腦病毒
正確解答為：AC　您的作答值為：C　回答錯誤了。TNO：U0040011
題目解析：
HTTPS（HyperText Transfer Protocol Secure，超文本傳輸安全協定）是一種透過網際網路進行安全通訊的傳輸協定。HTTPS經由HTTP進行通訊，但利用SSL/TLS來加密封包。HTTPS開發的主要目的，是提供對網站伺服器的身分認證、保護交換資料的隱私與完整性、保護帳戶和保持用戶通信、身分和網路瀏覽的私密性。

瀏覽網頁時，所使用的網址為https://tw.yahoo.com/，其又稱為下列哪一項？
A　ARC（Advanced RISC Computing）
B　DNS（Domain Name System）
C　URL（Uniform Resource Locator）
D　UNC（Uniform Naming Convention）
正確解答為：C　您的作答值為：C　答對了。TNO：U0040013
題目解析：
URL（Uniform Resource Locator，統一資源定位器）俗稱網頁位址或簡稱網址，是網際網路上標準的資源的位址，如同在網路上的門牌號碼，如「https://www.google.com/」或「https://tw.yahoo.com/」。統一資源定位符的標準格式為：[協定類型]://[伺服器位址]:[埠號]/[檔案路徑][檔名]?[查詢]#[片段ID]

使用SSL（Secure Sockets Layer）的方式，與網站進行安全性連線時，使用下列哪一項資訊進行網站身分識別？
A　電腦名稱
B　IP位址
C　使用者帳號
D　數位憑證（Digital Certificate）
正確解答為：D　您的作答值為：D　答對了。TNO：U0040014
題目解析：
SSL（Secure Sockets Layer，安全通訊協定）是一種安全協定，目的是為網際網路通訊提供安全及資料完整性保障，一旦客戶端和伺服器都同意使用SSL協定，會通過交握過程協商出一個有狀態的連接以傳輸資料，交握過程中伺服器會發回其數位憑證，此憑證通常包含伺服器的名稱、受信任的憑證頒發機構（CA）和伺服器的公鑰，讓客戶端確認其憑證的有效性。
