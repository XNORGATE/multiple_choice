關於網際網路使用的IPv4與IPv6的網路通訊協定，下列敘述哪些正確？(複選題)
A　IPv4能夠提供的IP位址數目約為2的32次方個
B　IPv6能夠提供的IP位址數目約為2的64次方個
C　從IPv4轉移到IPv6需要一段時間，所以會有IPv4與IPv6共同存在的情況
D　IPv6在服務品質上比IPv4更適合串流技術（Streaming）的應用
正確解答為：ACD　您的作答值為：AB　回答錯誤了。TNO：U0050001
題目解析：
IPv4使用32位元位址，IPv6採用128位元的位址，IPv6的目標是取代IPv4，雖然IPv6在1994年就已被IETF指定作為IPv4的下一代標準，但由於早期只支援IPv4的網路設備及相關應用程式皆須替換，因此在IPv6完全取代IPv4前，需要一些轉換機制，所以目前仍以IPv4與IPv6雙架構並存居多。IPv4與IPv6定義定址方式，因此與服務品質差異或連線穩定度並無相關。
2
IPv6是第六版的IP網路協定規範，下列哪一項IP位址的表示法不符合IPv6位址的規定？
A　FE80：403A：A2E8：0130：7879：235E：FFAC：AC16
B　403A：EF4：：A23
C　403A：：EF4：A23
D　FE23：：AC12：2781：：AC16
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：U0050002
題目解析：
IPv6長度為128位元，每16位元為一組，每組以冒號「:」隔開，可以分為8組，每組以4位16進位方式表示，如選項A「FE80:403A:A2E8:0130:7879:235E:FFAC:AC16」。IPv6在每項數字前導的0可以省略，如「2001:0DB8:02de:0000:0000:0000:0000:0e13」可以直接表示成「2001:DB8:2de:0:0:0:0:e13」。IPv6可以用雙冒號「::」表示一組0或多組連續的0，但只能出現一次。選項(D)「FE23::AC12:2781::AC16」因為出現兩次雙冒號「::」，所以不符規定，因為它有可能是「FE23:0000:0000:0000:AC12:2781:0000:AC16」或「FE23:0000:AC12:2781:0000:0000:0000:AC16」等情形之一，造成無法推斷。

B　eMule

網際網路早期的發展跟下列哪一項網路有關？
A　TANet
B　ARPANET
C　BITNET
D　USENET
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：U0050004
題目解析：
網際網路的起源可以追溯到1960年代美國聯邦政府委託進行的一項研究，目的是建立具備容錯的電腦通訊技術，美國國防部高等研究計劃署（ARPA）建立的ARPANET成為了網際網路發展的中心。TANet是臺灣學術網路（1990年代建立），BITNET是美國紐約市立大學和耶魯大學建立的一個學術研究網路（1980年代建立），USENET是一種分散式的網際網路交流系統（1980年代建立）。

網際網路所採用的串流技術（Streaming），主要是讓影音資料的遞送與播放達到下列哪一項效果？
A　影音資料全部傳遞完成之後才開始播放
B　影音資料一邊傳遞一邊在用戶端開始播放
C　避免遭到駭客攔截竊取影音資料
D　讓影音資料的顏色更鮮豔好看
正確解答為：B　您的作答值為：B　答對了。TNO：U0050005
題目解析：
Streaming串流技術是指將一連串的多媒體資料壓縮後，經過網際網路分段傳送資料，在網際網路上即時傳輸影音以供使用者觀賞的技術與過程，如果不使用此技術，就必須在觀賞前下載整個多媒體檔案。YouTube、Netflix、愛奇藝、Apple TV+、HBO Go、Amazon Prime Video、Spotify、Twitch、Steam都使用了Streaming串流技術。

以網域名稱的通用規範可判斷，下列哪一項最可能是教育機構使用的網址？
A　www.chinsan.edu
B　www.chinsan.gov
C　www.chinsan.com
D　www.chinsan.org
正確解答為：A　您的作答值為：A　答對了。TNO：U0050006
題目解析：
域名由一或多個部分組成，這些部分通常連接在一起，並由點分隔，例如www.google.com最右邊的一個標籤是頂級域名，頂級域名主要分成ccTLD（Country Code Top-level domains，國家和地區頂級域）和gTLD（General Top-level domains，通用頂級域）兩類。gTLD代表了組織類別，包括.edu（供教育機構使用），.gov（供政府及其屬下機構使用），.com（供商業機構使用），.mil（供軍事機構使用），.org（供非營利組織使用），.net（供網路服務供應商使用），.int（國際組織）等。

下列哪一項檔案格式不適合用來支援串流技術（Streaming）的視訊檔案格式？
A　Apple公司發展的MOV檔案格式
B　Microsoft公司發展的WMV檔案格式
C　Microsoft公司發展的AVI檔案格式
D　RealNetworks公司發展的RM檔案格式
正確解答為：C　您的作答值為：C　答對了。TNO：U0050007
題目解析：
AVI（Audio Video Interleave）是由Microsoft在1992年11月推出的一種多媒體檔案格式，檔案結構分為「Header」，「Chunks」和「Index」三部分。AVI本身只是提供了這樣一個框架，內部的影音編碼格式可以是任意的編碼形式，但由於索引放在檔案尾部，因此不適合作為串流媒體的檔案格式。

網際網路以IP位址來辨識網路上的節點，一般電腦的網路卡則以一種含有6組16進位數字的實體位址，區分網路區段上的電腦，這種位址的名稱是下列哪一項？
A　MAC位址（Media Access Control Address）
B　網路廣播位址
C　網路群播位址
D　LDAP位址（Lightweight Directory Access Protocol Address）
正確解答為：A　您的作答值為：A　答對了。TNO：U0050011
題目解析：
MAC位址（Media Access Control Address）是一個用來確認網路裝置位置的位址，MAC位址用於在網路中唯一標示一個網卡，每個網路卡都會有一個唯一的MAC位址，MAC位址48位元，以6組16進位表示，例如「00:0A:0B:0C:0D:0E」。

全球的網際網路IP位址分配以及網域名稱（Domain Name）的管理工作，目前由下列哪一項機構負責？
A　IEEE
B　ISO
C　NSF
D　ICANN
正確解答為：D　您的作答值為：A　回答錯誤了。TNO：U0050012
題目解析：
ICANN（Internet Corporation for Assigned Names and Numbers）是位於美國加州的非營利組織，主要由網際網路協會的成員組成，建立於1998年9月18日，目的是管理網域名稱和IP位址的分配等與網際網路相關的任務。IEEE（Institute of Electrical and Electronics Engineers，電機電子工程師學會）是國際性電子技術與電子工程師協會。ISO（International Organization for Standardization，國際標準化組織）是制定全世界工商業國際標準的國際標準建立機構。NSF（National Science Foundation，國家科學基金會）是一個美國政府獨立機構，該機構支持除醫學領域外的科學和工程學基礎研究和教育，負責醫學的同類機構為國家衛生研究院。

關於「網域名稱轉址（URL Redirection）」技術，下列敘述哪一項錯誤？
A　在使用者瀏覽某個網址時，將其導向另一個網址
B　在使用者輸入錯誤的網址時，試著將其導向正確的網址
C　讓使用者以更快的速度連上入口網站
D　將很長的網址轉成短的網址
正確解答為：C　您的作答值為：C　答對了。TNO：U0050014
題目解析：
網域名稱轉址（URL Redirection）技術是指當使用者瀏覽某個網址時，將其導向另一個網址的技術，常用在把一串網址很長的網站轉成較短的網址，也常用在使用者輸入錯誤的網址時，重新導向到正確的網址，例如Google就註冊了「googel.com」、「gooogle.com」等這些拼字不正確的網域名稱，並將其重新導向到正確的「google.com」。URL Redirection並無法讓使用者以更快的速度連上入口網站。
