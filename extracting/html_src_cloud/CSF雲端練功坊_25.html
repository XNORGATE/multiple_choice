阿德在瀏覽網頁的時候，將網頁儲存到自己的電腦上，其行為是否違法，下列敘述哪一項正確？
A　要看離線瀏覽時，對於網頁資料的使用行為是否在合理使用的範圍內才能判定
B　是一種違法的重製行為
C　是合法的行為，因為是儲存在自己的電腦上
D　假如是完整的網頁儲存則屬於違法的行為
正確解答為：A　您的作答值為：A　答對了。TNO：U0070018
題目解析：
離線瀏覽不屬於網路合法中繼性傳輸之暫時性重製情形，因此是否侵害著作財產權人之「重製權」，需要視離線瀏覽行為是否符合著作權法所規定重製行為的合理使用情形之一。依據著作權法第51條規定：「供個人或家庭為非營利之目的，在合理範圍內，得利用圖書館及非供公眾使用之機器，重製已公開發表之著作。」若離線瀏覽僅供個人為非營利目的的重製行為，只要是使用自己的電腦設備，而且重製的質量與比例是在合理的範圍內，對該網站網頁上相關著作的潛在市場與現在價值，不會造成不利的影響，應該可以主張離線瀏覽行為屬於重製行為的合理使用。

小明將台北市各著名小吃店的相關內容整理放在自己的網站上，他能否主張著作權，下列敘述哪一項正確？
A　可以的，別人不能隨意使用，要遵循著作權法的規範
B　必須被認定為具有創作性質的編輯著作方能主張著作權
C　不能主張著作權，因為台北著名小吃眾人皆知，不算創作
D　若取得店家同意則可主張著作權
正確解答為：B　您的作答值為：B　答對了。TNO：U0070019
題目解析：
對於作品是否受著作權保護的判斷標準大致可歸納為五項要件：一、必須是人類精神力作用的成果。二、必須經由表達而外顯。三、必須獨立創作且具有創作性。四、必須屬於文學、科學、藝術或其他學術範圍。五、不屬於著作權法第9條明文規定不保護的標的。因此小明在網站上所整理的相關內容，若是具有創作性質的編輯著作方能主張著作權。

共享軟體（Shareware）或免費軟體（Freeware）在下列哪一項情況下仍受著作權法保護？
A　作者死亡超過五十年
B　作者放在網路上供人下載使用
C　作者宣稱放棄著作權
D　原創作者已不可考
正確解答為：B　您的作答值為：B　答對了。TNO：U0070020
題目解析：
軟體屬於著作權法第五條的電腦程式著作，依著作權法第十條規定，著作人一旦完成著作，就受著作權法的保護，享有著作人格權與著作財產權。除非作者宣稱放棄著作權或是原創作者已不可考，而其年限依著作權法第30條規定：「著作財產權，除本法另有規定外，存續於著作人之生存期間及其死亡後五十年。」

關於自由軟體是否受著作權法的保護，下列敘述哪一項正確？
A　自由軟體完全公開程式碼，已經沒有保護的必要
B　仍受著作權法的保護以及自由軟體契約的約束
C　不論在什麼狀況下都不得修改後從事營利的用途
D　可自由下載使用，但不能再複製儲存另外一份
正確解答為：B　您的作答值為：B　答對了。TNO：U0070021
題目解析：
自由軟體（Free Software）根據自由軟體基金會對其的定義，是一類可以不受限制地自由使用、複製、研究、修改和分發的，尊重使用者自由的軟體。自由軟體不一定是免費軟體，並受到選定的「自由軟體授權協定」保護而發布。大部分的自由軟體可以無需費用自由取得，且其軟體原始碼可以自由修改並散布，但它仍受到著作權法的保護，軟體屬於著作權法第五條的電腦程式著作，依著作權法第十條規定，著作人一旦完成著作，就受著作權法的保護。

小鐘在沒有取得對方同意的情況下於自己的網頁上建立了一個連往歷史博物館的超連結（Hyperlink），其行為是否違法，下列敘述哪一項正確？
A　要看該超連結內容是否侵權來判定是否違法
B　其行為是違法的
C　其行為不違法，因為沒有複製其內容
D　要看網頁中引用內容的比例有多少來判定是否違法
正確解答為：A　您的作答值為：C　回答錯誤了。TNO：U0070022
題目解析：
參考經濟部智慧財產局民國103年8月28日電子郵件1030828號函釋、經濟部智慧財產局民國102年11月19日智著字第10200092090號函釋、經濟部智慧財產局民國102年6月4日電子郵件1020604b號函釋等內容，對於「提供超連結」或是「使用嵌入式語法」來連結外部未經授權之著作，認為提供超連結不構成重製及公開傳輸，且「使用嵌入式語法」也不屬於重製及公開傳輸。但應注意，如果明知該著作是未經合法授權之著作，仍有可能與違法上傳者，成立公開傳輸之共同正犯或幫助犯。另外需注意的是，並非所有網路內容只要提供嵌入機制，就能藉此分享至其他網路平台，還需要視其服務使用條款而定。如YouTube、Twitter等大多數平台的使用規範，在使用者上傳內容至平台時，必須連帶同意讓服務平台以二次授權形式，允許第三方用戶透過嵌入等方式使用。但是如Instagram平台，雖然在使用條款中包含二次授權使用模式，但在其提供的嵌入機制中，並未包含二次授權使用，因此第三方用戶想要藉由嵌入方式使用他人上傳的照片，最好還是先取得原創作者授權，否則可能面臨侵權訴訟。

關於網路著作的保護，下列敘述哪一項正確？
A　發表文章在報章雜誌上，因屬公開，已喪失著作權保護
B　著作在發表到網路上時，即受到著作權保護
C　若非營利性質，可隨意使用網路上的著作
D　轉載文章可隨意修改內容以符合實際需要
正確解答為：B　您的作答值為：B　答對了。TNO：U0070027
題目解析：
依著作權法第十條規定，著作人一旦完成著作，就受著作權法的保護。作者將文章或照片等張貼在報章雜誌或網路上時，並不代表作者同意網友可以任意轉載或作其他利用，若要合法利用他人著作只有二種途徑，一個是取得著作權人合法授權，另一個則是符合著作權法所規範的合理使用情形。而轉載文章也不可隨意修改其內容，且應清楚註明其出處。
