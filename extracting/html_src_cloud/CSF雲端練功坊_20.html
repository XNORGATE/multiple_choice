假如某ISP業者聲稱所提供的上網速率可達到「下行1G/上行600M」，則使用時真正得到實際的速率最有可能為下列哪一項？
A　下行800M/上行800M
B　下行800M/上行300M
C　下行1.2G/上行300M
D　下行1.2G/上行800M
正確解答為：B　您的作答值為：B　答對了。TNO：U0050046
題目解析：
ISP業者聲稱所提供的上網速率可達到「下行1G/上行600M」，為其下行與上行速率之上限值（單位為bps），因此實際速率只有選項(B)的「下行800M/上行300M」有可能。選項(A)上行速率超過上限的600M。選項(C)下行速率超過上限的1G。選項(D)則是下行與上行速率都超過上限，因此不太可能是實際得到的速率。

關於公共區域免費無線上網，下列敘述哪些正確？(複選題)
A　僅限於使用智慧型手機，筆記型電腦不適用
B　提供這類服務的公共區域必須裝設無線通訊熱點
C　必須使用4G蜂巢網路技術
D　經由公共區域免費無線上網可以連上網際網路
正確解答為：BD　您的作答值為：BD　答對了。TNO：U0050047
題目解析：
公共區域免費無線上網（如iTaiwan、Taipei Free）皆採用Wi-Fi無線網路，提供這類服務的公共區域必須裝設無線通訊的熱點，只要利用裝有Wi-Fi網路卡的設備（智慧型手機、平板電腦、筆記型電腦等設備），接收熱點區域所發出的無線訊號，便可連上網際網路。
3
下列哪一項網路協定的主要功能是進行IP位址與MAC位址的轉換，可將IP位址轉換成網路節點的實體位址（Physical Address）？
A　TCP（Transmission Control Protocol）
B　ARP（Address Resolution Protocol）
C　UDP（User Datagram Protocol）
D　DHCP（Dynamic Host Configuration Protocol）
正確解答為：B　您的作答值為：D　回答錯誤了。TNO：U0050048
題目解析：
ARP（Address Resolution Protocol，位址解析協定）是一個通過解析IP位址來找尋MAC位址的網路傳輸協定。TCP（Transmission Control Protocol，傳輸控制協定）則是一種連接導向且可靠的傳輸層通訊協定。UDP（User Datagram Protocol，使用者封包通訊協定）則一個簡單的傳輸層通訊協定，相對於TCP，UDP只提供資料的不可靠傳輸。DHCP（Dynamic Host Configuration Protocol，動態主機設定協定）是一個用於IP網路的應用層網路協定，主要用於內部網路或網路服務供應商自動分配IP位址與相關的參數給用戶。
4
關於點對點（Peer-to-Peer, P2P）檔案分享與網路架構，下列敘述哪些正確？(複選題)
A　參與P2P檔案分享的用戶端需要安裝P2P檔案分享軟體
B　分享的檔案經由一個中央伺服器轉送到需要檔案的用戶端
C　常見的P2P檔案分享軟體都直接使用Telnet程式為用戶的使用介面
D　P2P檔案分享是透過網際網路來傳送與分享檔案資源
正確解答為：AD　您的作答值為：AC　回答錯誤了。TNO：U0050049
題目解析：
P2P檔案分享的用戶端需要安裝P2P檔案分享軟體，常見的P2P檔案分享軟體（如eMule與BitTorrent）皆有圖形化使用者介面方便使用者操作，P2P檔案分享架構注重的是檔案的傳輸與分享，共享的檔案資源並不是在集中的伺服器上等待用戶端來下載，而是分散在所有參與者的硬碟上，這些參與者組成一個虛擬網路，每個用戶端都可以從這個虛擬網路裡的任何一個人的電腦裡下載檔案，而每一個下載者同時都會擔當了上傳的角色，因此越多人下載，即表示擁有越多上傳者將它們部分的上傳寬頻分享出來，然後再平均分給各個下載者，簡單來說，當你下載檔案時，同時你也會將下載完的檔案再分享出來平均分享給每一位下載者，這就是P2P的共享概念。

下列哪一項行動無線通訊上網的資料傳輸速率最高？
A　2G
B　3G
C　4G
D　5G
正確解答為：D　您的作答值為：D　答對了。TNO：U0050050
題目解析：
行動網路的世代演進中，1990年的2G使用頻率850~1900 MHz，峰值速度約10Kbps；2000年的3G使用頻率1.6~2.5 GHz，峰值速度約3.8 Mbps；2010年的4G使用頻率2~8 GHz，峰值速度約0.1~1 Gbps；2020年的5G使用頻率3~300 GHz，峰值速度約1~10 Gbps。
