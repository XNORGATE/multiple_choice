關於Google Chrome，下列敘述哪一項正確？
A　沒有Mac OS版本
B　沒有iOS版本
C　沒有Android版本

下列哪一項不是動態執行的瀏覽器可執行的元件？
A　Java
B　JavaScript
C　ActionScript
D　HTML
正確解答為：D　您的作答值為：C　回答錯誤了。TNO：U0010032
題目解析：
HTML是用於建立網頁的標準標記語言，HTML元素是構建網站的基石，允許嵌入圖像與物件，並且可以用於建立互動式表單，主要用來結構化資訊，瀏覽器使用HTML標籤和指令碼來詮釋網頁內容，因此HTML並非可執行的元件。

將網頁資料暫時放在使用者的電腦上，通常是使用下列哪一項技術？
A　Cookie
B　Visual Basic
C　C++
D　Media Player
正確解答為：A　您的作答值為：A　答對了。TNO：U0010033
題目解析：
因為HTTP協定是無狀態的，亦即伺服器不知道使用者上一個動作做了什麼，這嚴重影響互動式網頁的發展，Cookie就是用來維護使用者跟伺服器互動的狀態。Cookie的典型應用就是當使用者登入一個網站時，輸入帳號和密碼後可以勾選「自動登入」，當下一次存取同一網站時，使用者就可以不用再次輸入帳號和密碼就直接登入。

為了提高瀏覽器瀏覽網頁時的安全性，許多瀏覽器開發商皆加強了產品的防追蹤功能，關於防追蹤功能敘述，何者有誤？
A　Microsoft Edge的追蹤保護功能稱為Tracking Prevention
B　為了防止網站追蹤器收集使用者的瀏覽資料
C　Google Chrome、Mozilla Firefox皆有反追蹤保護功能
D　啟用反追蹤功能瀏覽網頁時不會有任何影響
正確解答為：D　您的作答值為：B　回答錯誤了。TNO：U0010034
題目解析：
封鎖網站的追蹤器，可能會造成部分網站無法依預期方式運作，例如影片可能無法播放、或使用者可能無法登入。
5
Microsoft Edge可以選擇的防追蹤等級有哪些？(複選題)
A　基本
B　平衡
C　嚴格
D　全鎖
正確解答為：ABC　您的作答值為：ABCD　回答錯誤了。TNO：U0010035
題目解析：
Microsoft Edge為了要偵測和封鎖已知的追蹤器，可以選擇三個等級的防止追蹤：基本、平衡、嚴格。預設會選取「平衡」。「基本」：封鎖可能有害的追蹤器，但允許大多數其他追蹤器以及那些將內容和廣告個人化的追蹤器。「平衡」：封鎖可能有害的追蹤器以及使用者未曾瀏覽之網站的追蹤器。「嚴格」：封鎖可能有害的追蹤器以及大多數跨網站的追蹤器，此選項封鎖大多數追蹤器，可能會造成部分網站無法依預期方式運作（如影片可能無法播放、或可能無法登入）。

安全加密的瀏覽通訊協定為下列哪一項？
A　FTP
B　HTTP
C　SSH
D　HTTPS
正確解答為：D　您的作答值為：D　答對了。TNO：U0010041
題目解析：
HTTPS（HyperText Transfer Protocol Secure）是透過網際網路進行安全通訊的傳輸協定，HTTPS經由HTTP進行通訊，但利用SSL/TLS來加密封包，HTTPS開發的主要目的，是提供對網站伺服器的身分認證，保護交換資料的隱私與完整性。FTP（File Transfer Protocol，檔案傳輸協定）是用於電腦網路上進行檔案傳輸的應用層協定。HTTP（HyperText Transfer Protocol）是用來傳輸多媒體文件的應用層協定，主要用於讓瀏覽器和伺服器進行溝通。SSL（Secure Sockets Layer，安全通訊協定）是一種安全協定，主要用於網際網路中兩個系統間的連線安全及資料完整性保障，兩個系統可以是伺服器與用戶端，或者伺服器至伺服器。

安全加密的瀏覽所使用的安全協定為下列哪一項？
A　ADSL
B　XML
C　SSL
D　UML
正確解答為：C　您的作答值為：C　答對了。TNO：U0010042
題目解析：
SSL（Secure Sockets Layer，安全通訊協定）是一種安全協定，主要用於網際網路中兩個系統間的連線安全及資料完整性保障，兩個系統可以是伺服器與用戶端，或者伺服器至伺服器。ADSL（Asymmetric Digital Subscriber Line，非對稱數位用戶線路）是利用電話線和數據機達到寬頻上網速度的網路線路。XML（Extensible Markup Language，可延伸標記式語言）是一種標記式語言，用來傳送及攜帶資料資訊，說明資料是什麼，以及攜帶資料資訊等。UML（Unified Modeling Language）是一種用於說明、可視化、建構一個正在開發的物件導向系統的方法。
