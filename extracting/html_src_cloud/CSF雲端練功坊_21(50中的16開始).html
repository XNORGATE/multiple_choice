雲端運算架構與技術7 / 1547 %
1
某公司想要進行雲端化，下列哪一項動作不見得是必要的？
A　考慮將公司的服務放到提供IaaS的運行商平台上
B　將原有伺服器的CPU升級
C　將資料庫放至PaaS的供應商
D　使用Gmail企業版取代自行架設Exchange伺服器
正確解答為：B　您的作答值為：B　答對了。TNO：U0060016
題目解析：
雲端化之後所有運算與資料皆都由雲端服務供應商提供，因此企業不需將原有伺服器的CPU升級。
2
當程式設計師將原來本機的資料庫換成平台即服務（Platform as a Service, PaaS）的資料庫時，下列敘述哪一項錯誤？
A　只需將連接資料庫的程式碼換成PaaS的位置，不需太多更動
B　大部分的程式碼皆不需要修改
C　如果是指令稿式（Script）的程式，程式必須重新編譯
D　目前大部分平台都提供了眾多應用程式介面（API）幫助使用者移植程式
正確解答為：C　您的作答值為：A　回答錯誤了。TNO：U0060017
題目解析：
Script主要是使用在自動化操作指令，通常類似直譯式程式語言，不用編譯即可直接執行，以Shell Script為例，透過Unix shell命令列直譯器來執行，不用編譯即可直接執行。
3
下列哪一項不是雲端運算的種類之一？
A　公有雲，如Amazon EC2
B　私有雲，如公司內部架設的雲端
C　混合雲，將公有雲和私有雲混合的雲端
D　個人雲，個人電腦中的雲端運算
正確解答為：D　您的作答值為：D　答對了。TNO：U0060018
題目解析：
美國國家標準與技術研究院（NIST）提出的雲端運算定義中，雲端運算有四種部署模式：私有雲（Private Cloud）、公有雲（Public Cloud）、混和雲（Hybrid Cloud）、社群雲（Community Cloud）。
4
關於基礎設施即服務（Infrastructure as a Service, IaaS），下列敘述哪一項錯誤？
A　將運算資源和運算設備分離
B　大部分使用虛擬化完成
C　Amazon EC2就是IaaS的一種應用
D　使用垂直擴充（Scale Up）代替水平擴充（Scale Out）
正確解答為：D　您的作答值為：C　回答錯誤了。TNO：U0060019
題目解析：
雲端運算的系統架構是完全不同的設計，傳統大型主機通常採用垂直擴充（Vertical Scaling / Scale Up）的設計架構，而雲端運算則採用水平擴充（Horizontal Scaling / Scale Out）的設計架構。垂直式的擴充，是指不斷提高單一伺服器的運算能力，在系統內增加更多的資源，如CPU、記憶體或GPU等；而水平式擴充則是不斷增加伺服器的數量，而不需要提高單一臺伺服器的運算能力，當使用服務的使用者越來越多，要處理的資料量越來越大時，只需要不斷增加新伺服器即可。
5
目前大部分的網際網路資料中心（Internet Data Center, IDC）都提供了下列哪一項雲端運算服務？
A　基礎設施即服務（Infrastructure as a Service, IaaS）
B　平台即服務（Platform as a Service, PaaS）
C　軟體即服務（Software as a Service, SaaS）
D　資料即服務（Data as a Service, DaaS）
正確解答為：A　您的作答值為：D　回答錯誤了。TNO：U0060020
題目解析：
目前大部分的網際網路資料中心都提供了IaaS（Infrastructure as a Service）雲端運算服務，使用者端無須購買伺服器、作業系統、軟體等，即可任意部署和運算、儲存、網路和其它基本的運算資源。
6
下列哪一項雲端運算的服務出現的時間最早？
A　基礎設施即服務（Infrastructure as a Service, IaaS）
B　平台即服務（Platform as a Service, PaaS）
C　軟體即服務（Software as a Service, SaaS）
D　都於2010年後才出現
正確解答為：C　您的作答值為：C　答對了。TNO：U0060021
題目解析：
SaaS的服務模式在雲端運算概念普及前就已普及，如1972年就發明的email服務，那時還沒有「雲端」這個名詞。IaaS的服務概念則在1996年才被人提出，但直到2006年，Amazon才發布全球第一個伺服器農場服務。
7
下列哪一項功能無法以基礎設施即服務（Infrastructure as a Service, IaaS）達成？
A　在Linux中線上擴充伺服器數量
B　在 Windows 10 中線上增加CPU數量
C　在Windows Server 2008 Enterprise x64中線上增加記憶體數量
D　在Windows 10中線上增加硬碟容量
正確解答為：B　您的作答值為：D　回答錯誤了。TNO：U0060022
題目解析：
在IaaS服務中，擴充伺服器數量跟硬碟容量基本上都沒問題，然而CPU與記憶體的線上即時擴充仍需要看安裝的作業系統是否有辦法識別並使用，以VMware vSphere的hot-add/plug功能為例，新版的Linux系統大多數均有支援，而Windows則僅有少部分的Server版本支援，如Windows Server 2008 Enterprise x64。
8
在雲端運算中，將伺服器硬體資源用服務發布成基礎設施即服務（Infrastructure as a Service, IaaS）的，最有名的技術為下列哪一項？
A　物件化
B　實體化
C　虛擬化
D　抽象化
正確解答為：C　您的作答值為：C　答對了。TNO：U0060023
題目解析：
雲端運算之所以能夠統整運算資源、彈性調配資源、應付大量運算需求、提供不同虛擬機器能使用或共享實體伺服器的資源，關鍵就在於虛擬化技術。
9
常用的虛擬化主要針對的對象為下列哪一項？
A　作業系統
B　硬體資源
C　應用程式
D　使用者介面
正確解答為：B　您的作答值為：B　答對了。TNO：U0060024
題目解析：
虛擬化技術透過軟體模擬硬體功能，進而建立虛擬的電腦系統，虛擬化技術提供不同應用軟體或虛擬機器能使用或共享實體伺服器的資源，讓不同的軟體和虛擬機器在同一個伺服器上面運行，以達到充分利用資源與迅速重新部署的靈活性。
10
關於虛擬化，下列敘述哪一項正確？
A　虛擬化可在任何硬體上執行
B　虛擬化能減少中央處理器（Central Processing Unit, CPU）的使用
C　虛擬化可同時讓實體硬碟有多個虛擬硬碟
D　虛擬化可加快應用程式的執行速度
正確解答為：C　您的作答值為：C　答對了。TNO：U0060025
題目解析：
虛擬化需要硬體的支援，才有辦法運行，虛擬化的目的主要在於讓多人或多應用能夠共享高效能的伺服器資源，虛擬化技術並無法減少CPU的使用或增加應用程式執行速度。
11
下列哪一項不是虛擬化中的主要技術？
A　網路
B　管理介面
C　伺服器
D　儲存設備
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：U0060026
題目解析：
虛擬化主要技術有伺服器虛擬化、網路虛擬化和儲存設備虛擬化。透過伺服器虛擬化（Server Virtualization），能夠在實體伺服器上以虛擬機的形式執行多個作業系統；透過網路虛擬化（Network Virtualization），能在實體網路上以虛擬網路的形式重現多種網路架構，程式在虛擬網路上執行時，如同在實體網路上運作一樣；透過儲存設備虛擬化（Storage Virtualization），能將多個實體儲存設備集合起來，創造出一個或多個虛擬的儲存設備。
12
虛擬化分為兩大類，企業應用較偏好的類別為下列哪一項？
A　原生架構（Native）
B　寄居架構（Hosted）
C　混合架構
D　共生架構
正確解答為：A　您的作答值為：C　回答錯誤了。TNO：U0060027
題目解析：
虛擬化的實現方式主要有兩種：原生架構（Native）與寄居架構（Hosted）。如下圖所示，原生架構是直接在硬體層上建置虛擬機監視器（Virtual Machine Monitor或Hypervisor），直接將硬體介面抽象化，負責對虛擬機提供硬體資源抽象，其性能較高，因此較受企業應用接受。寄居架構則是在原作業系統上運行一層虛擬機監視器，做寄居作業系統與原作業系統的溝通，進一步取得伺服器硬體資源，因寄居架構需經層層作業系統與虛擬機監視器的溝通而存取硬體資源，導致其性能較低。
13
虛擬化分為兩大類，最方便架設的類別為下列哪一項？
A　原生架構（Native）
B　寄居架構（Hosted）
C　混合架構
D　共生架構
正確解答為：B　您的作答值為：A　回答錯誤了。TNO：U0060028
題目解析：
寄居虛擬化是完全依賴於宿主作業系統，實作上較容易，但性能較低；原生架構完全脫離了宿主作業系統，實作上較複雜，但性能較高。
14
下列哪一項不是虛擬化技術中原生架構（Native）的最大特色？
A　支援硬體種類較少
B　Hypervisor消耗的資源（Overhead）較少
C　支援的企業功能較少
D　全球的裝機量較少
正確解答為：C　您的作答值為：B　回答錯誤了。TNO：U0060029
題目解析：
原生架構讓虛擬機監視器（Virtual Machine Monitor或Hypervisor）能直接存取資源，雖然效率較高，但必須與各種伺服器硬體製造商做配合，發展不同的Hypervisor，因此其支援的硬體種類較少。且原生架構不像寄居架構需依賴宿主作業系統，因此其消耗的資源也較少。雖然原生架構效能較高，但實作上較複雜，因此多用在企業應用，相較於寄居架構，其裝機量較少。原生架構的虛擬化軟體有VMware ESX、VMware vSphere、Xen、KVM、Adeos、CP/CMS、Hyper-V、LynxSecure、SIMMON、XtratuM、z/VM，且原生架構主要應用於企業或商用，因此支援的企業功能較多。
15
寄居架構（Hosted）裝機量最大的產品為下列哪一項？
A　VMware Workstation
B　VMware vSphere
C　Xen
D　KVM
正確解答為：A　您的作答值為：A　答對了。TNO：U0060030
題目解析：
VMware vSphere、Xen、KVM都屬於原生架構，寄居架構的軟體有VMware Workstation、Microsoft Virtual Server、Parallels Workstation、Parallels Desktop、Parallels Server、PearPC、QEMU、VirtualBox、Virtual Iron、VMware Fusion、VMware Player、VMware Server、Windows Virtual PC等，其中以VMware Workstation市佔率最高。
