在網際網路中常以URL（Uniform Resource Locator）來表示網路資源的位址，URL跟IP位址的主要差異為下列哪一項？
A　URL表示一個網際網路節點的唯一位址，IP位址可以更進一步地表示該檔案所在節點上的路徑
B　URL表示一個網際網路節點的唯一位址，IP位址可以更進一步地表示該節點上的資源的位址
C　IP位址表示一個網際網路檔案的唯一位址，URL可以更進一步地表示該檔案所在節點的位址
D　IP位址表示一個網際網路節點的唯一位址，URL可以更進一步地表示該節點上的資源的位址
正確解答為：D　您的作答值為：D　答對了。TNO：U0050017
題目解析：
IP位址表示一個網際網路節點的唯一位址。URL（Uniform Resource Locator）俗稱網頁位址或簡稱網址，是網際網路上標準的資源位址，可以更進一步地表示該節點上的資源的位址，例如某個網頁、某個檔案、某個照片。URL的標準格式為「[協定類型]://[伺服器位址]:[埠號]/[資源層級UNIX檔案路徑][檔名]?[查詢]#[片段ID]」，其中[埠號]、[查詢]、[片段ID]屬於選填項目。

下列哪些技術是目前無線寬頻通訊的主流技術？(複選題)
A　LoRa
B　Wi-Fi
C　RFID
D　LTE-A
正確解答為：BD　您的作答值為：B　回答錯誤了。TNO：U0050018
題目解析：
根據IEEE 802.16-2004標準，寬頻的定義是指擁有高於1 MHz的瞬間頻寬，以及支援高於1.5 Mbit/s的資料量，無線寬頻則是以無線傳輸方式達成寬頻的要求，目前的主流技術有Wi-Fi與4G行動通訊使用的LTE-A。Wi-Fi是基於IEEE 802.11標準的無線區域網路技術，第一代Wi-Fi基於IEEE 802.11原始標準，最高速率2 Mbps，第二代Wi-Fi基於IEEE 802.11b標準，最高速率11 Mbps，第三代Wi-Fi基於IEEE 802.11a/g標準，最高速率54 Mbps，第四代Wi-Fi基於IEEE 802.11n標準，最高速率600 Mbps，第五代Wi-Fi基於IEEE 802.11ac標準，最高速率6.9 Gbps，第六代Wi-Fi基於IEEE 802.11ax標準，最高速率9.6 Gbps。第四代行動通訊標準(4G)以LTE與LTE-A技術為主，LTE-A以MIMO多重天線收發技術，提供最多8x8 MIMO DL、4x4 MIMO UL，下載/上傳速率可達到1Gbps/500Mbps。而LoRa使用頻寬為125~500KHz，傳輸速率僅有300bps~50kbps，RFID主要用於身份識別，傳輸速度亦不到100kbps，因此LoRa與RFID皆無法滿足無線寬頻通訊的需求。

下列哪一項網路裝置的主要功能在於將資料封包於網際網路中轉送，需要決定封包轉送的路徑，通常可以連接聯外的數據專線，並且能將資料封包傳送到其他網域？
A　集線器（Hub）
B　交換器（Switch）
C　數據機（Modem）
D　路由器（Router）
正確解答為：D　您的作答值為：C　回答錯誤了。TNO：U0050019
題目解析：
路由器（Router）提供路由與轉送兩種重要機制，可以決定封包從來源端到目的端所經過的傳輸路徑，這個過程稱為路由，將路由器輸入端的封包轉送至適當的路由器輸出端，這稱為轉送，路由器屬於OSI第三層-網路層的設備。集線器（Hub）屬於OSI第一層-實體層的設備，集線器上有多個I/O埠，集線器會把收到的任何數位訊號，經過再生或放大，再從集線器的所有埠送出。交換器（Switch）屬於OSI第二層-資料鏈路層的設備，交換器內部的CPU會在每個埠成功連接時，通過將MAC位址和埠對應，形成一張MAC表，在轉送封包時，發往某MAC位址的封包將僅送往其對應的埠，而不是所有的埠。數據機（Modem）則是一個將數位訊號調變到類比訊號上進行傳輸，並解調收到的類比訊號以得到數位訊號的電子裝置，數據機可以使用不同的介質來傳送類比訊號，如光纖、無線電、電話線、同軸電纜等。

點對點（Peer-to-Peer, P2P）檔案分享應用程式可讓終端用戶間有效率的分享檔案，不僅容易找到自己需要的檔案資源，並能夠迅速取得，主要原因為下列哪些？(複選題)
A　用戶之間以網路硬碟共享的方式分享檔案
B　用戶的數目龐大，共享的檔案眾多
C　用戶需要的檔案可以同時從多個擁有該檔案的其他用戶端下載
D　用戶透過高效率集中的檔案伺服器進行檔案的分享
正確解答為：BC　您的作答值為：ABC　回答錯誤了。TNO：U0050023
題目解析：
P2P檔案分享應用程式注重的是檔案的傳輸與分享，共享的檔案資源並不是在集中的伺服器上等待用戶端來下載，而是分散在所有參與者的硬碟上，這些參與者組成一個虛擬網路，每個用戶端都可以從這個虛擬網路裡的任何一個人的電腦裡下載檔案，而每一個下載者同時都會擔當了上傳的角色，因此越多人下載，即表示擁有越多上傳者將它們部分的上傳寬頻分享出來，然後再平均分給各個下載者，簡單來說，當你下載檔案時，同時你也會將下載完的檔案再分享出來平均分享給每一位下載者，這就是P2P的共享概念。這種共享的概念，透過使用者的播種來產生種子，每一個種子代表一個已完整下載的檔案，只要有一個種子，便可以繼續分享檔案，其他使用者便有機會可以繼續播種分享給其他使用者，因此加入播種的人越多，下載的頻寬也越大（因為更多人分享上傳頻寬），種子產生的機會就越大（因為可以完成檔案下載的機會越大）。

一般的行動裝置連接網際網路所使用的網路介面跟透過桌上型電腦使用的網路介面，最主要的差異為何？
A　行動裝置的網路介面所提供的速率，高於桌上型電腦的網路介面所提供的速率
B　行動裝置的網路介面所提供的通訊品質，優於桌上型電腦的網路介面所提供的通訊品質
C　行動裝置通常使用無線的網路介面，桌上型電腦通常使用有線的網路介面
D　行動裝置網路介面的通訊費用，遠低於桌上型電腦網路介面的通訊費用
正確解答為：C　您的作答值為：C　答對了。TNO：U0050024
